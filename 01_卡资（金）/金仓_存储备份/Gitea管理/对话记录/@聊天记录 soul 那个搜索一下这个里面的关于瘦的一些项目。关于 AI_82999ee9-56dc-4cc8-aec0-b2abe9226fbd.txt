## 二、AI 分销项目（核心可做项目）

### 2.1 项目定义

- **名称**：**AI 工具分销 + 淘客组合**
- **一句话**：把「AI 工具分销网站」做聚合，发到群里做收益，单粉产值可比传统淘客提升约 **8 倍**（实测未到 30 倍，但投产比还可以）。

### 2.2 为什么可做

| 对比项       | 传统淘客                     | AI 淘客（本方案）                 |
|--------------|------------------------------|------------------------------------|
| 单粉产值     | 约 1.5～3 元/月/群           | 约 10～20 元/月/群                |
| 逻辑         | 京东/淘宝联盟链接发群        | AI 工具分销网站聚合发群 + 企微推送 |
| 现状         | 越来越难做、产值低           | 有打法、闭环简洁                  |

### 2.3 商业闭环（三步）

1. **找产品 + 做内容**
   - 找：好用的 **AI 工具**（含抖音上好用的、有分销/分享内容的）。
   - 做：产品 + 内容整合（网站聚合 + 可发群的内容）。

2. **找流量 + 算投产比**
   - 找群、找流量。
   - 把**投产比、运营数据**跑出来、算清楚。

3. **找资源、跑通闭环**
   - 拿**运营数据**去和淘客、优质资源谈合作。
   - **不用宝妈分销**：直接整合资源，给合作方「原收益的 1.2～1.5 倍」，让对方把群交给你运作，对方不用请人。

### 2.4 分工需求（谁来做）

- **至少 1 人**：每天找新的 AI 工具/网站，生成新内容，推到群里。
- **至少 1 人**：找合作社群、合作渠道，配置企业微信，把内容推到合作群/企微。

### 2.5 卡若可贡献资源

- 手上有**很多淘客群**，以及做淘客的朋友（他们也有很多群）。
- 若有人愿意做，资源可贡献出来，但前提是**自己先跑通、和现有事情有关联**；无关联则仅作分享，谁想做可自己做。

### 2.6 当前状态与卡点（90 场）

- 已测试，投产比还可以，**约淘客的 8 倍**，未到 30 倍。
- **在推进**：召集小伙伴往下做；尤威/南风/小虎做**项目记录、进度百分比、卡点、问题**。
- **材料**：先跑通最简运营闭环，再补资料；逻辑和部分材料已有。

def get_token():
    r = httpx.post(
        "https://open.feishu.cn/open-apis/auth/v3/tenant_access_token/internal",
        json={"app_id": APP_ID, "app_secret": APP_SECRET},
        timeout=15,
    )
    d = r.json()
    if d.get("code") == 0:
        return d["tenant_access_token"]
    print(f"❌ 获取 token 失败: {d}")
    return None


def get_chats(token):
    h = {"Authorization": f"Bearer {token}"}
    chats, pt = [], None
    while True:
        params = {"page_size": 100}
        if pt:
            params["page_token"] = pt
        r = httpx.get(
            "https://open.feishu.cn/open-apis/im/v1/chats",
            headers=h,
            params=params,
            timeout=15,
        )
        d = r.json()
        if d.get("code") != 0:
            break
        items = d.get("data", {}).get("items", [])
        chats.extend(items)
        pt = d.get("data", {}).get("page_token")
        if not pt or not d.get("data", {}).get("has_more"):
            break
    return chats


def load_webhooks():
    raw = os.getenv("FEISHU_CHAT_WEBHOOKS", "")
    if raw:

    path = os.path.join(os.path.dirname(SCRIPT_DIR), "chat_webhooks.json")
    if os.path.exists(path):
        try:
            with open(path, "r", encoding="utf-8") as f:

        except Exception:
            pass
    return {}


def send_via_webhook(webhook_url: str, content: str) -> bool:
    r = httpx.post(
        webhook_url,
        json={"msg_type": "text", "content": json.dumps({"text": content})},
        timeout=15,
    )
    d = r.json()
    return d.get("StatusCode") == 0 or d.get("code") == 0


def send_via_api(token: str, chat_id: str, content: str) -> bool:
    r = httpx.post(
        "https://open.feishu.cn/open-apis/im/v1/messages",
        headers={"Authorization": f"Bearer {token}"},
        params={"receive_id_type": "chat_id"},
        json={
            "receive_id": chat_id,
            "msg_type": "text",
            "content": json.dumps({"text": content}),
        },
        timeout=15,
    )
    return r.json().get("code") == 0
