## 二、从 Mac 连接电视（投屏）

### 1. 确保同网

- 海尔电视：设置 → 网络 → 连接与 Mac **相同的 WiFi**（推荐 5GHz）。
- Mac：菜单栏 WiFi 图标确认已连该 WiFi。

### 2. 打开屏幕镜像

1. 点击 Mac 右上角 **控制中心**（或菜单栏）
2. 点击 **「屏幕镜像」**
3. 在「镜像或扩展至：」下选择 **「卧室的电视」**
4. 若提示输入密码，在电视上查看并输入

### 3. 镜像 vs 扩展显示器

- **镜像**：电视与 Mac 显示相同画面（当前你看到的模式）。
- **扩展**：电视当第二块屏，可单独拖窗口。
  - 操作：在「屏幕镜像」里选中「卧室的电视」后，可看到 **「用作单独显示器」** 或类似选项，选它即切换为扩展。

---

## 三、海尔电视品牌连接要点（搜索整理）

- 海尔智能电视支持 AirPlay 的型号：需电视系统/固件支持，部分新款内置。
- 若电视不支持 AirPlay：可外接 **Apple TV** 机顶盒，电视选对应 HDMI 信源即可。
- 有线方案（最稳定）：Mac 用 **HDMI 或 USB-C 转 HDMI** 接电视，电视信源选该 HDMI。

---

## 四、从电脑「控制」电视的几种方式

| 方式 | 说明 |
|------|------|
| **AirPlay 投屏** | Mac 只能把画面/声音投到电视，不能像遥控器那样换台、调音量。 |
| **电视遥控 / 手机 App** | 换台、音量、开关机等仍用电视自带遥控器或海尔/厂商 App。 |
| **扩展屏使用** | 用 Mac 键盘鼠标控制拖到电视上的窗口，等于「控制投过去的内容」。 |
| **HDMI + CEC** | 若用 HDMI 连接且电视支持 CEC，部分机型可用 Mac 或遥控联动开关机。 |

**总结**：当前连接是 **AirPlay 投屏**，Mac 负责「投什么画面」，电视端操作（换台、音量、设置）需用遥控器或电视/海尔 App。

- **投屏卡顿**：尽量用 5GHz WiFi，或改用 HDMI 有线。
- **想当扩展屏**：在屏幕镜像里选中该设备后，选择「用作单独显示器」（或系统等价选项）。

## 一、在路由器里加 Hosts

1. 电脑浏览器打开路由器管理页（常见 192.168.110.1 或 192.168.1.1），登录。
2. 找到 **自定义 Hosts** / **静态 DNS** / **域名黑名单** / **家长控制-网站过滤**（不同品牌名称不同）。
3. 新增以下条目（IP 填 **127.0.0.1**，主机名填下面域名，一行一个或按路由器格式）：

```
127.0.0.1  t7z.cupid.ptqy.gitv.tv
```

4. 保存并让路由器**立即生效**（有的需重启路由器）。

> **铁律**：对远程设备做任何点击/操作前，必须先截屏 → 拉取到本机 → 分析界面位置 → 再执行 `input tap x y` 或 keyevent。不得乱点击。

## 能力说明

- **做什么**：对 ADB 可达设备（电视、Android 盒子、手机等）先截屏并拉取到本机，供分析后再操作；支持截图数量上限（默认保留≤10 张，超过则删最早 5 张）。
- **不做什么**：不在未截屏、未分析的情况下对设备发 tap/keyevent。

## 执行流程

1. **截屏**：`adb -s <设备> shell screencap -p /sdcard/screen.png`
2. **拉取**：`adb -s <设备> pull /sdcard/screen.png <本机路径>`
3. **分析**：查看截图，确定目标按钮/区域在分辨率下的坐标（如 1920×1080）。
4. **再操作**：`adb -s <设备> shell input tap <x> <y>` 或 `input keyevent <code>`
5. **截图数量控制**：若同一任务下截图数≥10，删除本机保存目录中最早的 5 张，再继续。
