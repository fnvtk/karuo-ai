# 跨平台手机屏幕控制程序功能迭代记录

## 项目概述

基于scrcpy技术开发的跨平台手机屏幕控制程序，支持MacBook和Windows系统，实现对Android手机的屏幕镜像、远程控制和文件传输功能。

## 系统架构图

```
+------------------------------------------+
|                                          |
|             桌面端应用程序               |
|        (Electron + React + Node.js)      |
|                                          |
+------------------+---------------------+
                   |
                   | ADB协议/WebSocket/HTTP
                   |
+------------------v---------------------+
|                                          |
|              手机端应用程序              |
|               (Android原生)              |
|                                          |
+------------------------------------------+
```
