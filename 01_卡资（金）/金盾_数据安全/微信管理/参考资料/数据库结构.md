# 微信数据库结构参考

> macOS 微信本地数据库表结构说明

---

## 数据库文件位置

```
~/Library/Containers/com.tencent.xinWeChat/Data/Library/Application Support/
└── [32位用户hash]/
    ├── Message/
    │   ├── msg_0.db         # 消息数据库 (按hash分片)
    │   ├── msg_1.db
    │   └── ...
    ├── Contact/
    │   └── wccontact_new2.db  # 联系人数据库
    ├── Group/
    │   └── group_new.db       # 群组数据库
    └── Favorites/
        └── favorites.db       # 收藏夹数据库
```

## 消息表结构 (Message)

### Chat_xxx 表

每个联系人/群聊一个表，表名格式：`Chat_[md5hash]`

| 字段 | 类型 | 说明 |
|:-----|:-----|:-----|
| mesLocalID | INTEGER | 本地消息ID (主键) |
| mesMesSvrID | INTEGER | 服务端消息ID |
| msgCreateTime | INTEGER | 消息创建时间戳 (秒) |
| msgContent | TEXT | 消息内容 |
| msgType | INTEGER | 消息类型 |
| msgStatus | INTEGER | 消息状态 |
| msgImgStatus | INTEGER | 图片状态 |
| mesDes | INTEGER | 消息方向 (0=发出, 1=收到) |
| msgSource | TEXT | 消息来源 (XML格式) |

### 消息类型 (msgType)

| 值 | 类型 | 说明 |
|:---|:-----|:-----|
| 1 | 文本 | 普通文字消息 |
| 3 | 图片 | 图片消息 |
| 34 | 语音 | 语音消息 |
| 42 | 名片 | 个人/公众号名片 |
| 43 | 视频 | 视频消息 |
| 47 | 表情 | 动态表情 |
| 48 | 位置 | 地理位置 |
| 49 | 链接 | 文章/小程序/文件等 |
| 10000 | 系统 | 系统消息 |
| 10002 | 撤回 | 撤回消息 |

### 子类型 (用于 msgType=49)

| 子类型 | 说明 |
|:-------|:-----|
| 1 | 链接 |
| 3 | 音乐 |
| 4 | 视频 |
| 5 | 文章 |
| 6 | 文件 |

| 24 | 收藏笔记 |

## 查询示例

### 获取所有聊天记录表

```sql
SELECT name FROM sqlite_master 
WHERE type='table' AND name LIKE 'Chat_%';
```

### 查询指定时间范围消息

```sql
SELECT * FROM Chat_xxx 
WHERE msgCreateTime BETWEEN 1640000000 AND 1650000000
ORDER BY msgCreateTime DESC;
```

### 统计消息数量

```sql
SELECT COUNT(*) as count, 
       DATE(msgCreateTime, 'unixepoch') as date
FROM Chat_xxx
GROUP BY date
ORDER BY date DESC;
```

### 查询群成员

```sql
SELECT m.memberName, m.displayName, c.nickName
FROM GroupMember m
LEFT JOIN Friend c ON m.memberName = c.userName
WHERE m.chatroomName = '群ID';
```

## 加密参数

### 微信 3.x (SQLCipher v3)

```sql
PRAGMA cipher_page_size = 1024;
PRAGMA kdf_iter = 64000;
PRAGMA cipher_hmac_algorithm = HMAC_SHA1;
```

### 微信 4.x (SQLCipher v4)

```sql
PRAGMA cipher_page_size = 4096;
PRAGMA kdf_iter = 256000;
PRAGMA cipher_hmac_algorithm = HMAC_SHA256;
PRAGMA cipher_kdf_algorithm = PBKDF2_HMAC_SHA256;
```
