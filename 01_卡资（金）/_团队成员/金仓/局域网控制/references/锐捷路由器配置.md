## ⚠️ 优化建议（待执行）

### P0 - 紧急

1. **开启 WiFi 加密**：当前 Security: None，任何人可连入。建议开启 WPA3 或 WPA2-PSK (AES)
2. **更换信道**：从 DFS 信道 64 切换至非 DFS 信道（36/40/44/48 或 149/153/157/161）

### P1 - 重要

3. **检查 MCS 协商**：MacBook 仅协商到 MCS 1 (144 Mbps)，WiFi 6 + 80MHz 应可达 600+ Mbps
   - 可能原因：距离远、信号干扰、路由器功率设置低
4. **NAS 跨网段路由**：NAS 在 192.168.1.201（公司 H3C 网络），当前网络无法访问
   - 方案A：路由器添加静态路由到 192.168.1.0/24
   - 方案B：VPN 隧道连接两个网络
   - 方案C：确认两个路由器是否在同一物理网络

### P2 - 建议

5. **DNS 优化**：当前 223.5.5.5 (阿里)，可考虑添加备用 DNS 114.114.114.114
6. **设备识别**：192.168.110.81 设备身份未知，建议确认是否为授权设备
7. **管理后台 HTTPS**：当前为 HTTP，建议开启 HTTPS
