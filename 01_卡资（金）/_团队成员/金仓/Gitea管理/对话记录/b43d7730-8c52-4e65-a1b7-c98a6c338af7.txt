## 一、核心目标

**一句话目标**：让完成≥5次测评的用户，在预约咨询师并付款后，通过存客宝自动添加用户微信，实现"测评→预约→付款→自动加好友"的私域闭环。

### 核心链路
```
用户完成测评(≥5次) → 预约咨询师 → 填写手机号 → 完成支付(398元)
→ 系统检查条件 → 触发存客宝AI → 自动添加用户微信好友
→ 咨询师与用户建立1对1私域连接
```

### 关键指标
| 指标 | 目标值 |
|:---|:---|
| 触发条件 | 用户完成≥5次测评 + 预约支付成功 |
| 添加成功率 | >60%（行业平均水平） |
| 响应时间 | 支付成功后≤30秒触发添加 |
| 咨询师绑定率 | 100%（每个咨询师必须绑定存客宝手机号） |

## 二、业务流程

### 2.1 用户端完整流程

```
┌─────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌──────────┐
│ 选择     │ → │ 查看详情  │ → │ 填写     │ → │ 完成     │ → │ 存客宝   │
│ 咨询师   │   │ 点击预约  │   │ 手机号   │   │ 支付     │   │ 自动添加  │
└─────────┘   └──────────┘   └──────────┘   └──────────┘   └──────────┘
     ↓              ↓              ↓              ↓              ↓
  浏览列表      查看资质/评价   验证手机号格式   微信支付398元   AI自动发送
  搜索筛选      擅长/价格      选择日期时间      支付回调确认    好友请求
                              选择咨询方式
```

### 2.2 后端触发流程

```
用户点击"立即支付" → 创建预约订单(status: pending_payment)
    → 调用微信支付 → 支付成功回调
    → 更新订单状态(status: paid)
    → 查询用户测评次数
    → IF 测评次数 ≥ 5:
        → 查询咨询师绑定的存客宝手机号
        → 调用存客宝API（传入：用户手机号 + 咨询师存客宝手机号）
        → 创建存客宝任务记录(status: pending)
        → 存客宝异步执行添加
        → 回调更新任务状态(status: sent/accepted/failed)
    → ELSE:
        → 仅记录预约，不触发存客宝
```

### 2.3 咨询师后台绑定流程

```
管理员进入后台 → 咨询师管理 → 编辑咨询师
→ 填写"存客宝绑定手机号"字段
→ 保存 → 该手机号将作为存客宝AI添加好友的发起方
```

import { NextRequest, NextResponse } from "next/server"
import { connectToDatabase, COLLECTIONS } from "@/lib/mongodb"
import { ObjectId } from "mongodb"

/** 存客宝触发最低测评次数（默认5次） */
const MIN_TEST_COUNT = parseInt(process.env.CUNKEBAO_MIN_TEST_COUNT || "5")

/** 手机号正则校验 */
const PHONE_REGEX = /^1[3-9]\d{9}$/

/** 生成订单号 */
function generateBookingNo(): string {
  const now = new Date()
  const dateStr = now.toISOString().replace(/[-T:\.Z]/g, "").slice(0, 14)
  const random = Math.random().toString(36).substring(2, 8).toUpperCase()
  return `BK${dateStr}${random}`
}

/** GET - 查询预约列表 */
export async function GET(req: NextRequest) {
  try {
    const { searchParams } = new URL(req.url)
    const userId = searchParams.get("userId")
    const consultantId = searchParams.get("consultantId")
    const status = searchParams.get("status")
    const page = parseInt(searchParams.get("page") || "1")
    const pageSize = parseInt(searchParams.get("pageSize") || "20")

    const { db } = await connectToDatabase()
    const collection = db.collection(COLLECTIONS.BOOKINGS)

    // 构建查询条件
    const query: Record<string, unknown> = {}
    if (userId) query.userId = userId
    if (consultantId) query.consultantId = consultantId
    if (status) query.status = status

    const total = await collection.countDocuments(query)
    const bookings = await collection
      .find(query)
      .sort({ createdAt: -1 })
      .skip((page - 1) * pageSize)
      .limit(pageSize)
      .toArray()

    return NextResponse.json({
      success: true,
      data: { list: bookings, total, page, pageSize },
    })
  } catch (error: unknown) {
    console.error("[知己] 查询预约列表失败:", error)
    return NextResponse.json({
      success: true,
      data: { list: [], total: 0, page: 1, pageSize: 20 },
    })
  }
}

/** POST - 创建预约订单 */
export async function POST(req: NextRequest) {
  try {
    const body = await req.json()
    const { userId, consultantId, phone, date, time, type, amount } = body

    // === 参数校验 ===
    if (!userId) {
      return NextResponse.json({ success: false, error: "缺少用户ID" }, { status: 400 })
    }
    if (!consultantId) {
      return NextResponse.json({ success: false, error: "缺少咨询师ID" }, { status: 400 })
    }
    if (!phone || !PHONE_REGEX.test(phone)) {
      return NextResponse.json({ success: false, error: "请填写正确的11位手机号" }, { status: 400 })
    }
    if (!date || !time) {
      return NextResponse.json({ success: false, error: "请选择预约日期和时间" }, { status: 400 })
    }

    const { db } = await connectToDatabase()

    // === 查询咨询师信息 ===
    const consultant = await db.collection(COLLECTIONS.CONSULTANTS).findOne({
      _id: new ObjectId(consultantId),
    })
    if (!consultant) {
      return NextResponse.json({ success: false, error: "咨询师不存在" }, { status: 404 })
    }

    // === 查询用户测评次数（决定是否触发存客宝） ===
    const testCount = await db.collection(COLLECTIONS.TEST_RECORDS).countDocuments({ userId })
    const shouldTriggerCunkebao = testCount >= MIN_TEST_COUNT && !!consultant.cunkebaoPhone && consultant.cunkebaoEnabled !== false

    // === 创建预约订单 ===
    const bookingNo = generateBookingNo()
    const newBooking = {
      bookingNo,                                   // 订单号
      userId,                                       // 用户ID
      consultantId,                                 // 咨询师ID
      consultantName: consultant.name || "",         // 咨询师名称（冗余）
      phone,                                         // 用户手机号（核心字段）
      date,                                          // 预约日期
      time,                                          // 预约时间
      type: type || "video",                         // 咨询方式
      amount: amount || 398,                         // 支付金额（元）
      paymentStatus: "pending",                      // 支付状态: pending/paid/refunded
      paymentId: "",                                 // 微信支付订单号
      // 存客宝相关
      cunkebaoEligible: shouldTriggerCunkebao,       // 是否满足存客宝触发条件
      cunkebaoTaskId: "",                            // 存客宝任务ID
      cunkebaoStatus: shouldTriggerCunkebao ? "pending" : "none", // 存客宝状态
      userTestCount: testCount,                      // 用户测评次数（快照）
      // 订单状态
      status: "pending_payment",                     // pending_payment/paid/confirmed/completed/cancelled
      createdAt: new Date().toISOString(),
      updatedAt: new Date().toISOString(),
    }

    const result = await db.collection(COLLECTIONS.BOOKINGS).insertOne(newBooking)

    console.log(`[知己] 预约订单创建成功: ${bookingNo}, 用户测评${testCount}次, 存客宝触发: ${shouldTriggerCunkebao}`)

    return NextResponse.json({
      success: true,
      data: {
        bookingId: result.insertedId.toString(),
        bookingNo,
        status: "pending_payment",
        amount: newBooking.amount,
        cunkebaoEligible: shouldTriggerCunkebao,
        userTestCount: testCount,
      },
    })
  } catch (error: unknown) {
    console.error("[知己] 创建预约订单失败:", error)
    return NextResponse.json({ success: false, error: "创建预约失败" }, { status: 500 })
  }
}

/** PUT - 更新订单状态（支付回调/手动更新） */
export async function PUT(req: NextRequest) {
  try {
    const body = await req.json()
    const { bookingId, paymentStatus, paymentId, status: orderStatus } = body

    if (!bookingId) {
      return NextResponse.json({ success: false, error: "缺少订单ID" }, { status: 400 })
    }

    const { db } = await connectToDatabase()
    const collection = db.collection(COLLECTIONS.BOOKINGS)

    // 查找订单
    const booking = await collection.findOne({ _id: new ObjectId(bookingId) })
    if (!booking) {
      return NextResponse.json({ success: false, error: "订单不存在" }, { status: 404 })
    }

    // 构建更新数据
    const updateData: Record<string, unknown> = {
      updatedAt: new Date().toISOString(),
    }
    if (paymentStatus) updateData.paymentStatus = paymentStatus
    if (paymentId) updateData.paymentId = paymentId
    if (orderStatus) updateData.status = orderStatus

    // === 支付成功 → 触发存客宝 ===
    if (paymentStatus === "paid" && booking.cunkebaoEligible) {
      updateData.status = "paid"
      updateData.paymentStatus = "paid"

      // 查询咨询师的存客宝手机号
      const consultant = await db.collection(COLLECTIONS.CONSULTANTS).findOne({
        _id: new ObjectId(booking.consultantId),

      if (consultant?.cunkebaoPhone) {
        // 创建存客宝任务
        const taskId = `CKB_${Date.now()}_${Math.random().toString(36).substring(2, 8).toUpperCase()}`
        const cunkebaoTask = {

          userPhone: booking.phone,
          consultantId: booking.consultantId,

        // 更新订单的存客宝任务关联
        updateData.cunkebaoTaskId = taskId
        updateData.cunkebaoStatus = "pending"

        console.log(`[知己·存客宝] 任务创建: ${taskId}, 用户手机: ${booking.phone}, 咨询师存客宝: ${consultant.cunkebaoPhone}`)

        // TODO: 实际调用存客宝API（等存客宝接口对接后启用）
        // await callCunkebaoAPI(cunkebaoTask)

    } else if (paymentStatus === "paid") {
      // 支付成功但不触发存客宝（测评次数不足）
      updateData.status = "paid"
      updateData.paymentStatus = "paid"

## 存客宝核心目标（每次对话必读必更新）

> **一句话目标**：让完成≥5次测评的用户，在预约咨询师并付款后，通过存客宝AI自动添加用户微信，实现私域闭环。

### 存客宝业务闭环
```
用户完成测评(≥5次) → 预约咨询师 → 填写手机号 → 完成支付(398元)
→ 系统自动判断条件 → 触发存客宝AI → 自动添加用户微信好友
→ 咨询师与用户建立1对1私域连接
```

### 关键规则
1. **触发条件**：用户测评完成次数 ≥ 5次 + 预约支付成功 + 咨询师已绑定存客宝手机号
2. **咨询师绑定**：每个咨询师在后台登记时必须绑定一个存客宝手机号
3. **手机号收集**：预约流程中必须填写用户手机号（11位，正则校验）
4. **自动添加**：支付成功后30秒内触发存客宝API，AI自动发送好友请求
5. **失败重试**：最多重试3次，失败后支持后台手动触发

### 存客宝相关API
| API | 方法 | 用途 |
|:---|:---:|:---|
| `/api/booking` | POST | 创建预约订单（含手机号） |
| `/api/booking` | PUT | 更新支付状态 → 触发存客宝 |
| `/api/booking` | GET | 查询预约列表 |
| `/api/cunkebao` | GET | 查询存客宝任务状态 |
| `/api/cunkebao` | POST | 手动触发/重试存客宝添加 |
| `/api/cunkebao/callback` | POST | 存客宝结果回调 |

### 存客宝数据集合
| 集合 | 用途 |
|:---|:---|
| `bookings` | 预约订单（含手机号+支付+存客宝状态） |
| `cunkebao_tasks` | 存客宝自动添加任务记录 |

### 存客宝环境变量（.env.local）
```
CUNKEBAO_API_URL=         # 存客宝API地址
CUNKEBAO_API_KEY=         # 存客宝API密钥
CUNKEBAO_CALLBACK_URL=    # 回调地址
CUNKEBAO_MIN_TEST_COUNT=5 # 触发最低测评次数
```
