### 一、你开口的那一刻

```
👤 你说话
    ↓
🧠 卡若AI 收到
    ↓
📖 自动读取：记忆.md + 总索引 + 相关 Skill
```

每次对话开始，都会先加载你的记忆、总索引和可能用到的 Skill，保证上下文完整。

### 二、核心四步（不可跳过）

```
┌─────────────────────────────────────────────────────────────┐
│  📐 思考 → 拆解 → 读取 → 执行（铁律，谁都不能跳过）           │
└─────────────────────────────────────────────────────────────┘

   ① 🧠 思考（深度理解）
      ├─ 你要什么？结合上下文、记忆
      ├─ 置信度：高/中/低（低就先追问）
      └─ 不确定性：uncertainty_areas
      
   ② 📋 拆解（计划）
      ├─ 步骤1 → 步骤2 → 步骤3...
      ├─ 每步有完成标准（definitions_of_done）
      └─ 标注关键路径、可并行步骤
      
   ③ 📂 读取
      └─ 记忆、文档、Skill、总索引
      
   ④ ⚡ 执行
      ├─ 按步骤做，每步汇报进度
      └─ 失败 → 重规划（保留已完成）
```

也就是说：先想清楚、再拆成计划、再读资料、最后按计划执行，中间不跳过任何一步。
