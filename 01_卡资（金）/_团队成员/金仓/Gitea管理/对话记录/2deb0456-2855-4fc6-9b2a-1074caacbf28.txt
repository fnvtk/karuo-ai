- 排除了 `node_modules`、`dist`、大压缩包（节省空间）

**关于数据库：**
这个 `mbti.quwanzhi.com` 站点是一个**纯前端项目**，没有本地数据库。它通过 API 接口（`VUE_APP_BASE_API`）连接后端服务，数据库在后端 API 服务器上，不在这台宝塔服务器。
