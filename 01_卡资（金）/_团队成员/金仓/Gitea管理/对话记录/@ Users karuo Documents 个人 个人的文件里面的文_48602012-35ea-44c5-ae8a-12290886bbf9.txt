  contents: # 本地免费恢复方案：PhotoRec 操作指南（不依赖 Git / Time Machine）

**适用**：误删文件、清空废纸篓后 Time Machine 无法恢复时，用本机已安装的 **PhotoRec** 做深度恢复。  
**重要**：恢复出来的文件**必须保存到另一块盘/另一个分区**，不能写回被恢复的盘，否则会覆盖已删数据。

## 一、你本机已具备

- **PhotoRec** 已通过 Homebrew 安装（`/usr/local/bin/photorec`）
- **恢复文件保存位置**：你有挂载的「MacBook Pro的备份」卷，可用来存放恢复出的文件，避免写回系统盘

---

## 二、操作步骤（按顺序做）

### 1. 创建恢复文件存放目录

在「MacBook Pro的备份」上建一个专门文件夹（避免和备份混在一起）：

```bash
mkdir -p "/Volumes/MacBook Pro的备份/PhotoRec恢复_$(date +%Y%m%d)"
```

恢复出来的文件会保存在这个目录下（例如 `PhotoRec恢复_20260213`）。

### 2. 用 sudo 启动 PhotoRec

在终端执行：

```bash
sudo photorec
```

按提示输入本机登录密码（输入时不显示，输完回车即可）。

### 3. 选择要扫描的磁盘

- 用 **上下方向键** 选磁盘。
- 你的「个人」文件在系统数据卷上，对应的是 **APFS 容器** 里的 **Data** 卷。
- 列表中可能显示为：
  - **/dev/disk3**（或 **/dev/rdisk3**，带 `r` 的更快）
  - 或 **/dev/disk0**
- 选包含「Data」或容量约 895 GB 的那一项（即你放文档的那块）。
- 选好后按 **Enter**。

若不确定，可先选 **/dev/rdisk3**（与当前系统 Data 卷对应），再在下一步选分区。

### 4. 选择分区（Partition）

- 用方向键选 **\[Proceed\]** 或 **\[Search\]** 进入分区列表。
- 选择 **Data** 或容量约 895 GB 的分区（即你的文档所在分区）。
- 再选 **\[Search\]** 开始扫描。

### 5. 文件系统类型

- 选 **Other**（APFS 在此归为 Other）。

### 6. 扫描范围（可选）

- **Whole**：整分区扫描，更慢但覆盖更全（建议第一次用）。
- **Free**：只扫未分配空间，较快但可能漏掉一些已删文件。

### 7. 选择恢复文件保存位置（必做）

- 用方向键选 **..** 回到上层，直到进入 **/Volumes**。
- 选 **MacBook Pro的备份**。
- 再进入你刚建的目录，例如 **PhotoRec恢复_20260213**。
- 确认路径无误后选 **Y** 或 **Yes** 开始恢复。

**务必不要选系统盘上的目录**（例如不要选 Macintosh HD / Data 下的目录），否则会覆盖被删数据。

### 8. 等待扫描结束

- 扫描时间取决于分区大小，可能几十分钟到数小时。
- 恢复出的文件会出现在 `recup_dir.1`、`recup_dir.2` 等子文件夹里，可按扩展名、修改时间排序查找。
- 恢复出的**文件名会变**（多为 f1234567.pdf 这类），需要自己按类型/时间筛选。

---

## 三、可选：只恢复某几类文件（减少无关文件）

在分区选择界面不要先选 **Search**，而是：

1. 选 **File Opt**（或 **Options** 里的文件类型）。
2. 用空格键**取消**不需要的类型（如只保留 doc/pdf/md/txt 等）。
3. 再返回选 **Search** 开始扫描。

这样恢复结果里会少很多图片、缓存等，便于在「个人」相关文档里查找。

---

## 四、若本机没有「MacBook Pro的备份」卷

- 必须先接**外置 U 盘/移动硬盘**，在磁盘工具里格式化为 Mac 可用格式并挂载。
- 在 PhotoRec 第 7 步里选 **/Volumes/你的外置盘名** 下的一个空文件夹作为保存位置。
- 同样**不要**把恢复目录选在系统盘（Data 卷）上。

---

## 五、其他免费方案（备用）

| 工具        | 说明 |
|-------------|------|
| **TestDisk** | 与 PhotoRec 同源，`brew install testdisk` 已包含；更适合分区表/分区丢失的恢复，也可尝试恢复已删文件（部分情况能保留原文件名）。 |
| **PhotoRec** | 按文件签名扫描，不依赖文件系统，适合「彻底删除」后的深度恢复。 |

两者均为开源免费，无需 Git、无需 Time Machine，纯本机即可使用。
