## 使用方式

### 方式1：AI 辅助拆书（推荐）

1. 用户发送书籍内容
2. AI 使用 `references/卡读提示词.md` 拆解
3. AI 调用脚本写入 XMind

```python
# AI 执行
from scripts.write_to_xmind import write_book_to_xmind

note_data = {
    "summary": "这是一本...",
    "gold": ["金-1：...", "金-2：...", "金-3：...", "金-4：..."],
    "water": ["水-1：...", "水-2：...", "水-3：...", "水-4：..."],
    # ... 其他数据
}

write_book_to_xmind("书名", "作者", "创业", note_data)
```

### 方式2：命令行写入

```bash
# 测试模式（不实际写入）
python scripts/write_to_xmind.py "原则" "瑞·达利欧" "投资" --test

# 实际写入
python scripts/write_to_xmind.py "原则" "瑞·达利欧" "投资"
```

### 方式3：手动操作 XMind

1. **新建标签页**: `⌘+T` 或底部右键
2. **重命名**: `《书名》- 作者`
3. **复制结构**: 从已有书籍复制五行结构
4. **填入内容**: 按模板填写
5. **插入链接**: `⌘+K` → 内部链接到主图分类

## XMind 快捷键

| 操作 | 快捷键 |
|------|--------|
| 新建标签页 | `⌘+T` |

## 脚本说明

### 1. write_to_xmind.py - 写入 XMind

**功能**: 自动将书籍笔记写入 XMind

```bash
# 测试模式
python scripts/write_to_xmind.py "书名" "作者" "分类" --test

# 实际写入
python scripts/write_to_xmind.py "原则" "瑞·达利欧" "投资"
```

**特性**:
- 自动创建五行结构标签页
- 自动在主图对应分类下添加链接（链接指向 rootTopic.id）
- 防重复检查
- ID 格式兼容 XMind 原生格式（26位混合字母数字）

### 2. list_books.py - 列出书籍

**功能**: 查看已有的书籍列表

```bash
python scripts/list_books.py
```

**输出**:
- 各分类下的书籍数量
- 所有书籍标签页列表
- 链接状态（🔗有链接 / 📄无链接）

### 3. parse_ai_output.py - 解析 AI 输出

**功能**: 将 AI 拆书输出解析为 XMind 数据格式

```bash
# 先保存 AI 输出到文件
# 然后解析并写入
python scripts/parse_ai_output.py /tmp/notes.md "书名" "作者" "分类"
```
