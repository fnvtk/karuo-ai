## 快速开始

### 1. 选择文章类型

| 类型 | 适合场景 | 核心特点 |
|:---|:---|:---|
| **A. 日志型** | 会议记录、项目复盘 | 时间线+对话+数据 |
| **B. 干货型** | 私域知识、方法论 | 痛点+案例+步骤 |
| **C. 故事型** | 个人经历、创业感悟 | 冲突+转折+领悟 |
| **D. 观点型** | 挑战传统、行业趋势 | 颠覆+论证+号召 |

### 2. 准备素材

```yaml
必备素材:
 - 主题: "文章要讲什么"
 - 对话/事件: "真实发生的事"
 - 数据: "具体数值（不要编造）"

可选素材:
 - 图片: "配图路径"
 - 目标人群: "写给谁看"
```

### 3. 使用提示词

1. 打开 `references/提示词模板.md`
2. 复制对应类型的提示词
3. 填入你的素材
4. 发给AI生成

### 4. 检查清单

```
□ 字数1500-1900
□ "卡若"最多1次
□ 每句话后空一行
□ 有具体数据
□ 有对话片段
□ 结尾有行动号召
```

---

## 核心规则（记住这5条）

1. **每句话后空一行** - 这是卡若日记的标志
2. **短句为主** - 不超过20字
3. **真实数据** - 用素材中的数据，不编造
4. **隐性植入** - 不直说"会员""合作伙伴"
5. **开篇有冲突** - 制造悬念或矛盾
