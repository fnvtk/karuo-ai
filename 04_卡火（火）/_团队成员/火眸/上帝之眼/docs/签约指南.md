## 方案1：BigQuant + 湘财证券（推荐）

### 前提条件

你的湘财证券账户需要满足以下条件之一：
- 资金账号以 **57** 开头（重庆开户）
- 资金账号以 **13** 开头（成都开户）

你的账号是 `600201668`，可能需要线下签约。

### 签约步骤

#### 第一步：注册BigQuant账号

1. 访问 https://bigquant.com
2. 点击"注册"，使用手机号注册
3. 完成邮箱验证

#### 第二步：申请实盘权限

1. 登录BigQuant
2. 访问 https://bigquant.com/trading/group
3. 填写湘财证券资金账号：`600201668`
4. 提交申请

#### 第三步：湘财证券APP签约

1. 下载"湘财百宝湘"APP
   - iOS: App Store搜索"湘财百宝湘"
   - Android: 各大应用市场搜索

2. 登录你的证券账户

3. 找到签约入口：
   ```
   首页 → 业务办理 → 开通金刚钻服务 → 宽邦量化平台
   ```

4. 签署以下协议：
   - 《量化交易风险揭示书》
   - 《宽邦量化平台使用协议》

5. 提交等待审核（1-2个工作日）

#### 第四步：程序化交易报备（重要）

如果你要使用全自动交易功能，需要完成报备：

1. 联系湘财证券客户经理
2. 说明要进行"程序自动化交易"
3. 完成报备流程

## 方案2：掘金量化（无需客户端）

掘金量化提供纯Python SDK，不需要任何客户端。

### 注册开通

1. 访问 https://www.myquant.cn
2. 注册账号
3. 申请实盘权限（需要绑定券商账户）

### 安装SDK

```bash
pip3 install gm
```

### 使用示例

```python
from gm.api import *

# 设置token（在掘金官网获取）
set_token('your_token')

# 初始化
def init(context):
    context.symbol = 'SHSE.600519'  # 贵州茅台
    subscribe(context.symbol, frequency='1d')

# 数据事件
def on_bar(context, bars):
    # 获取账户
    account = context.account()
    
    # 买入
    order_volume(symbol=context.symbol, volume=100, 
                 side=OrderSide_Buy, order_type=OrderType_Market)

# 运行
run(strategy_id='your_strategy_id',
    filename='main.py',
    mode=MODE_LIVE,  # 实盘模式
    token='your_token')
```

## 常见问题

### Q: 签约需要多长时间？

A: 在线签约：即时生效。线下签约：1-2个工作日。

### Q: 需要最低资金要求吗？

A: BigQuant没有最低资金要求，但建议至少1万元以上用于实盘测试。

### Q: 可以先用模拟盘吗？

A: 可以，BigQuant支持模拟交易，不需要签约即可使用。

### Q: 手续费是多少？

A: 与你在湘财证券开户时约定的佣金一致，BigQuant平台不额外收费。
