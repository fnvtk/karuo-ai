#### 3️⃣ 风控优化

**当前**: 固定5%止损
**优化**: 动态止损 + 移动止盈

```python
class RiskManager:
    def __init__(self):
        self.initial_stop = -0.05  # 初始止损5%
        self.trailing_stop = 0.03  # 回撤3%止盈
        self.max_profit = {}
    
    def check(self, code, current_price, cost):
        pnl = (current_price - cost) / cost
        
        # 记录最高盈利
        if code not in self.max_profit:
            self.max_profit[code] = pnl
        else:
            self.max_profit[code] = max(self.max_profit[code], pnl)
        
        # 初始止损
        if pnl <= self.initial_stop:
            return 'STOP_LOSS'
        
        # 移动止盈（盈利超过10%后启用）
        if self.max_profit[code] > 0.10:
            if pnl < self.max_profit[code] - self.trailing_stop:
                return 'TRAILING_STOP'
        
        return 'HOLD'
```

#### 5️⃣ 仓位管理优化

**当前**: 固定15%仓位

```python
def kelly_position(win_rate, avg_win, avg_loss):
    """凯利公式计算最优仓位"""
    # f = (bp - q) / b
    # b = avg_win / avg_loss
    # p = win_rate
    # q = 1 - win_rate
    
    b = abs(avg_win / avg_loss)
    p = win_rate
    q = 1 - p
    
    kelly = (b * p - q) / b
    
    # 保守使用半凯利
    return max(0.05, min(0.25, kelly * 0.5))
```
