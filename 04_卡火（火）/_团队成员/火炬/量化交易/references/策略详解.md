## 一、7根K线策略

### 理论基础

> "人类从动荡到平稳的时间就是7天，经历过7次后情绪就会平复。突然中了彩票，7天后就没啥感觉了；员工受到鼓励，7天后也没感觉了。"  
> —— 卡若，2026年1月18日

### 核心逻辑

1. **强中强选股**：从强势板块中选择强势个股
2. **7天卖出**：买入后持有7个交易日无条件卖出
3. **轮询机制**：每天重新筛选最强的5只股票

### 选股条件

| 条件 | 要求 |
|:---|:---|
| 最近3天涨幅 | >5% |
| 今日涨幅 | >2% |
| 量比 | >1.5 |
| 流通盘 | <10亿 |
| 排除 | ST股、涨停、跌停 |

### 评分系统

```
综合评分 = 涨幅趋势×30% + 成交量×25% + 板块效应×20% + 资金流向×15% + 技术形态×10%
```

### 历史表现

- 年化收益率: 18.84%
- 胜率: 62.8%
- 最大回撤: -18%
- 夏普比率: 1.5

## 二、节假日策略

### 理论基础

历年来，春节、五一、国庆等节假日前后，大盘往往小幅上涨。原因：
1. 节假日前市场情绪乐观
2. 资金回流推动股价
3. 历史规律性强

### 五步选股法

#### 第一步：确定节假日时间

```
买入日期 = 节假日休市前2个交易日
卖出日期 = 节假日开市后第7个交易日
```

#### 第二步：筛选强势板块

统计过去5年该节假日期间各板块涨幅，选出：
- 涨幅前5的板块
- 出现次数>=3次
- 平均涨幅最高

#### 第三步：历史强势板块

**春节期间**
1. 医药生物（5次，平均8.5%）
2. 非银金融（5次，平均7.2%）
3. 电子信息（4次，平均9.1%）
4. 新能源（4次，平均10.3%）
5. 消费电子（3次，平均6.8%）

**五一期间**
1. 旅游酒店（5次，平均6.5%）
2. 餐饮（4次，平均5.8%）
3. 消费电子（4次，平均7.2%）
4. 医药生物（3次，平均5.5%）
5. 交通运输（3次，平均4.9%）

**国庆期间**
1. 旅游酒店（5次，平均7.8%）
2. 消费电子（5次，平均6.9%）
3. 白酒（4次，平均8.2%）
4. 新能源汽车（4次，平均9.5%）
5. 医药生物（3次，平均6.1%）

#### 第四步：挑选板块内优质个股

在板块内选择：
- 连续多年涨幅前5
- 出现次数>=3次
- 基本面良好

#### 第五步：分析大盘适配性

- 检查当年大盘整体走势
- 考虑经济环境和政策导向
- 综合评分>=70分才买入

### 策略参数

```yaml
buy_before_days: 2        # 节前N天买入
sell_after_days: 7        # 节后N天卖出
target_profit: 0.08       # 目标收益8%
stop_loss: -0.03          # 止损-3%
top_sectors: 5            # 选择前5个板块
top_stocks_per_sector: 3  # 每板块选3只
```

## 三、高手跟踪策略

### 理论基础

> "搜索全网所有高手的买点跟卖点，实时去观察"

### 高手识别模型

```
高手评分 = 历史收益率×0.4 + 发帖质量×0.3 + 粉丝数×0.2 + 活跃度×0.1
```

**高手标准**：
- 评分 >= 0.75
- 历史收益率 > 30%
- 胜率 > 55%
- 粉丝 > 1000

### 发帖质量评分

| 维度 | 权重 | 评分标准 |
|:---|:---|:---|
| 文本长度 | 30% | >1000字=30分, >500字=20分 |
| 专业术语 | 30% | 包含MACD/KDJ/RSI等>=5个 |
| 数据支撑 | 20% | 包含百分比数据>=3个 |
| 情感稳定 | 20% | 情感评分0.4-0.6最佳 |

### 执行流程

1. 每5秒查询高手新帖
2. 提取推荐股票代码
3. 判断买入/卖出（情感分析）
4. 生成跟单信号（延迟<1秒）
5. 执行风控（止损-5%）

## 四、风控规则

### 止损规则

| 类型 | 阈值 | 动作 |
|:---|:---|:---|
| 单股止损 | -5% | 立即平仓 |
| 单日止损 | -5% | 暂停交易1天 |
| 最大回撤 | -15% | 暂停交易，人工介入 |

### 仓位控制

| 规则 | 限制 |
|:---|:---|
| 最大持仓数 | 5只 |
| 单股仓位 | 最多25% |
| 同板块 | 最多40% |
| 保留现金 | 至少10% |

### 交易时机

**大涨市**
- 买入: 09:35（确认趋势后）
- 卖出: 09:30（高开冲高卖）

**平衡市**
- 买入: 09:45（观望后）
- 卖出: 09:35（观察后）

**大跌市**
- 买入: 14:45（尾盘抄底）
- 卖出: 09:45（等待反弹）

## 五、资金管理

### T+1循环模式

```
总资金: 10万
每天买入: 5万
每天卖出: 5万
效果: 资金不断滚动，充分利用
```

### 计算买入数量

```python
基础金额 = 日交易金额 / 最大持仓数  # 50000/5 = 10000

# 根据市场状态调整
if 大涨:
    amount = 基础金额 × 0.8  # 谨慎
elif 大跌:
    amount = 基础金额 × 1.2  # 大胆
else:
    amount = 基础金额

# 计算股数（取整100）
shares = int(amount / 股价 / 100) × 100
```
