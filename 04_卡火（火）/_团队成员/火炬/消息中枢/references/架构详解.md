## 核心组件

### 1. Gateway（网关）

**职责**：
- WebSocket 控制平面
- 会话管理
- 消息路由
- 工具调度
- 事件分发

**关键文件**：
```
src/gateway/
├── index.ts          # 入口
├── server.ts         # WebSocket 服务器
├── handlers/         # 消息处理器
├── routes/           # 路由配置
└── middleware/       # 中间件
```

### 2. Channels（通道）

**支持的通道**：

| 通道 | 协议/库 | 配置键 |
|:---|:---|:---|
| WhatsApp | Baileys | `channels.whatsapp` |
| Telegram | grammY | `channels.telegram` |
| Slack | Bolt | `channels.slack` |
| Discord | discord.js | `channels.discord` |
| Signal | signal-cli | `channels.signal` |
| iMessage | imsg | `channels.imessage` |
| Google Chat | Chat API | `channels.googlechat` |
| Microsoft Teams | Bot Framework | `channels.msteams` |

**通道抽象**：
```typescript
interface Channel {
  id: string;
  send(message: Message): Promise<void>;
  onMessage(handler: MessageHandler): void;
  connect(): Promise<void>;
  disconnect(): Promise<void>;
}
```

### 3. Sessions（会话）

**会话模型**：
- `main` - 主会话（直接对话）
- 群组隔离 - 每个群独立会话
- 激活模式 - mention/always

**会话存储**：
```
~/.clawdbot/sessions/
├── main.json
├── group-xxx.json
└── ...
```

### 4. Agent Runtime

**Pi Agent**：
- RPC 模式运行
- 工具流式传输
- 块流式传输

**工作区**：
```
~/clawd/
├── AGENTS.md      # Agent 指令
├── SOUL.md        # 人格设定
├── TOOLS.md       # 工具配置
└── skills/        # 技能目录
```

### 5. Tools（工具）

**内置工具**：

| 工具 | 功能 |
|:---|:---|
| `bash` | 执行 shell 命令 |
| `browser` | 浏览器控制 |
| `canvas` | 可视化工作区 |
| `discord` | Discord 操作 |
| `cron` | 定时任务 |
| `sessions_*` | 跨会话通信 |

### 6. Memory（记忆）

**嵌入存储**：
- SQLite + sqlite-vec 向量扩展
- OpenAI/Gemini 嵌入
- 批量处理

**搜索**：
- 语义搜索
- 混合搜索

## 数据流

### 1. 入站消息流

```
用户消息 → 通道适配器 → Gateway → 路由 → 会话 → Agent → 响应
```

### 2. 出站消息流

```
Agent 响应 → Gateway → 通道路由 → 通道适配器 → 用户
```

### 3. 工具调用流

```
Agent → 工具请求 → Gateway → 工具执行 → 结果 → Agent
```

## 安全架构

### 1. DM 配对策略

```
dmPolicy: "pairing"  # 默认
├── 未知发送者 → 配对码
├── 用户验证 → 加入白名单
└── 后续消息 → 正常处理
```

### 2. 沙箱模式

```
sandbox.mode: "non-main"
├── main 会话 → 宿主机执行
└── 其他会话 → Docker 沙箱
```

### 3. 工具权限

```
allowlist: ['bash', 'read', 'write', 'edit']
denylist: ['browser', 'canvas', 'nodes']
```

## 配置层级

```
优先级从高到低：
1. 环境变量
2. CLI 参数
3. ~/.clawdbot/moltbot.json
4. 默认值
```

## 扩展点

### 1. 自定义通道

```typescript
// extensions/my-channel/index.ts
export default {
  id: 'my-channel',
  connect: async () => { /* ... */ },
  send: async (msg) => { /* ... */ }
}
```

### 2. 自定义工具

```typescript
// 通过 TOOLS.md 注册
tools:
  - name: my_tool
    description: "..."
    script: scripts/my_tool.py
```

### 3. 自定义 Skills

```
skills/my-skill/
├── SKILL.md
├── scripts/
├── references/
└── assets/
```

## 性能考量

- **连接池**：复用 WebSocket 连接
- **消息队列**：异步处理高并发
- **会话裁剪**：自动压缩历史
- **媒体缓存**：临时文件生命周期管理
