## 🔗 联动指令

```
# 需求确定后，自动展开架构
@联动 需求→架构：基于 [需求文档.md] 生成系统架构

# 需求确定后，自动展开原型
@联动 需求→原型：基于 [用户故事] 生成页面结构

# 数据库确定后，生成后端Model
@联动 数据库→后端：基于 ER 图生成 Pydantic 模型

# 接口确定后，生成前后端代码
@联动 接口→前端：基于接口文档生成API调用代码
@联动 接口→后端：基于接口文档生成路由代码

# 需求变更后
@联动 需求→全量：基于新需求更新所有相关文档
```

## 📊 常用组合

### 新项目启动

```
1. @拆解需求 [一句话描述]
2. @成本估算
3. @技术选型
4. @架构图
5. @页面结构
```

### 功能开发

```
1. @生成接口 [功能名]
2. @生成页面 [页面名]
3. @生成路由 [接口名]
4. @生成集合 [数据名]
```

### 上线部署

```
1. @生成部署脚本
2. @生成Nginx
3. @生成手册
```
