## 使用方法

```python
import json
from 功能插件.宝塔部署器 import 宝塔部署器

# 读取配置
with open("部署配置/soul_部署配置.json", "r") as f:
    配置 = json.load(f)

# 执行部署
部署器 = 宝塔部署器()
结果 = 部署器.部署Node网站(配置)

print(结果)
```
