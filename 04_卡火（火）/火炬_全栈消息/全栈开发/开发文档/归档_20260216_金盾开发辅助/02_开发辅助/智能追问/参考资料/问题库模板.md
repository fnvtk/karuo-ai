## 三、快速追问模板

### 3.1 极简版（5分钟澄清）

```
1. 做什么？（一句话）
2. 给谁用？
3. 核心功能3个？
4. 必须有的技术约束？
5. 什么时候要？
```

### 3.2 标准版（15分钟澄清）

```
【目标】
1. 解决什么问题？
2. 主要用户是谁？
3. 成功标准是什么？

【边界】
4. 1.0必须有什么？
5. 明确不做什么？
6. 技术约束？

【异常】
7. 核心链路失败怎么办？
8. 需要日志/告警吗？

【验收】
9. 测试用例想到哪些？
10. 谁来验收？
```

### 3.3 完整版（30分钟澄清）

按SKILL.md中的五轮框架完整执行。
