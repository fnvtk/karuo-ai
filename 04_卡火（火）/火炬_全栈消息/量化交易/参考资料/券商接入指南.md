# 湘财证券量化交易接入指南

## 一、接入方式对比

| 方式 | 优点 | 缺点 | 推荐度 |
|:---|:---|:---|:---|
| **easytrader** | 简单易用，支持同花顺 | 基于GUI，速度慢 | ⭐⭐⭐⭐ |

## 二、easytrader接入（推荐）

### 1. 安装

```bash
pip install easytrader
```

### 2. 准备客户端

1. 下载湘财证券同花顺客户端
2. 安装并登录
3. 保持客户端运行

### 3. 连接代码

```python
import easytrader

# 使用通用同花顺客户端
user = easytrader.use('universal_client')

# 连接客户端（Windows）
user.connect(r'C:\同花顺\xiadan.exe')

# 或者连接已登录的客户端
user.connect(r'C:\同花顺\xiadan.exe', connect=True)
```

### 4. 基本操作

```python
# 查询资金
balance = user.balance
print(f"可用资金: {balance['可用资金']}")

# 查询持仓
position = user.position
for p in position:
    print(f"{p['证券代码']} {p['证券名称']}: {p['持仓数量']}股")

# 买入
user.buy('600519', price=1800, amount=100)

# 卖出
user.sell('600519', price=1850, amount=100)

# 撤单
user.cancel_entrust(entrust_no='12345')

# 今日成交
trades = user.today_trades

# 今日委托
entrusts = user.today_entrusts

# 一键打新
user.auto_ipo()
```

### 5. Mac用户解决方案

easytrader基于Windows GUI，Mac用户需要：

**方案A：远程Windows**
```python
# 在Windows服务器上运行easytrader服务
# Mac通过HTTP调用

import requests

def buy(stock_code, price, amount):
    response = requests.post('http://windows-server:5000/buy', json={
        'stock_code': stock_code,
        'price': price,
        'amount': amount
    })
    return response.json()
```

**方案B：虚拟机**
1. 安装Parallels Desktop或VMware
2. 创建Windows虚拟机
3. 在虚拟机中运行同花顺客户端

## 三、MiniQMT接入

### 1. 开通权限

联系湘财证券客户经理，申请MiniQMT量化交易权限。

### 2. 安装

```bash
pip install easytrader[miniqmt]
```

### 3. 配置客户端

1. 打开QMT客户端
2. 设置 → 系统设置 → 勾选"极简模式"或"独立交易模式"

### 4. 连接代码

```python
import easytrader

# 使用MiniQMT
user = easytrader.use('miniqmt')

# 连接
user.connect()

# 登录
user.login({
    'account': '你的资金账号',
    'password': '你的交易密码'
})
```

### 5. 交易操作

```python
# 限价买入
user.buy('600519', price=1800, amount=100)

# 市价买入
user.market_buy('600519', amount=100)

# 限价卖出
user.sell('600519', price=1850, amount=100)

# 市价卖出
user.market_sell('600519', amount=100)

# 查询委托
entrusts = user.query_entrust()

# 查询成交
trades = user.query_trade()

# 撤单
user.cancel_entrust(order_id='12345')
```

## 四、BigQuant平台（湘财官方）

### 1. 注册账号

访问 https://bigquant.xcsc.com 注册湘财证券BigQuant账号

### 2. API使用

```python
from bigquant import auth, trade

# 认证
auth.login('your_account', 'your_password')

# 下单
trade.order('600519', price=1800, amount=100, side='buy')

# 查询持仓
positions = trade.get_positions()
```

### 3. 注意事项

- 需要湘财证券实盘账户
- 部分功能需要付费订阅
- 在线平台，受网络影响

### 2. 异常处理

```python
import logging

logger = logging.getLogger(__name__)

def safe_buy(user, stock_code, price, amount):
    """安全买入"""
    try:
        result = user.buy(stock_code, price=price, amount=amount)
        logger.info(f'买入成功: {stock_code} {amount}股 @ {price}')
        return result
    except Exception as e:
        logger.error(f'买入失败: {e}')
        return None
```

### 3. 订单确认

```python
def buy_with_confirm(user, stock_code, price, amount):
    """买入并确认"""
    # 下单
    result = user.buy(stock_code, price=price, amount=amount)
    
    # 等待确认
    import time
    time.sleep(1)
    
    # 查询委托
    entrusts = user.today_entrusts
    
    # 找到对应委托
    for e in entrusts:
        if e['证券代码'] == stock_code:
            if e['委托状态'] == '已成':
                return {'success': True, 'result': e}
            elif e['委托状态'] == '废单':
                return {'success': False, 'error': '废单'}
    
    return {'success': False, 'error': '未知状态'}
```

### 4. 交易日检查

```python
from datetime import date
import chinese_calendar

def is_trading_day(d: date = None) -> bool:
    """检查是否是交易日"""
    if d is None:
        d = date.today()
    
    # 周末不交易
    if d.weekday() >= 5:
        return False
    
    # 节假日不交易
    if chinese_calendar.is_holiday(d):
        return False
    
    return True
```

## 六、故障排查

### 问题1：连接失败

```
错误: 无法连接到客户端
```

**解决**：
1. 确保同花顺客户端已打开并登录
2. 检查xiadan.exe路径是否正确
3. 以管理员身份运行Python

### 问题2：下单失败

```
错误: 委托失败，资金不足
```

**解决**：
1. 检查可用资金
2. 检查买入数量是否为100的倍数
3. 检查价格是否在涨跌停范围内

### 问题3：查询超时

```
错误: 查询超时
```

**解决**：
1. 检查网络连接
2. 减少查询频率
3. 添加重试机制

```python
import time

def query_with_retry(func, max_retries=3):
    for i in range(max_retries):
        try:
            return func()
        except Exception as e:
            if i < max_retries - 1:
                time.sleep(1)
            else:
                raise e
```
