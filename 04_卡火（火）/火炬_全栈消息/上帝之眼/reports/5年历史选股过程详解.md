## 📊 卡若六步选股法流程图（完整版）

```
第一步：统计5年历史板块数据
    │
    │   ✅ 涨幅统计：
    │   - 统计每年节假日期间涨幅前5的板块
    │   - 统计5年内出现次数
    │   - 选出出现次数最多的【2个板块】→ 买入区
    │
    │   ❌ 跌幅统计（禁区）：
    │   - 统计每年节假日期间跌幅前5的板块
    │   - 列出禁止买入的板块 → 禁区
    │
    ↓
第二步：板块内选出5年内出现次数最多的个股
    │
    │   ✅ 涨幅统计：
    │   - 统计每年该板块内涨幅前5的个股
    │   - 选出出现次数最多的【每板块3只候选】
    │   - ⚠️ 最终每板块最多选【2只】→ 买入区
    │
    │   ❌ 跌幅统计（禁区）：
    │   - 列出每年该板块内跌幅前5的个股
    │   - 禁止买入的个股列表 → 禁区
    │
    ↓
第三步：排除涨停股、跌停股（不排除ST）
    │
    ↓
第四步：综合评分系统（三维度）
    │
    │   📊 维度1：卡若均线理论（30分）
    │   - 天堂线-邪恶线-地狱线系统
    │   - 牛市排列+20分，金叉+10分...
    │
    │   📈 维度2：大盘买卖点判断（附加分）
    │   - 高开高走+10分，低开高走+5分
    │   - 高开低走-10分，低开低走-20分
    │
    │   📅 维度3：历史出现次数（30分）
    │   - 每次出现+6分，最高30分
    │
    │   🏆 维度4：高手跟踪加分（30分）【新增】
    │   - 检查模拟盘高手是否持有该股
    │   - 高手评分×权重，最高30分
    │
    ↓
第五步：板块均衡配置
    │
    │   ⚖️ 配置规则：
    │   - 最终选出【3只】股票
    │   - 每个板块【最多2只】
    │   - 确保板块分散，降低集中风险
    │
    ↓
第六步：确定买入时间节点
    │
    │   根据大盘开盘状态决定买入时间
```

## 🏆 高手跟踪策略详解（新增维度）

### 数据来源

| 平台 | 类型 | 数据内容 | 采集方式 |
|:---|:---|:---|:---|
| **雪球** | 模拟/实盘 | 组合持仓、调仓记录、收益曲线 | API/爬虫 |
| **同花顺** | 模拟盘 | 高手排行、买卖记录、持仓明细 | 爬虫 |
| **聚宽** | 量化平台 | 策略持仓、实盘跟单 | API |
| **米筐** | 量化平台 | 策略信号、因子值 | API |

### 高手识别模型

```python
# 高手评分算法
高手评分 = (
    历史收益率 × 0.4 +    # 赚钱能力（权重40%）
    胜率 × 0.3 +          # 判断准确（权重30%）
    回撤控制 × 0.2 +      # 风险管理（权重20%）
    活跃度 × 0.1          # 持续输出（权重10%）
)

# 筛选条件
筛选条件：
├─ 评分 >= 0.75（满分1.0）
├─ 历史收益率 > 30%
├─ 胜率 > 55%
└─ 最大回撤 < 20%
```

### 高手持仓加分规则

| 条件 | 加分 | 说明 |
|:---|:---|:---|
| 5位以上高手持有 | +30分 | 强烈推荐 |
| 3-4位高手持有 | +20分 | 推荐 |
| 1-2位高手持有 | +10分 | 参考 |
| 无高手持有 | +0分 | 无加分 |
| 高手近期卖出 | -10分 | 减分警告 |
