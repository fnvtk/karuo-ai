# 策略详解

## 一、强中强策略

### 理论基础

> "强中强的选择，就像在广东、浙江（最富裕的省）中选首富，明年出首富的概率最高"  
> —— 卡若

**核心思想**：从强势板块中选择强势个股，持有7天轮动

### 7天情绪周期理论

> "人类从动荡到平稳的时间就是7天。突然中了彩票，7天后就没啥感觉了；员工受到鼓励，7天后也没感觉了。"

**应用**：
- 股价动能在7天后减弱
- 持有7天无条件卖出
- 不断轮换，保持持有最强股票

### 选股五步法

```
第一步：筛选强势板块
    │
    ├── 按今日涨幅排序
    ├── 选择涨幅前5的板块
    └── 要求量比>1.5
    │
第二步：板块内选股
    │
    ├── 3日涨幅>5%
    ├── 今日涨幅>2%
    ├── 量比>1.5
    └── 流通盘<10亿
    │
第三步：排除条件
    │
    ├── 排除ST股
    ├── 排除涨停股（买不到）
    └── 排除跌停股
    │
第四步：综合评分
    │
    ├── 涨幅趋势 × 30%
    ├── 成交量 × 25%
    ├── 板块效应 × 20%
    ├── 资金流向 × 15%
    └── 技术形态 × 10%
    │
第五步：买入前5名
```

### 历史表现

- 年化收益率：18.84%
- 胜率：62.8%
- 最大回撤：-18%
- 夏普比率：1.5

## 二、模拟股策略（高手跟踪）

### 理论基础

> "搜索全网所有高手的买点跟卖点，实时去观察"

**核心思想**：跟踪各平台高手，学习他们的买卖点

### 数据源

| 平台 | 类型 | 数据内容 |
|:---|:---|:---|
| 雪球 | 模拟/实盘 | 组合持仓、调仓记录、收益曲线 |
| 同花顺 | 模拟盘 | 高手排行、买卖记录、持仓明细 |
| 聚宽 | 量化平台 | 策略持仓、实盘跟单 |
| 米筐 | 量化平台 | 策略信号、因子值 |

### 高手识别模型

```python
高手评分 = (
    历史收益率 × 0.4 +    # 赚钱能力
    胜率 × 0.3 +          # 判断准确
    回撤控制 × 0.2 +      # 风险管理
    活跃度 × 0.1          # 持续输出
)

筛选条件：
├─ 评分 >= 0.75
├─ 历史收益率 > 30%
├─ 胜率 > 55%
└─ 最大回撤 < 20%
```

### 跟单逻辑

```
监控高手动态
    │
    ├── 检测到调仓
    │       │
    │       ├── 买入操作
    │       │       │
    │       │       ├── 提取股票代码
    │       │       ├── 生成买入信号
    │       │       └── 风控检查 → 执行买入
    │       │
    │       └── 卖出操作
    │               │
    │               ├── 检查是否持有
    │               └── 生成卖出信号 → 执行卖出
    │
    └── 延迟控制 < 1秒
```

### 数据采集代码

```python
# 雪球组合爬取示例
import aiohttp
import re

class XueqiuCrawler:
    BASE_URL = "https://xueqiu.com"
    
    async def get_portfolio_changes(self, portfolio_id: str):
        """获取组合调仓记录"""
        url = f"{self.BASE_URL}/cubes/rebalancing/history.json"
        params = {
            'cube_symbol': portfolio_id,
            'count': 20
        }
        
        async with aiohttp.ClientSession() as session:
            async with session.get(url, params=params) as response:
                data = await response.json()
                
                changes = []
                for item in data.get('list', []):
                    changes.append({
                        'stock_code': item.get('target_code'),
                        'stock_name': item.get('target_name'),
                        'action': 'buy' if item.get('target_weight', 0) > item.get('prev_weight', 0) else 'sell',
                        'weight_change': item.get('target_weight', 0) - item.get('prev_weight', 0),
                        'created_at': item.get('created_at')
                    })
                
                return changes
```

## 三、板块分析策略

### 理论基础

板块轮动是A股的显著特征，资金在不同板块之间流动。

### 数据维度

| 维度 | 数据源 | 用途 |
|:---|:---|:---|
| 板块涨跌幅 | 东方财富 | 判断当日强弱 |
| 主力资金 | 同花顺 | 判断资金流向 |
| 成交量 | 交易所 | 判断活跃度 |
| 新闻情绪 | 财经网站 | 判断市场情绪 |

### 板块评分模型

```python
板块得分 = (
    今日涨幅 × 0.35 +      # 当日表现
    5日涨幅 × 0.25 +       # 短期趋势
    主力净流入 × 0.25 +    # 资金关注
    新闻情绪 × 0.15        # 市场情绪
)
```

### 轮动规律

**牛市轮动**：
```
券商 → 银行 → 地产 → 周期 → 消费 → 科技 → 医药
```

**熊市轮动**：
```
医药 → 消费 → 公用事业 → 黄金
```

## 四、节假日因子策略

### 理论基础

历年来，春节、五一、国庆等节假日前后，大盘往往小幅上涨。

### 原因分析

1. **节假日前**：市场情绪乐观
2. **节假日中**：资金回流准备
3. **节假日后**：开门红效应

### 历史数据（2018-2024）

| 节假日 | 平均涨幅 | 最佳板块 |
|:---|:---|:---|
| 春节 | 8% | 医药生物、非银金融、新能源 |
| 五一 | 5% | 旅游酒店、餐饮、消费电子 |
| 国庆 | 6% | 旅游酒店、白酒、新能源汽车 |

### 执行规则

```
节假日前2个交易日
    │
    ├── 选择历年该节假日涨幅前5的板块
    │       │
    │       └── 按出现次数排序（连续几年都强）
    │
    ├── 板块内选择连续多年上榜的个股
    │       │
    │       └── 出现次数 >= 3次
    │
    └── 买入前5只
            │
            ├── 目标收益：8%
            ├── 止损：-3%
            └── 持有至节后第7个交易日
```

## 五、策略组合使用

### 推荐组合

**保守型**：
- 节假日策略 + 强中强策略
- 预期年化：15-20%

**进取型**：
- 强中强策略 + 模拟股策略 + 板块分析
- 预期年化：25-35%

**全能型**：
- 四个策略全开
- 预期年化：20-30%

### 资金分配

```
总资金 = 100%
    │
    ├── 强中强策略：40%
    ├── 模拟股策略：30%
    ├── 节假日策略：20%
    └── 现金保留：10%
```
