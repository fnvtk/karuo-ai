# 上帝之眼 - 策略总结（卡若原创版）

> 基于5年历史数据实时分析，不是固定数据

---

## 核心策略：节假日因子（五步选股法）

### 卡若原创五步选股法

```
第一步：统计5年历史，选出出现次数最多的2个板块
    │
    │   计算方式：
    │   - 统计每年节假日期间（停盘日→复盘后第7根K线）
    │   - 取每年涨幅前5的板块
    │   - 统计5年内出现次数
    │   - 选出出现次数最多的【2个板块】
    │
    ↓
第二步：在板块内选出5年内出现次数最多的个股
    │
    │   同样方法：
    │   - 统计每年该板块内涨幅前5的个股
    │   - 选出出现次数最多的【每板块3只】
    │
    ↓
第三步：排除涨停股、跌停股（不排除ST）
    │
    ↓
第四步：卡若均线理论评分
    │
    │   天堂线-邪恶线-地狱线系统
    │   （详见下方说明）
    │
    ↓
第五步：买入前3名
```

### 与旧版的区别

| 项目 | 旧版 | 卡若原创版 |
|:---|:---|:---|
| 板块选择 | 按今日涨幅排序 | 按5年历史出现次数 |
| 板块数量 | 5个 | **2个** |
| 个股选择 | 按今日涨幅+3日涨幅+量比 | 按5年历史出现次数 |
| 每板块个股 | 不固定 | **3只** |
| ST股 | 排除 | **不排除** |
| 评分 | 多维度评分 | **卡若均线理论** |
| 买入数量 | 5只 | **3只** |

## 卡若均线系统（天堂-邪恶-地狱）

### 三条线定义

```python
天堂线（白线）= SMA((MA5 + MA10 + MA20) / 3, 5)
邪恶线（绿线）= SMA(MA60, 5)
地狱线（红线）= SMA((MA120 + MA240) / 2, 5)
```

### 评分规则

| 条件 | 得分 | 说明 |
|:---|:---|:---|
| 天堂 > 邪恶 > 地狱 | +40分 | 牛市排列 |
| 天堂 > 邪恶 | +30分 | 上涨趋势 |
| 天堂上穿邪恶 | +20分 | 金叉信号 |
| 价格 > 天堂线 | +10分 | 强势位置 |
| 历史出现次数 | 每次+10分 | 最高50分 |

**总分 = 均线评分(0-50) + 出现次数评分(0-50) = 0-100分**

### 买入信号

```
✅ 牛市排列：天堂 > 邪恶 > 地狱 → 强烈买入
✅ 金叉：天堂线上穿邪恶线 → 买入
✅ 价格站上天堂线 → 持有/加仓
```

### 卖出信号

```
❌ 死叉：天堂线下穿邪恶线 → 卖出
❌ 价格跌破邪恶线 → 减仓
❌ 熊市排列：天堂 < 邪恶 < 地狱 → 清仓
```

## 时间范围定义

### 节假日涨幅统计周期

```
起点：停盘前最后一个交易日（买入日）
终点：复盘后第7个交易日

例如2024年春节：
- 停盘前：2024-02-08
- 复盘日：2024-02-19
- 第7个K线：2024-02-27（约）

统计这期间的涨幅来排序
```

### 买入卖出时机

```
买入：停盘前2个交易日
卖出：复盘后第7个交易日
持有周期：约14天
```
