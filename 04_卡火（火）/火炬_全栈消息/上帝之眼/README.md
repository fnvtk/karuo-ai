## 📊 策略概览

### 卡若六步选股法

```
第一步：5年历史板块统计（涨幅+跌幅）→ 选2个板块 + 标记禁区
第二步：板块内5年个股统计 → 每板块3只候选 + 标记禁区
第三步：排除涨停/跌停（不排除ST）
第四步：四维度评分（均线+历史+高手+大盘）
第五步：板块均衡配置（每板块最多2只，共选3只）
第六步：确定买入时间节点（根据大盘开盘状态）
```

### 四维度评分

| 维度 | 分值 | 说明 |
|:---|:---|:---|
| 卡若均线理论 | 0-30分 | 天堂-邪恶-地狱系统 |
| 历史出现次数 | 0-30分 | 5年统计 |
| 高手跟踪加分 | 0-30分 | 全平台高手持仓 |
| 大盘买卖点 | -20~+10分 | 高开/低开判断 |
