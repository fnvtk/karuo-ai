## 一、量化使用的策略体系

### 1.1 策略分类（梁文锋本人表述）

梁文锋在访谈中明确区分了**四种股票策略**，并指出主战场与容量：

| 类型 | 说明 | 规模/地位 |
|------|------|-----------|
| **日间量价模型** | 多因子、Alpha 等常说的「选股+择时」 | 主战场，约 **2000 亿** 规模 |
| **日内回转（T0）** | 股票 T0 | 大几百亿 |
| **基本面模型** | 低频、大容量 | 目前非重点 |
| **事件驱动模型** | 事件驱动 | 目前非重点 |

**结论**：幻方主做**日间量价 + T0**；基本面与事件驱动为储备或次要。

### 1.2 方法论演进（时间线）

| 阶段 | 内容 |
|------|------|
| **2017 以前** | 传统**多因子**：选股+择时，模仿 WorldQuant 式「多人挖因子」，竞争谁因子更有效 |
| **2017 之后** | 多因子框架逐渐被**人工智能（深度学习）**取代 |
| **2019 之后** | 再被**更新的集成框架**取代，持续迭代 |
| **当前** | 几乎全部策略由 **AI 模型** 出信号；「基金经理 = 一堆服务器」 |

**他本人原话**：「区别不是交易，不是研究手段，而是**决策方式**——用数量化方法决策还是用人决策。程序决策是科学、有最优解；人做决策是艺术、要凭感觉。」

### 1.3 数据与模型框架（公开可推部分）

- **数据输入**（行业报道归纳）：行情（量价）、全市场基本面、结构化另类数据；累计 **10PB+** 级数据。
- **模型**：基于深度神经网络，对「特定周期内价格变化」建模；**策略工厂**每日生成大量组合策略，经**蒙特卡洛**等筛选最优解（报道中曾提「10 万种组合」级规模）。
- **因子**：从「人工挖因子」过渡到**机器/深度学习自动因子挖掘与信号挖掘**，迭代速度快。

### 1.4 产品线与收益目标

- **指数增强**：Long-only，赚 Alpha + Beta。
- **对冲/中性**：衍生品对冲，追求稳健 Alpha。
- **业绩预期**：梁文锋称「私募投资人期望高，一年跑赢指数低于 25% 会不满意」；幻方在百亿私募中近年收益排名靠前（近半年/一年/三年多进入前五量级）。

## 二、思维逻辑与框架

### 2.1 投资哲学（核心三句）

1. **「一定有办法对价格建模」**  
   引自西蒙斯，作为长期信念：价格可被数理与算法刻画，坚持做量化。

2. **「量化赚的是技术面流派原来赚的钱」**  
   人类投资者分技术面与基本面；当前量化主要替代的是**技术面**那部分；未来会进一步瓜分**基本面**流派的空间。

3. **「作为对冲基金，使命是提高中国二级市场的有效性」**  
   把量化视为市场效率的贡献者；效率提高后「直接买指数就是真正的价值投资」。

### 2.2 决策框架

- **决策方式 > 交易/研究手段**：核心区分是「程序决策」还是「人决策」。
- **科学 vs 艺术**：程序决策 = 科学、有最优解；人工决策 = 艺术、凭感觉。
- **人的角色**：仍需大量程序员与研究员，但**不做主观择时/选股**，而是做模型、数据、系统。

### 2.3 行业与终局判断

- **摩尔定律类比**：量化行业能力约每 18 个月翻一倍，但行业平均收益未必同步升，因为**市场有效性在提高**。
- **美国参照**：共同基金指数化 + 对冲基金量化化；桥水、AQR、文艺复兴等头部多为量化/宏观量化；**低频策略容量更大**（如宏观、股票基本面）。
- **未来**：技术面波动收敛后，量化会去瓜分**基本面**的钱；多策略结合、策略类型拓展是方向。

### 2.4 人才与管理逻辑

- **看能力不看经验**：核心岗位以应届或毕业一两年为主；无经验者会反复摸索、更易找到符合当前实际的解法。
- **无 KPI、无硬性任务**：交重要的事、不干预；第一年可能「做不出什么」，第二年才出成绩。
- **创新驱动**：自下而上、自然分工；每人可随时调用训练集群算力、灵活拉人协作；**热爱与好奇心**优先于短期商业回报。
- **销售逻辑**：不按「卖多少给提成」设计，而是鼓励发展圈子、建立信任，「让客户觉得靠谱」。

### 2.5 从量化到 AGI 的连贯逻辑

- **动机**：对 AI 能力边界的好奇；金融足够复杂，AGI 是下一个更难的事。
- **技术延续**：模型、数据、算力三者组合不变；从 1 张 GPU 到 100、1000、一万张，自建机房与萤火是自然延伸。
- **DeepSeek 定位**：独立于幻方，专注**基础模型与前沿研究**，不做垂类应用；短期不融资，问题在芯片禁运而非资金。
- **AGI 路线**：押注数学与代码、多模态、自然语言三条线；语言大模型为通往 AGI 的必经之路。

## 四、对上帝之眼 / 策略设计的可借鉴点（简要）

1. **策略分层**：明确日间量价 vs T0 vs 基本面/事件驱动的定位与资源分配。  
2. **决策方式**：坚持「信号与仓位由模型/程序出」，人负责研究与系统，不干预单笔决策。  
3. **数据与算力**：数据分层（行情/基本面/另类）、算力与训练管线可参考 3FS/萤火思路（不涉及具体策略复现）。  
4. **迭代节奏**：多因子 → 深度学习 → 集成框架的演进；持续做因子/信号挖掘与蒙特卡洛式组合筛选。  
5. **组织与创新**：小团队可借鉴「少干预、重能力与热情、自然分工」；不追求照搬无 KPI，但可取其「目标清晰、过程自主」的精神。
