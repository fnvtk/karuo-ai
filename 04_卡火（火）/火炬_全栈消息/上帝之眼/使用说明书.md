## 一、系统简介

### 什么是上帝之眼？

上帝之眼是一套A股量化交易系统，核心功能：

1. **数据采集**：从雪球、同花顺、东方财富等平台采集数据
2. **策略分析**：强中强、模拟股、板块分析、节假日因子
3. **自动交易**：生成信号 → 风控检查 → 自动下单
4. **高手跟踪**：监控各平台高手买卖点，智能跟单

### 2. 策略配置

创建 `config/strategy.yaml`：

```yaml
# ====================================
# 策略配置
# ====================================

# 启用的策略
enabled:
  - strong_pick       # 强中强
  - expert_track      # 模拟股
  - sector_rotate     # 板块分析
  - holiday_factor    # 节假日

# 强中强策略
strong_pick:
  hold_days: 7              # 持有天数
  stop_loss: -0.05          # 止损
  target_profit: 0.05       # 目标收益
  min_rise_rate: 0.05       # 3日最低涨幅
  min_volume_ratio: 1.5     # 最小量比
  max_capital: 10           # 最大流通盘（亿）

# 模拟股策略
expert_track:
  platforms:                # 数据源
    - xueqiu
    - ths
    - joinquant
  top_n: 10                 # 跟踪前N名
  min_score: 0.75           # 最低高手评分
  delay_threshold: 1.0      # 最大延迟（秒）

# 板块分析策略
sector_rotate:
  top_sectors: 5            # 关注前N个板块
  min_change_pct: 0.01      # 最小涨幅
  data_sources:
    - eastmoney
    - ths

# 节假日策略
holiday_factor:
  buy_before_days: 2        # 节前N天买入
  sell_after_days: 7        # 节后N天卖出
  stop_loss: -0.03          # 止损
  target_profit: 0.08       # 目标收益
```

## 四、策略模块

### 1. 节假日策略（卡若原创五步选股法）

**原理**：基于5年历史数据，选出节假日期间表现最好的板块和个股

**执行流程**：

```
第一步：统计5年历史数据
    │
    ├── 统计每年节假日期间（停盘日→复盘后第7根K线）
    ├── 取每年涨幅前5的板块
    ├── 统计出现次数，选出【2个板块】
    │
第二步：板块内选股
    │
    ├── 同样方法统计5年数据
    ├── 选出每板块出现次数最多的【3只个股】
    │
第三步：排除条件
    │
    ├── 排除涨停股
    ├── 排除跌停股
    └── 【不排除ST】
    │
第四步：卡若均线理论评分
    │
    ├── 天堂线-邪恶线-地狱线系统
    ├── 牛市排列+40分
    ├── 上涨趋势+30分
    ├── 金叉信号+20分
    ├── 价格在天堂线上方+10分
    └── 历史出现次数×10分（最高50分）
    │
第五步：买入前【3名】
```

**卡若均线系统**：
```python
天堂线 = SMA((MA5 + MA10 + MA20) / 3, 5)  # 白线
邪恶线 = SMA(MA60, 5)                      # 绿线
地狱线 = SMA((MA120 + MA240) / 2, 5)       # 红线
```

### 2. 模拟股策略

**原理**：跟踪各平台高手买卖点

**数据源**：

| 平台 | 数据内容 | 采集方式 |
|:---|:---|:---|
| 雪球组合 | 持仓、调仓、收益 | API/爬虫 |
| 同花顺模拟盘 | 排行榜、买卖点 | 爬虫 |
| 聚宽 | 策略持仓 | API |
| 米筐 | 策略信号 | API |

**高手筛选**：
```
评分 = 历史收益×0.4 + 胜率×0.3 + 回撤控制×0.2 + 活跃度×0.1

条件：
├─ 评分 >= 0.75
├─ 历史收益 > 30%
├─ 胜率 > 55%
└─ 最大回撤 < 20%
```

**跟单逻辑**：
1. 监控高手新调仓
2. 提取买入/卖出股票
3. 生成跟单信号
4. 风控检查
5. 执行交易

### 3. 板块分析策略

**原理**：分析板块轮动规律

**数据维度**：
- 板块涨跌幅
- 主力资金流向
- 成交量变化
- 板块新闻情绪

**输出**：
- 当日强势板块TOP5
- 板块轮动预测
- 龙头股推荐

### 4. 节假日策略

**原理**：历年节假日前后有规律性行情

**节假日时间表（2026）**：

| 节假日 | 买入日期 | 卖出日期 |
|:---|:---|:---|
| 春节 | 约2月6日 | 约2月20日 |
| 五一 | 约4月28日 | 约5月12日 |
| 国庆 | 约9月28日 | 约10月14日 |

**历史强势板块**：
- 春节：医药生物、非银金融、新能源
- 五一：旅游酒店、餐饮、消费
- 国庆：旅游酒店、白酒、新能源车

## 六、风险控制

### 1. 止损规则

| 类型 | 阈值 | 动作 |
|:---|:---|:---|
| 单股止损 | -5% | 立即平仓 |
| 单日止损 | -5% | 暂停交易1天 |
| 最大回撤 | -15% | 人工介入 |

### 2. 仓位控制

| 规则 | 限制 |
|:---|:---|
| 最大持仓数 | 5只 |
| 单股仓位 | 最多25% |
| 同板块 | 最多40% |
| 保留现金 | 至少10% |

### 3. 交易时间

| 时段 | 时间 | 操作建议 |
|:---|:---|:---|
| 开盘 | 09:30-10:00 | 谨慎，观察趋势 |
| 上午 | 10:00-11:30 | 最佳交易时段 |
| 下午 | 13:00-14:30 | 可以交易 |
| 尾盘 | 14:30-15:00 | 止盈止损为主 |

## 七、故障排查

### 问题1：连接券商失败

```
错误：无法连接到同花顺客户端
```

**解决**：
1. 确认同花顺客户端已打开并登录
2. 检查client_path路径是否正确
3. Mac用户需使用Windows虚拟机

### 问题2：数据采集失败

```
错误：获取雪球数据失败
```

**解决**：
1. 检查网络连接
2. 检查上帝之眼后端服务是否启动
3. 检查Cookie是否过期

### 问题3：下单失败

```
错误：委托失败，资金不足
```

**解决**：
1. 检查可用资金
2. 检查买入数量是否为100的整数倍
3. 检查是否超过仓位限制

### 问题4：策略无信号

```
现象：长时间没有交易信号
```

**解决**：
1. 检查策略配置是否正确
2. 检查是否在交易时间
3. 检查市场条件是否满足策略要求
