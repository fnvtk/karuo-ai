# 通道配置指南

## 配置文件位置

```
~/.clawdbot/moltbot.json
```

## WhatsApp

使用 Baileys 库实现。

### 登录

```bash
moltbot channels login
# 扫描二维码完成登录
```

### 配置

```json5
{
  "channels": {
    "whatsapp": {
      "allowFrom": ["+8613800138000"],  // 允许的发送者
      "groups": ["群组ID1", "群组ID2"],  // 群组白名单，包含 "*" 允许所有
      "dmPolicy": "pairing"  // pairing | open
    }
  }
}
```

### 凭证存储

```
~/.clawdbot/credentials/
```

## Telegram

使用 grammY 框架。

### 获取 Bot Token

1. 在 Telegram 中找 @BotFather
2. 发送 `/newbot`
3. 按提示创建机器人
4. 获得 Token

### 配置

```json5
{
  "channels": {
    "telegram": {
      "botToken": "123456:ABCDEF",  // 或设置环境变量 TELEGRAM_BOT_TOKEN
      "allowFrom": ["用户ID1"],
      "groups": {
        "*": {
          "requireMention": true  // 群组中需要 @机器人
        }
      }
    }
  }
}
```

## Slack

使用 Bolt SDK。

### 创建 Slack App

1. 访问 https://api.slack.com/apps
2. 创建新应用
3. 启用 Socket Mode
4. 添加 Bot Token Scopes
5. 安装到工作区

### 配置

```json5
{
  "channels": {
    "slack": {
      "botToken": "xoxb-...",  // 或 SLACK_BOT_TOKEN
      "appToken": "xapp-...",  // 或 SLACK_APP_TOKEN
      "dm": {
        "policy": "pairing",  // pairing | open
        "allowFrom": ["U01234567"]
      }
    }
  }
}
```

## Discord

使用 discord.js。

### 创建 Bot

1. 访问 https://discord.com/developers/applications
2. 创建应用
3. 创建 Bot
4. 获取 Token
5. 生成邀请链接并添加到服务器

### 配置

```json5
{
  "channels": {
    "discord": {
      "token": "YOUR_TOKEN",  // 或 DISCORD_BOT_TOKEN
      "dm": {
        "policy": "pairing",
        "allowFrom": ["用户ID"]
      },
      "guilds": {
        "服务器ID": {
          "channels": ["频道ID1", "频道ID2"]
        }
      },
      "mediaMaxMb": 25  // 媒体大小限制
    }
  }
}
```

### 命令注册

```json5
{
  "channels": {
    "discord": {
      "commands": {
        "native": true,   // 斜杠命令
        "text": true      // 文本命令
      }
    }
  }
}
```

## Signal

需要 signal-cli。

### 安装 signal-cli

```bash
# macOS
brew install signal-cli

# Linux
# 下载 releases 并配置 PATH
```

### 注册

```bash
signal-cli -u +1234567890 register
signal-cli -u +1234567890 verify CODE
```

### 配置

```json5
{
  "channels": {
    "signal": {
      "phone": "+1234567890",
      "allowFrom": ["+8613800138000"]
    }
  }
}
```

## iMessage

仅 macOS。需要 Messages.app 已登录。

### 配置

```json5
{
  "channels": {
    "imessage": {
      "allowFrom": ["+8613800138000", "email@example.com"],
      "groups": ["*"]  // 包含 "*" 允许所有群组
    }
  }
}
```

## Microsoft Teams

需要 Azure Bot Framework。

### 设置步骤

1. 创建 Azure Bot 资源
2. 配置 Teams App
3. 发布到组织

### 配置

```json5
{
  "channels": {
    "msteams": {
      "appId": "...",
      "appPassword": "...",
      "allowFrom": ["user@company.com"],
      "groupAllowFrom": ["*"],  // 或 groupPolicy: "open"
    }
  }
}
```

## Google Chat

使用 Chat API。

### 设置步骤

1. 创建 Google Cloud 项目
2. 启用 Chat API
3. 配置 OAuth 或服务账号
4. 发布 Chat App

```json5
{
  "channels": {
    "googlechat": {
      "credentials": "/path/to/service-account.json",
      "allowFrom": ["user@company.com"]
    }
  }
}
```

## DM 配对策略

### pairing（默认）

未知发送者收到配对码，需要验证：

```bash
moltbot pairing approve <channel> <code>
```

### open

允许所有人发送消息（需要显式配置）：

```json5
{
  "channels": {
    "whatsapp": {
      "dmPolicy": "open",
      "allowFrom": ["*"]  // 必须包含 "*"
    }
  }
}
```
