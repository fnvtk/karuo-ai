## 认证方式分类

| 认证方式 | 特点 | 适用场景 | 处理难度 |
|----------|------|----------|----------|
| Cookie/Session | 服务端维护状态 | 传统PHP/Java网站 | ⭐ |
| JWT Token | 无状态，自包含 | SPA应用、API服务 | ⭐⭐ |
| OAuth 2.0 | 第三方授权 | 开放平台 | ⭐⭐⭐ |
| API Key | 静态密钥 | 开发者API | ⭐ |
| Basic Auth | HTTP基本认证 | 内部系统 | ⭐ |
| WebSocket Auth | 长连接认证 | 实时应用 | ⭐⭐⭐ |
| 二维码登录 | 扫码确认 | 移动端关联 | ⭐⭐⭐⭐ |
| 短信验证码 | 动态验证 | 安全要求高 | ⭐⭐⭐⭐ |

## 1. Cookie/Session 认证

### 工作原理

```
用户 → 登录请求 → 服务器验证 → 创建Session → 返回SessionID(Cookie)
     ↑                                                    │
     └──────────────── 后续请求携带Cookie ←──────────────────┘
```

### 请求示例

```http
POST /api/login HTTP/1.1
Content-Type: application/x-www-form-urlencoded

username=admin&password=123456
```

### 响应示例

```http
HTTP/1.1 200 OK
Set-Cookie: PHPSESSID=abc123; Path=/; HttpOnly
Set-Cookie: remember_token=xyz789; Path=/; Max-Age=604800

{"code": 200, "message": "登录成功"}
```

### 处理策略

```python
import requests

session = requests.Session()

# 登录
response = session.post('https://example.com/login', data={
    'username': 'admin',
    'password': '123456'
})

# 后续请求自动携带Cookie
data = session.get('https://example.com/api/data')
```

## 2. JWT Token 认证

### 工作原理

```
用户 → 登录请求 → 服务器验证 → 生成JWT → 返回Token
     ↑                                      │
     └──── 后续请求Header携带Token ←─────────┘
```

### Token结构

```
Header.Payload.Signature

eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.
eyJzdWIiOiIxMjM0NTY3ODkwIiwibmFtZSI6IkpvaG4ifQ.
SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c
```

### 请求示例

```http
POST /api/auth/login HTTP/1.1
Content-Type: application/json

{"username": "admin", "password": "123456"}
```

### 响应示例

```json
{
    "code": 200,
    "data": {
        "access_token": "eyJhbGciOiJIUzI1NiIs...",
        "refresh_token": "eyJhbGciOiJIUzI1NiIs...",
        "expires_in": 3600,
        "token_type": "Bearer"
    }
}
```

### 处理策略

```python
import requests

# 登录获取Token
response = requests.post('https://example.com/api/auth/login', json={
    'username': 'admin',
    'password': '123456'
})
token = response.json()['data']['access_token']

# 后续请求携带Token
headers = {'Authorization': f'Bearer {token}'}
data = requests.get('https://example.com/api/data', headers=headers)
```

## 3. OAuth 2.0 认证

### 授权模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| Authorization Code | 授权码模式 | Web应用 |
| Password | 密码模式 | 受信任客户端 |
| Client Credentials | 客户端模式 | 服务器间调用 |
| Implicit | 隐式模式 | 单页应用(已不推荐) |

### Password模式示例

```http
POST /oauth/token HTTP/1.1
Content-Type: application/x-www-form-urlencoded

grant_type=password&
username=admin&
password=123456&
client_id=my_client&
client_secret=my_secret&
scope=read write
```

### 响应示例

```json
{
    "access_token": "2YotnFZFEjr1zCsicMWpAA",
    "token_type": "Bearer",
    "expires_in": 3600,
    "refresh_token": "tGzv3JOkF0XG5Qx2TlKWIA",
    "scope": "read write"
}
```

### 处理策略

```python
from requests_oauthlib import OAuth2Session

# 创建OAuth会话
client_id = 'my_client'
client_secret = 'my_secret'
token_url = 'https://example.com/oauth/token'

# Password模式获取Token
oauth = OAuth2Session(client=BackendApplicationClient(client_id=client_id))
token = oauth.fetch_token(
    token_url=token_url,
    username='admin',
    password='123456',
    client_id=client_id,
    client_secret=client_secret
)

# 后续请求
response = oauth.get('https://example.com/api/data')
```

## 4. WebSocket 认证

### 工作原理（参考存客宝）

```
1. HTTP请求获取Token
2. 建立WebSocket连接
3. 发送认证消息
4. 维持心跳
```

### 认证流程

```python
import websockets
import json

# 1. 获取Token
token = await get_token(username, password)

# 2. 建立WebSocket连接
async with websockets.connect('wss://example.com/ws') as ws:
    
    # 3. 发送认证消息
    auth_msg = {
        "type": "auth",
        "token": token
    }
    await ws.send(json.dumps(auth_msg))
    
    # 4. 接收认证结果
    response = await ws.recv()
    
    # 5. 心跳保活
    while True:
        await ws.send(json.dumps({"type": "ping"}))
        await asyncio.sleep(30)
```

## 5. 二维码登录

### 工作原理

```
1. 获取二维码（含临时ticket）
2. 展示二维码
3. 轮询登录状态
4. 用户扫码确认
5. 返回Token
```

### 处理策略

```python
import qrcode
import asyncio

async def qr_login():
    # 1. 获取二维码
    qr_data = await get_qr_code()
    ticket = qr_data['ticket']
    qr_url = qr_data['url']
    
    # 2. 显示二维码
    qr = qrcode.make(qr_url)
    qr.show()
    
    # 3. 轮询状态
    while True:
        status = await check_scan_status(ticket)
        
        if status['code'] == 'scanned':
            print("已扫码，等待确认...")
        elif status['code'] == 'confirmed':
            return status['token']
        elif status['code'] == 'expired':
            raise Exception("二维码已过期")
        
        await asyncio.sleep(2)
```

## 自动识别认证类型

```python
def detect_auth_type(login_response):
    """
    自动检测认证类型
    """
    headers = login_response.headers
    
    # 检查Cookie
    if 'Set-Cookie' in headers:
        cookies = headers.get('Set-Cookie', '')
        if 'session' in cookies.lower() or 'sid' in cookies.lower():
            return 'cookie_session'
    
    # 检查响应体
    try:
        data = login_response.json()
        
        # JWT特征
        token_fields = ['token', 'access_token', 'jwt', 'accessToken']
        for field in token_fields:
            if field in str(data):
                value = find_nested_value(data, field)
                if value and '.' in value and len(value.split('.')) == 3:
                    return 'jwt'
                elif value:
                    return 'token'
        
        # OAuth特征
        if 'token_type' in data and 'expires_in' in data:
            return 'oauth2'
            
    except:
        pass
    
    return 'unknown'
```

## 最佳实践

1. **优先使用官方API**：如果网站提供开放API，优先使用
2. **尊重robots.txt**：遵守网站的爬虫规则
3. **控制请求频率**：避免对服务器造成压力
4. **妥善保管凭证**：Token和密码不要硬编码
5. **处理Token过期**：实现自动刷新机制
6. **异常处理**：处理网络错误、认证失败等情况
