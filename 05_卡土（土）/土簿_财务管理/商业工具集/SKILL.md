# 商业工具集

> 一句话说明：基于 MCP 商业工具进行竞品分析、ROI 计算、市场调研与商业可行性评估。

---

## 触发条件

用户说以下关键词时自动激活：
- 商业分析、竞品分析
- MCP商业、商业工具
- ROI分析、市场调研
- "这个项目能赚钱吗"、"先算账"

---

## 执行步骤

### 1. 需求理解
- 明确分析目标（竞品/市场/ROI/可行性）
- 确认行业与目标客群

### 2. 数据采集
- 使用 MCP 商业工具采集公开数据
- 竞品信息整理（产品、定价、流量、口碑）

### 3. 分析建模
- ROI 计算：CAC、LTV、回款周期、毛利结构
- 市场规模估算
- 竞争格局分析

### 4. 报告输出
- 结论 + 建议 + 风险提示
- 可视化图表（如适用）

---

## 输出格式

```
[商业工具集] 分析完成
├─ 目标：xxx
├─ 结论：xxx
├─ 关键数据：CAC=xx, LTV=xx, ROI=xx
├─ 建议：xxx
└─ 风险：xxx
```
