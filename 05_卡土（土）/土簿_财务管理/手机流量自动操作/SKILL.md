# 手机流量自动操作

> 一句话说明：通过 AutoGLM 等工具实现手机端自动化操作，包括流量获取、内容分发、自动互动等。

---

## 触发条件

用户说以下关键词时自动激活：
- 手机自动化、AutoGLM
- 手机刷量、手机流量
- 自动操作手机
- 抖音自动、微信自动

---

## 执行步骤

### 1. 环境检查
- 确认手机连接状态（USB/WiFi ADB）
- 检查 AutoGLM 或自动化工具是否就绪
- 确认目标平台（抖音/微信/小红书等）

### 2. 任务配置
- 定义自动化任务（浏览/点赞/评论/关注/私信）
- 设置频率与时间间隔（避免风控）
- 配置目标账号/内容/关键词

### 3. 执行与监控
- 启动自动化任务
- 实时监控执行状态
- 异常检测与自动暂停

### 4. 数据统计
- 记录执行结果（成功/失败/跳过）
- 生成任务报告

### 一键命令

```bash
# 环境检查
bash scripts/check_env.sh

# 启动自动化
python scripts/auto_flow.py --platform douyin --task browse --count 100
```

## 安全原则

- 操作频率需控制在平台安全线内，避免封号
- 敏感操作（如私信、转账）需二次确认
- 不用于违法违规操作
