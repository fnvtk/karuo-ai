## 全链路步骤（HutaoCards · 已验证 2026-03-18）

### 第 1 步：注册账号

1. 打开 `https://hutaocards.com/`
2. 点击「注册」，输入邮箱 `zhiqun@qq.com`
3. 获取邮箱验证码（用 QQ 邮箱 IMAP 自动拉取：`02_卡人（水）/水桥_平台对接/QQ邮箱拉取/qq_mail_fetch.py`）
4. 设置密码，完成注册

### 第 2 步：充值（支付宝）

1. 登录后在首页点「充值」
2. 输入充值金额（最低 $25 起）
3. 选择「支付宝」支付方式
4. 扫描二维码或通过 ADB 推送支付链接到手机支付宝
5. 确认支付，通常即时到账

**ADB 自动支付流程**（可选，需连接工作手机）：
- 项目路径：`/Users/karuo/Documents/开发/2、私域银行/工作手机`
- 从 HutaoCards 支付页面解码 QR → 提取支付 URL → `adb shell am start -a android.intent.action.VIEW -d <URL>`
- 在手机支付宝确认支付

### 第 3 步：申请虚拟卡

1. 点「Card」→「申请卡片」
2. 选「合作卡」tab
3. 选择卡片 BIN（推荐 428852 用于 AI 平台）
4. 弹窗选「非实名开卡」（$12，无需 KYC）或「实名开卡」（$8，需 KYC）
5. 点「下一步」
6. 填写邮箱地址、充值金额（最低 $10）
7. 点「确认开卡」→ 等 5 秒 → 点「我已阅读，继续开卡」
8. 确认弹窗 → 点「确认开卡」
9. 邮箱验证码（6 位）→ 逐个填入输入框 → 点「确认提交」
10. 等待处理完成，卡片即时生效

### 第 4 步：使用虚拟卡

- **订阅 Cursor**：Settings → Billing → 输入卡号/有效期/CVV
- **订阅 ChatGPT Plus**：Settings → Subscription → 输入卡片信息
- **订阅 Claude Pro**：Settings → Billing → 输入卡片信息
- **绑定 GitHub Copilot**：Billing → Add payment method

### 第 5 步：充值与管理

- 「Card」页面查看余额
- 「充值」按钮追加金额
- 「详情」查看卡号/CVV
- 「冻结」临时冻结卡片
- 「销卡」永久注销
