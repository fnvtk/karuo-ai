## 二、分配路由（谁管谁、谁分配谁）

```
卡若AI（大总管）
    │
    ├─ 分配给 卡资（金）→ 卡资 再分配给 金仓、金盾
    ├─ 分配给 卡人（水）→ 卡人 再分配给 水溪、水泉、水桥
    ├─ 分配给 卡木（木）→ 卡木 再分配给 木叶、木根、木果
    ├─ 分配给 卡火（火）→ 卡火 再分配给 火炬、火锤、火眼、火种
    └─ 分配给 卡土（土）→ 卡土 再分配给 土基、土砖、土渠、土簿
```

**路由规则**：用户任务 → 触发词匹配 → 确定负责人 → 负责人对应成员 → 读该成员下 SKILL.md 执行。多技能按 金→水→木→火→土 优先级。用户可 `@卡资` `@金仓` 等指定。
