## 三、分销机制（知己实现）

- **规则**：一级 20%、二级 10%；30 天绑定；7 天后结算；最低提现 10 元。
- **核心**：`lib/distribution/service.ts`（申请分销员、绑定推荐、佣金计算、提现）。
- **空中分账**：`lib/profit-sharing/service.ts`（按商品类型分账：咨询师 60%、平台 40% 等）。
- **API**：`app/api/distribution/`、`app/api/referral/bind/`。
- **类似项目**：复制 distribution + profit-sharing 目录，改比例与集合名即可。

## 五、AI 向量化 / RAG（知己实现）

- **RAG 检索引擎**：`lib/rag.ts`  
  - OpenAI Embeddings（text-embedding-3-small，1536 维）  
  - MongoDB 存储向量；余弦相似度检索；未配置 Key 时用占位向量（开发可跑通）
- **知识库管理**：`app/admin/knowledge/`（导入、向量化、检索）。  
- **向量化脚本**：`scripts/vectorize-knowledge.mjs`（批量生成 embedding 写回库）。  
- **类似项目**：复制 rag.ts + 知识库 API + vectorize 脚本，改集合名与分类即可。

## 七、点击埋点与数据统计（Soul 创业实验，可复用到任意站点）

- **目标**：任意 Web / 小程序 / App 在迭代时，**可点击元素与业务标签**均映射为可聚合事件，支撑看板与运营决策。
- **参考仓库**：`一场soul的创业实验-永平`（小程序 `miniprogram/utils/trackClick.js`；接口 `POST /api/miniprogram/track`；表 `user_tracks`；管理端统计与 module/action/target 对齐）。
- **命名习惯**：`trackClick(module, action, target, extra)`；`extra` 内固定带 `module`、`page`，`target` 需能区分 Tab、分享、购买等不同业务。
- **文档落盘**：新站建议在 `开发文档/10、项目管理/` 维护 `埋点锚点登记表.md`（或实施计划中单列锚点表），与后台图表维度一致。
- **权威 Skill**：`04_卡火（火）/火炬_全栈消息/全栈开发/SKILL.md` **§1.10**；前端执行要点见同目录 `前端开发/SKILL.md` **§五**。
