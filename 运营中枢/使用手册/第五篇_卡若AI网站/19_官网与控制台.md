## 19.1 官网页面


| 路径              | 页面   | 功能                                    |
| --------------- | ---- | ------------------------------------- |
| `/`             | 首页   | 居中对话区 + 底部输入框，直接与卡若AI 对话；可展开团队与执行流程面板 |
| `/skills`       | 能力一览 | 技能卡片画廊，按金/水/木/火/土筛选 + 搜索              |
| `/about`        | 关于   | 五行团队卡片 + 统计数据 + 可实现范围                 |
| `/usage`        | 使用指南 | 7 步垂直时间线：登录→配置→编排→文档→任务→网关→闭环         |
| `/faq`          | 常见问题 | 手风琴式 FAQ 展开                           |
| `/contact`      | 联系   | 表单 + 系统状态卡片                           |
| `/chat`         | 对话   | 完整对话界面，v0 风格，快捷 prompt 气泡             |
| `/gene-capsule` | 基因胶囊 | 概念说明 + 已上线胶囊下载列表                      |


## 19.2 控制台页面


| 路径                   | 页面     | 功能                          |
| -------------------- | ------ | --------------------------- |
| `/console`           | 总览     | 4 张统计卡 + 7 模块入口 + 最近 5 条任务  |
| `/console/login`     | 登录     | 居中卡片 + 渐变 Logo              |
| `/console/skills`    | 技能配置   | 列表/分类、编辑、AI 整理、AI 新增、从主仓库同步 |
| `/console/workflows` | 流程编排   | 步骤可视化 + 一键执行                |
| `/console/tasks`     | 任务控制台  | 状态色点 + 复盘记录                 |
| `/console/docs`      | 文档操控   | 调研/生成/更新按钮                  |
| `/console/gateway`   | API 网关 | 主备网关区分 + 一键切换               |
| `/console/users`     | 账号权限   | 角色 badge + inline 编辑        |
| `/console/platform`  | 平台治理   | 接口开关 + 限流 + 基因胶囊配置          |


## 19.3 官网与控制台的区别


| 维度  | 官网       | 控制台       |
| --- | -------- | --------- |
| 受众  | 所有访客     | 管理员       |
| 权限  | 无需登录（只读） | 需登录（读写）   |
| 接口  | 只调只读 API | 可调所有 CRUD |
| 作用  | 展示、说明、入口 | 配置、管理、运维  |
