# 卡若AI 使用手册

> **版本**：2.0 | **更新**：2026-03-12
> **维护者**：卡若AI（大总管）
> **说明**：本手册以「书」的形式组织，每篇一个文件夹，每章一个 .md 文件。新增功能或优化后按章节就近更新。

---

## 序

卡若AI 是卡若的个人数字管家与智能大总管。它不是一个单独的 App，而是一套**身份 + 团队 + 技能 + 规则**的体系，可以加载到任何 AI 平台（Cursor、Claude、ChatGPT、API 等）中运行。

本手册是卡若AI 的完整使用指南——从认识它的架构，到掌握 69 个技能的使用方法，再到卡若AI 网站的全部功能，以及进阶的外网调用、并行处理、技能开发等。

## 如何使用本手册

- **新用户**：从第一篇开始顺序阅读，了解架构后看第二篇快速上手
- **查技能**：直接翻第三篇对应组别，或查附录 A 速查表
- **查规范**：翻第四篇运营中枢
- **查网站功能**：翻第五篇
- **开发者/进阶**：翻第六篇

## 手册维护规则
