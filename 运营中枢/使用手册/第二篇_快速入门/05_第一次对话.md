## 5.1 对话流程

每次对话遵循固定流程（强制）：

```
输入需求
  ↓
深度思考（结合五行全部能力）
  ↓
在对话中展示：思考结果 + 任务拆解 + 执行计划
  ↓
执行（可 1～6 线程并行）
  ↓
反复验证（至少两轮）
  ↓
交付 + 强制复盘（🎯📌💡📝▶）
```

## 5.2 触发词使用

直接用自然语言描述需求即可，卡若AI 会自动匹配技能：

| 你说 | 卡若AI 做 |
|:---|:---|
| "帮我写今天的飞书日志" | 匹配 W07 飞书管理 → 执行写入脚本 |
| "这个 Bug 帮我修一下" | 匹配 F05 代码修复 → 定位并修复 |
| "帮我把视频切片发到各平台" | 匹配 M01h 多平台分发 → 一键分发 |
| "分析一下这个竞品" | 匹配 E01 商业工具集 → 竞品分析 |
| "帮我找一个 MCP 服务" | 匹配 W12 MCP 搜索 → 搜索并安装 |

也可以用 `@成员名` 指定：如 `@火炬 帮我做全栈开发`。

## 5.3 示例对话

**用户**：帮我整理一下桌面上的文件

**卡若AI**：
1. **思考**：文件整理属于水组（卡人）→ 水溪（整理归档）→ W01 文件整理
2. **拆解**：① 扫描桌面文件 ② 按类型分类 ③ 移动到对应目录 ④ 生成清单
3. **执行**：按步骤操作
4. **复盘**：🎯 目标·结果·达成率 → 📌 过程 → 💡 反思 → 📝 总结 → ▶ 下一步

## 5.4 对话结束时

每次对话收尾会自动执行：
- **复盘**：完整五块复盘
- **经验沉淀**：有价值的写入经验库
- **Gitea 同步**：有文件变更时自动推送
- **手册更新**：有技能变更时更新对应章节
