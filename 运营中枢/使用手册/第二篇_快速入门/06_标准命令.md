## 6.1 所有平台通用命令


| 命令           | 触发方式     | 做什么                                  |
| ------------ | -------- | ------------------------------------ |
| **技能查找**     | 用户说任何需求  | 查 SKILL_REGISTRY → 读对应 SKILL.md → 执行 |
| **复盘**       | 所有对话强制   | 每轮/每步结尾输出完整复盘块（🎯📌💡📝▶）            |
| **沉淀**       | 发现有价值经验  | 写入 `水溪_整理归档/经验库/待沉淀/`                |
| **基因胶囊**     | 打包/解包/继承 | `gene_capsule.py pack/unpack/list`   |
| **Gitea 同步** | 有文件变更    | `bash 金仓/Gitea管理/脚本/自动同步.sh`         |
| **Pipeline** | "新项目上线"等 | 按 Pipeline 执行清单逐步执行                  |
| **对外输出**     | 报告/图片/导出 | 只写 `卡若Ai的文件夹/` 对应子目录                 |


## 6.2 Cursor 特有命令


| 命令     | 说明                                    |
| ------ | ------------------------------------- |
| 每日对话收集 | 每天首次对话时执行 `collect_chat_daily.py`     |
| 飞书今日日志 | 直接执行 `write_today_three_focus.py`，不询问 |
| MD 预览  | 所有 .md 用 Markdown Preview Enhanced 打开 |
| 终端命令   | AI 直接执行，不输出让用户复制                      |


## 6.3 常规操作原则

- **优先命令行**：能用命令完成的不手动
- **先搜后做**：执行前搜索是否有现成脚本/流程
- **不重复造轮子**：有现成的直接用
- **不反复提问**：能判断的自动执行，不问"是否执行"
