## 9.1 职责概述

木组负责内容生产与产品模板：视频切片与多平台分发、网站逆向、项目脚手架、PPT、架构图、个人介绍等。

## 9.2 木叶 · 视频内容（8 技能）

| 编号 | 技能 | 触发词 | 做什么 |
|:--|:---|:---|:---|
| M01 | 视频切片 | 视频剪辑、切片发布、片头片尾 | 长视频→切片+字幕+动效包装 |
| M01b | 抖音视频解析 | 抖音链接、抖音下载 | 链接→解析→文案+无水印下载 |
| M01c | 抖音发布 | 抖音发布、抖音上传 | 纯 API 视频上传发布 |
| M01d | B站发布 | B站发布、bilibili发布 | API 分片上传发布 |
| M01e | 视频号发布 | 视频号发布、微信视频号 | 腾讯云 API 上传发布 |
| M01f | 小红书发布 | 小红书发布、RED发布 | 逆向 creator API 发布 |
| M01g | 快手发布 | 快手发布、kuaishou发布 | 逆向 cp.kuaishou.com 发布 |
| M01h | 多平台分发 | 一键分发、全平台发布 | 一键分发到 5 平台，支持定时排期 |

## 9.3 木根 · 逆向分析（1 技能）

| 编号 | 技能 | 触发词 | 做什么 |
|:--|:---|:---|:---|
| M02 | 网站逆向分析 | 逆向分析、模拟登录 | 网站 API 分析与 SDK 生成 |

## 9.4 木果 · 项目模板（7 技能）

| 编号 | 技能 | 触发词 | 做什么 |
|:--|:---|:---|:---|
| M03 | 项目生成 | 生成项目、五行模板 | 按五行模板生成新项目 |

| M05 | 个人档案生成器 | 个人档案、档案生成 | 自动生成个人介绍档案 |

## 9.5 使用场景

- "帮我把这个长视频切片并发到全平台" → M01 + M01h
- "帮我做一份汇报 PPT" → M07 PPT制作
- "画一张系统架构图" → M08 Next AI Draw
- "分析一下这个网站的登录接口" → M02 网站逆向
