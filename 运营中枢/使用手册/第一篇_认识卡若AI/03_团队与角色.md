## 3.1 团队全景

```
卡若AI（大总管）
├── 卡资（金）── 金仓（存储备份）、金盾（数据安全）
├── 卡人（水）── 水溪（整理归档）、水泉（规划拆解）、水桥（平台对接）
├── 卡木（木）── 木叶（视频内容）、木根（逆向分析）、木果（项目模板）
├── 卡火（火）── 火炬（全栈消息）、火锤（代码修复）、火眼（智能追问）、火种（知识模型）
└── 卡土（土）── 土基（商业分析）、土砖（技能复制）、土渠（流量招商）、土簿（财务管理）
```

**统计**：5 负责人 → 14 成员 → 69 技能

## 3.2 各组职责

### 金组 · 卡资 — 基础设施守护

保证底层稳定：NAS、服务器、Gitea、备份、数据库、系统监控、设备管理。

- **金仓**（存储备份）：NAS 管理、Gitea、磁盘清理、容灾备份、照片分类、算力管控、服务器、系统监控、iPhone/iCloud/局域网/微信/数据库/文件整理（14 技能）
- **金盾**（数据安全）：存客宝、远程部署、数据库安全、微信安全、Vercel 部署、个人档案（7 技能）

### 水组 · 卡人 — 信息流程调度

管理信息流转：归档、规划、飞书、MCP、运营报表、短信。

- **水溪**（整理归档）：文件整理、文档清洗、对话归档、项目调研、自动记忆管理（5 技能）
- **水泉**（规划拆解）：需求拆解、任务规划（2 技能）
- **水桥**（平台对接）：飞书管理、智能纪要、小程序、Soul 创业实验、运营报表、MCP、Excel 日报、卡猫复盘、接收短信（9 技能）

### 木组 · 卡木 — 产品内容创造

负责内容生产：视频、逆向、模板、PPT、画图。

- **木叶**（视频内容）：视频切片、抖音解析、抖音/B站/视频号/小红书/快手发布、多平台分发（8 技能）
- **木根**（逆向分析）：网站逆向分析（1 技能）
- **木果**（项目模板）：项目生成、开发模板、个人档案、PPT 制作、AI Draw、卡若个人介绍、前端生成（7 技能）

### 火组 · 卡火 — 技术研发优化

技术核心：全栈、修复、追问、知识模型。

- **火炬**（全栈消息）：全栈开发、前端开发、全栈测试、消息中枢、读书笔记、文档清洗、开发文档操控、上帝之眼、量化交易、对话归档、本地项目启动（11 技能）
- **火锤**（代码修复）：代码修复（1 技能）
- **火眼**（智能追问）：智能追问（1 技能）
- **火种**（知识模型）：读书笔记、本地模型、本地代码库索引、卡若日记写作（4 技能）

### 土组 · 卡土 — 商业复制裂变

管商业与财务：分析、复制、流量、记账。

- **土基**（商业分析）：商业工具集（1 技能）
- **土砖**（技能复制）：技能工厂、全站捆绑分销体系、基因胶囊（3 技能）
- **土渠**（流量招商）：流量自动化、手机流量自动操作（2 技能）
- **土簿**（财务管理）：财务管理、公司财务、家庭财务、商业工具集（4 技能）

## 3.3 分配规则

1. 用户说需求 → 大总管按关键词匹配技能
2. 按 金→水→木→火→土 优先级分配
3. 负责人派给对应成员 → 成员读 SKILL.md 执行
4. 用户可用 `@成员名` 指定（如 `@火炬 全栈开发`）
