# 飞书复盘总结发群（对话结束后）

> **强制**：每次对话完成、复盘写完后，将**简洁复盘总结**发到指定飞书群。长对话必发，每次完成的任务都发。  
> 更新：2026-03-12

---

## 一、规则

- **时机**：对话结束、完成「卡若复盘」后。
- **内容**：简洁版复盘总结（建议 ≤500 字），包含：日期时间、目标·结果·达成率、完成的任务要点、下一步（若有）。
- **对象**：飞书群（通过 bot v2 webhook 推送）。
- **频率**：每次对话完成都发；**长对话尤其必须发**。

---

## 二、脚本与 webhook

| 项 | 说明 |
|:---|:---|
| **脚本** | `02_卡人（水）/水桥_平台对接/飞书管理/脚本/send_review_to_feishu_webhook.py` |
| **默认 webhook** | 脚本内已配置默认群 webhook；可用环境变量 `FEISHU_REVIEW_WEBHOOK` 覆盖。 |
| **调用方式** | 见下方「使用方式」。 |

---

## 三、使用方式

```bash
# 直接传入简洁总结
python3 "02_卡人（水）/水桥_平台对接/飞书管理/脚本/send_review_to_feishu_webhook.py" "【卡若AI复盘】YYYY-MM-DD HH:mm
🎯 目标·结果·达成率（一行）
📌 完成的任务要点 1、2、3
▶ 下一步（或无）"

# 从文件读
python3 .../send_review_to_feishu_webhook.py --file /path/to/summary.txt
```
