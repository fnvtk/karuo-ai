# 投屏后默认使用外接设备的声音和麦克风

## 问题

MacBook 扩展屏投屏后，系统仍默认使用 **MacBook 的扬声器和麦克风**，需要每次手动改。

## 办法一：每次投屏后手动改（不装任何东西）

1. 投屏连接好后，点 **控制中心**（菜单栏右上角）
2. 在 **「声音」** 模块里，点 **扬声器图标右侧的 ⭕（AirPlay/输出）**
3. 在列表里选你的 **投屏设备**（如 Apple TV、外接显示器）
4. 若要改用外接设备的麦克风：**系统设置 → 声音 → 输入** → 选对应设备

## 办法二：一键脚本（需先安装小工具）

### 1. 安装命令行切换工具（一次性）

```bash
brew install switchaudio-osx
```

### 2. 投屏后运行脚本

```bash
# 默认切到 Apple TV 的声音输出
bash "/Users/karuo/Documents/个人/卡若AI/运营中枢/工作台/投屏后切换音频到外接设备.sh"

# 若你的投屏设备不叫 Apple TV，可传设备名，例如：
# bash .../投屏后切换音频到外接设备.sh "HDMI"
# bash .../投屏后切换音频到外接设备.sh "Apple TV" "卡苹的麦克风"   # 同时改麦克风
```

### 3. 查看当前有哪些音频设备

```bash
# 输出设备
SwitchAudioSource -a -t output

# 输入设备（麦克风）
SwitchAudioSource -a -t input
```

把上面命令里的设备名填进脚本参数即可。

## 说明

- macOS 没有「一连投屏就自动切到外接设备」的官方选项，只能投屏后**手动选一次**或**跑一次脚本**。
- 脚本放在：`卡若AI/运营中枢/工作台/投屏后切换音频到外接设备.sh`，可拖进「快捷指令」做成桌面/菜单栏一键执行。
