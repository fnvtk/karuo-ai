## 四、模型/模式说明

### Cursor 默认模型

Cursor 通常使用以下模型之一：
- **Claude Sonnet**（Anthropic）
- **GPT-4**（OpenAI）
- **GPT-4 Turbo**

### 查看方式

1. **Cursor Settings → Models**：最直观
2. **配置文件**：`~/.cursor/User/settings.json` 中的 `cursor.general.model`
3. **对话询问**：在 Cursor 中直接问"你使用的是什么模型？"

### Agent 模式

Cursor 的 Agent 模式通常为：
- **默认模式**：根据上下文自动选择
- **Composer 模式**：代码生成与编辑
- **Chat 模式**：对话与问答
