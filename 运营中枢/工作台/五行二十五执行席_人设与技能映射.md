# 五行二十五执行席 · 人设与技能映射

> **定位**：在**不改** `01_～05_` 物理目录、**不改** `SKILL_REGISTRY.md` 路径的前提下，为每条五行线固定 **5 个执行席**（调度席），每席有 **人设 + MBTI + PDP 行为风格 + 次级五行气 + 技能编号锚**，便于卡若AI **拆任务、派活、协同**。
>
> **物理掌管人不变**：执行席是「编制视角」，多条席可落在同一成员目录（如 水桥 兼 水-叁、水-肆；金仓 兼 金-壹～肆）。**不是**「金仓再下设 5 人、金盾再下设 5 人」——仍为 **卡资线合计 5 席**，仅按职能拆给金仓/金盾执行。派发时 **`@金仓` `@水岸`** 等，或先喊席名再落到成员。
>
> **真源**：技能路径与触发词只认 **`SKILL_REGISTRY.md`**；项目视图另见 **`五负责人_五职能柱_按项目技能矩阵.md`**。

**版本**：1.0 | **更新**：2026-03-29

## 一、总则

| 概念 | 说明 |
|:---|:---|
| **五行负责人** | 卡资 / 卡人 / 卡木 / 卡火 / 卡土（各 1 人，人设见各组 `0X_卡X/SKILL.md` 第一节） |
| **执行席** | 每负责人下辖 **固定 5 席**；席名可口述，用于任务拆解 RACI |
| **次级五行气** | 本行内的气质侧写（金锐/金稳/水润…），**不替代**殿级五行归属 |
| **MBTI** | 人设参考维度，便于口吻与协作预期 |
| **PDP 五型** | 老虎 / 孔雀 / 考拉 / 猫头鹰 / 变色龙（**行为风格**，非证书） |
| **技能锚** | 仅列 **编号**，路径查注册表 |

### 1.1 跨组协同（高频）

| 从 → 到 | 典型场景 |
|:---|:---|
| 金 → 水 | 备份/日志就绪 → 飞书、纪要、归档 |
| 水 → 火 | 需求/排期冻结 → 开发、测试、修复 |
| 木 → 火 | 模板/切片/物料 → 全栈接入与发布 |
| 火 → 金 | 部署/端口/容器 → 主机与监控 |
| 土 → 全组 | ROI/财务/复制 → 立项与复盘对齐数据 |
| 水岸 → 全组 | 项目中枢、Soul 套件 → 分流 K01～K03 与 Stream |

### 1.2 卡火 · 开发向统筹（以功能为基准）

五席对应 **工程化交付 → 质量修复 → 需求澄清与复盘 → 知识/本地智能 → 观测扩展**，与 `五负责人_五职能柱_按项目技能矩阵.md` 火组柱 **一～五** 对齐，**不增删**火炬/火锤/火眼/火种目录。
