## 🏛️ 记忆宫殿命名规则

### 核心理念

```
五行元素 → 意象联想 → 专员命名

金 → 金属器具（剑、盾、仓、链）
水 → 水流形态（溪、泉、桥）
木 → 树木部位（叶、根、果）
火 → 火焰形态（炬、锤、眼、种）
土 → 土地建筑（基、砖、渠）
```

