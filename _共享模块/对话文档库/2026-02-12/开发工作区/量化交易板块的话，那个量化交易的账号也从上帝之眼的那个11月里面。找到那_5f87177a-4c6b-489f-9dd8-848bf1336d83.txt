  contents: # 量化交易流程与券商配置说明

> 整合自上帝之眼策略库与自动交易设计，支持国内券商（含湘财证券电话委托）。  
> 更新：2026-02-12

## 一、量化交易整体流程

1. **信号产出**：策略引擎（7根K线、节假日强中强、高手跟踪）产出买卖信号 → 写入 `trading_signals`。
2. **用户配置**：在「我的」中绑定量化交易账号、选择策略、设置风控（跟单比例、单笔上限、最大仓位）。
3. **自动化执行**：系统按用户开启的 `user_auto_trading` 配置，将信号对接到所选账户执行（实盘需对接券商接口或电话委托）。
4. **风控与记录**：止损 -5%、单笔上限、仓位比例限制；每笔交易建议记录日志。

---

## 二、交易所与数据源

| 类型     | 说明 |
|----------|------|
| **A 股** | 上交所、深交所，上帝之眼信号以 A 股为主 |
| **行情** | 新浪财经、东方财富等（见 `开发文档/1、需求/02-数据源清单.md`） |
| **策略库** | 本仓库 `开发文档/2、架构/策略库/`：7根K线、节假日、高手跟踪等 |

## 三、量化相关「插件」与程序

- **策略引擎**：`backend/app/services/strategy/seven_kline.py`（含节假日与高手跟踪骨架）。
- **信号 API**：`GET /api/v1/signals`。
- **自动化配置**：`user_auto_trading` 表 + `GET/PUT /api/v1/me/auto-trading`。
- **账户绑定**：`app_accounts` 表 + `GET/POST /api/v1/me/accounts`；券商字段 `broker` 可填「湘财证券」「雪球」「模拟券商」等。
- **可选扩展**：回测引擎 `backend/app/services/backtest/backtest_engine.py`；后续可接 Tushare/AKShare、Broker API 或电话委托中间件。

## 四、国内券商与湘财证券电话委托

### 4.1 湘财证券电话委托

- **用途**：国内不方便用 PC/APP 时，可通过券商提供的**电话委托**进行下单、查询。
- **典型流程**：
  1. 拨打湘财证券客服/委托电话（以官网或营业部告知为准）。
  2. 按语音提示输入资金账号、密码。
  3. 选择「买入」「卖出」等，按提示输入股票代码、价格、数量。
  4. 确认后完成委托。
- **在上帝之眼中的配置**：
  - 在「我的」→「绑定账户」中，券商选择「湘财证券」；
  - 账户名称可填写如「湘财·电话委托」；
  - 自动化交易若对接电话委托，需额外中间件（语音或按键协议），当前版本以「账户绑定 + 信号展示」为主，实盘电话委托由用户按信号自行拨打操作。
- **合规与风险**：电话委托需本人操作，注意账号密码安全；自动交易需遵守证券法规。

### 4.2 其他券商

- **broker** 可填：华泰证券、中信证券、国信证券、雪球、模拟券商等。
- 若后续接入某券商 OpenAPI，可在 `app_accounts.extra` 中存 `api_endpoint`、`env` 等配置。

## 五、胜利资源与风控提醒

- 文档与材料中提到的「胜利时急于兑现」等，属于**心理与风控**范畴：上涨后因害怕回撤而过早卖出。建议在自动化策略中：
  - 严格执行止盈/止损规则（如 7 根 K 线策略持有 7 日、-5% 止损）；
  - 不因短期波动随意关闭自动化或情绪化撤单。
- 「胜利资源」如指某份内部资源文档，可把其要点（如心理规则、止盈纪律）整理进本说明或风控条款，供用户在使用说明中查看。

## 六、使用流程（面向用户，可放在「我的」小问号说明中）

1. **绑定量化交易账号**：我的 → 量化交易账号（或 绑定账户）→ 选择/输入券商（如湘财证券）、账户名称；若为电话委托，选「湘财证券」并阅读电话委托说明。
2. **配置自动化交易**：我的 → 自动化交易 → 开启开关 → 选择要跟的策略（如 7K、HOLIDAY）、选择刚绑定的账户、设置跟单比例/单笔上限/最大仓位 → 保存。
3. **查看信号**：首页「交易信号」或 /signals 查看当前策略信号；决定由系统自动跟单或自行按信号电话/APP 下单。
4. **风控**：-5% 止损、单笔上限与仓位比例生效；建议定期查看持仓与成交记录。

'use client'

import { useState, useEffect } from 'react'
import Link from 'next/link'
import { meApi } from '@/lib/api'
import AppHeader from '@/app/components/AppHeader'
import Card from '@/app/components/ui/Card'
import HelpTooltip from '@/app/components/ui/HelpTooltip'

const STRATEGIES = [
  { id: '7K', name: '7根K线' },
  { id: 'HOLIDAY', name: '节假日强中强' },
  { id: 'EXPERT_FOLLOW', name: '高手跟单' },
]

export default function AutoTradingPage() {
  const [enabled, setEnabled] = useState(false)
  const [strategyIds, setStrategyIds] = useState<string[]>([])
  const [accountId, setAccountId] = useState<string | null>(null)
  const [followRatio, setFollowRatio] = useState(1)
  const [maxPositionPct, setMaxPositionPct] = useState(0.2)
  const [singleOrderLimit, setSingleOrderLimit] = useState<number | ''>('')
  const [accounts, setAccounts] = useState<Array<{ account_id: string; broker: string; name: string; is_default?: boolean }>>([])
  const [loading, setLoading] = useState(true)
  const [saving, setSaving] = useState(false)

  useEffect(() => {
    (async () => {
      try {
        const [config, list] = await Promise.all([meApi.autoTrading(), meApi.accounts()])
        if (config?.data) {
          setEnabled(config.data.enabled)
          setStrategyIds(config.data.strategy_ids || [])
          setAccountId(config.data.account_id ?? null)
          setFollowRatio(config.data.follow_ratio ?? 1)
          setMaxPositionPct(config.data.max_position_pct ?? 0.2)
          setSingleOrderLimit(config.data.single_order_limit ?? '')
        }
        if (list?.data) setAccounts(list.data)
      } catch (e) {
        console.error(e)
      } finally {
        setLoading(false)
      }
    })()
  }, [])

  const toggleStrategy = (id: string) => {
    setStrategyIds((prev) => (prev.includes(id) ? prev.filter((s) => s !== id) : [...prev, id]))
  }

  const save = async () => {
    setSaving(true)
    try {
      await meApi.saveAutoTrading({
        enabled,
        strategy_ids: strategyIds,
        account_id: accountId,
        follow_ratio: followRatio,
        max_position_pct: maxPositionPct,
        single_order_limit: singleOrderLimit === '' ? null : Number(singleOrderLimit),
      })
    } catch (e) {
      console.error(e)
    } finally {
      setSaving(false)
    }
  }

  if (loading) {
    return (
      <div className="min-h-screen bg-[var(--godeye-bg)]">
        <AppHeader title="自动化交易" showBack backHref="/settings" />
        <div className="px-4 py-6 text-center text-[var(--godeye-text-muted)] text-sm">加载中...</div>
      </div>
    )
  }

  return (
    <div className="min-h-screen bg-[var(--godeye-bg)]">
      <div className="flex items-center">
        <AppHeader title="自动化交易" showBack backHref="/settings" />
        <HelpTooltip>
          {`开启后按所选策略与风控参数执行或推送信号。流程：先绑定量化交易账号，再在此选择该账户与策略。
风控：-5% 止损自动生效；可设单笔上限、最大仓位比例。`}
        </HelpTooltip>
      </div>

      <div className="px-4 pt-2 pb-6 space-y-4">
        <Card className="rounded-2xl p-4 space-y-4">
          <div className="flex items-center justify-between">
            <span className="text-sm font-medium text-[var(--godeye-text)]">启用自动化交易</span>
            <button
              type="button"
              onClick={() => setEnabled(!enabled)}
              className={`w-12 h-7 rounded-full flex items-center transition-colors ${enabled ? 'bg-godeye-primary justify-end' : 'bg-black/10 justify-start'}`}
            >
              <span className="w-5.5 h-5.5 m-0.5 rounded-full bg-white shadow-sm block" style={{ width: 22, height: 22 }} />
            </button>
          </div>

          <div>
            <label className="text-[var(--godeye-text-secondary)] text-sm flex items-center gap-1 mb-2">
              选择策略
              <HelpTooltip title="可多选：7根K线、节假日、高手跟单等" />
            </label>
            <div className="flex flex-wrap gap-2">
              {STRATEGIES.map((s) => (
                <button
                  key={s.id}
                  type="button"
                  onClick={() => toggleStrategy(s.id)}
                  className={`px-3 py-1.5 rounded-xl text-sm ${strategyIds.includes(s.id) ? 'bg-godeye-primary text-white' : 'bg-black/5 text-[var(--godeye-text-secondary)]'}`}
                >
                  {s.name}
                </button>
              ))}
            </div>
          </div>

          <div>
            <label className="text-[var(--godeye-text-secondary)] text-sm block mb-1">执行账户</label>
            <select
              value={accountId ?? ''}
              onChange={(e) => setAccountId(e.target.value || null)}
              className="w-full rounded-2xl px-3 py-2.5 text-[var(--godeye-text)] bg-white border border-black/8"
            >
              <option value="">请选择账户</option>
              {accounts.map((a) => (
                <option key={a.account_id} value={a.account_id}>
                  {a.name}（{a.broker}）
                </option>
              ))}
            </select>
            {accounts.length === 0 && (
              <p className="text-xs text-[var(--godeye-text-muted)] mt-1">
                请先在 <Link href="/settings/accounts/bind" className="text-godeye-primary">量化交易账号</Link> 中绑定账户
              </p>
            )}
          </div>

          <div>
            <label className="text-[var(--godeye-text-secondary)] text-sm block mb-1">跟单比例（1=100%）</label>
            <input
              type="number"
              min={0.1}
              max={2}
              step={0.1}
              value={followRatio}
              onChange={(e) => setFollowRatio(Number(e.target.value))}
              className="w-full rounded-2xl px-3 py-2.5 text-[var(--godeye-text)] bg-white border border-black/8"
            />
          </div>
          <div>
            <label className="text-[var(--godeye-text-secondary)] text-sm block mb-1">单只最大仓位（如 0.2=20%）</label>
            <input
              type="number"
              min={0.01}
              max={1}
              step={0.01}
              value={maxPositionPct}
              onChange={(e) => setMaxPositionPct(Number(e.target.value))}
              className="w-full rounded-2xl px-3 py-2.5 text-[var(--godeye-text)] bg-white border border-black/8"
            />
          </div>
          <div>
            <label className="text-[var(--godeye-text-secondary)] text-sm block mb-1">单笔上限（元，选填）</label>
            <input
              type="number"
              min={0}
              placeholder="不填则不限制"
              value={singleOrderLimit}
              onChange={(e) => setSingleOrderLimit(e.target.value === '' ? '' : Number(e.target.value))}
              className="w-full rounded-2xl px-3 py-2.5 text-[var(--godeye-text)] bg-white border border-black/8"
            />
          </div>

          <button
            onClick={save}
            disabled={saving}
            className="w-full rounded-2xl py-3 font-bold text-white bg-godeye-primary hover:opacity-90 disabled:opacity-50"
          >
            {saving ? '保存中...' : '保存配置'}
          </button>
        </Card>
      </div>
    </div>
  )
}
