## 能力概述

通过对话直接操控存客宝私域系统，实现：
- **消息发送**：向指定设备、微信号、好友/群发送消息
- **设备管理**：查看设备状态、重启设备、批量操作
- **微信管理**：查看微信号列表、好友/群组、添加删除
- **场景获客**：创建/管理获客场景、查看统计
- **工作台任务**：朋友圈同步、社群推送、自动点赞等
- **流量池管理**：用户资产数字化、标签管理
- **数据统计**：获客趋势、转化率分析

## API接口清单

### 1. 认证接口

| 接口 | 方法 | 说明 |
|:---|:---|:---|
| `/token` | POST | 获取访问令牌 |

**认证方式**：Bearer Token

```python
from scripts.ckb_sdk import CunkebaoSDK

sdk = CunkebaoSDK()
sdk.login(username="xxx", password="xxx")
```

### 2. 消息发送（核心能力）

**协议**：WebSocket  
**地址**：`wss://s2.siyuguanli.com:9993`

| 命令 | 说明 | 关键参数 |
|:---|:---|:---|
| CmdSendMsg | 发送消息 | wechatAccountId（发信微信号）, toWxid（接收方）, msgType, content |
| CmdAddFriend | 添加好友 | wechatAccountId, wxid, message |
| CmdCreateChatroom | 创建群聊 | wechatAccountId, memberWxids, name |

**消息发送示例**：

```python
# 指定微信号发送消息
sdk.send_message(
    wechat_account_id="wx_account_001",  # 用哪个微信号发
    to_wxid="wxid_friend123",            # 发给谁
    msg_type="text",                      # 消息类型
    content="你好！"                       # 消息内容
)
```

**关键说明**：
- `wechatAccountId`：指定用哪个存客宝绑定的微信号发送
- `toWxid`：指定发给哪个微信号或群

## 风控规则

消息发送必须遵守以下风控限制：

| 规则 | 限制 |
|:---|:---|
| 发送间隔 | 3-8秒随机间隔 |
| 每日上限 | 单号200条/天 |
| 同一目标 | 60秒内不重复发 |

```python
# SDK已内置风控
sdk.send_message_safe(...)  # 自动控制频率
```

## 对话交互流程

当用户发起请求时：

```
用户: "用微信号A给张三发消息：明天见"
     ↓
1. 识别意图：发送消息
2. 解析参数：
   - 发信号：微信号A → 查询wechatAccountId
   - 接收方：张三 → 查询好友列表找到wxid
   - 内容：明天见
3. 调用SDK：
   sdk.send_message(
       wechat_account_id="wx_account_001",
       to_wxid="wxid_zhangsan",
       msg_type="text",
       content="明天见"
   )
4. 返回结果：
   "✅ 已通过微信号A向张三发送消息"
```

        self,
        wechat_account_id: str,
        to_wxid: str,
        content: str,
        msg_type: str = "text"
    ) -> Dict:
        """
        发送消息（带风控）
        
        自动处理：
        - 发送间隔（3-8秒随机）
        - 每日上限检查
        - 同一目标间隔检查
        """
        # 检查每日上限
        today = time.strftime("%Y-%m-%d")
        daily_key = f"{wechat_account_id}:{today}"
        if self._daily_send_count.get(daily_key, 0) >= DAILY_SEND_LIMIT:
            logger.warning(f"微信号 {wechat_account_id} 今日发送已达上限")
            return {"success": False, "message": "今日发送已达上限"}
        
        # 检查同一目标间隔
        target_key = f"{wechat_account_id}:{to_wxid}"
        last_time = self._last_send_time.get(target_key, 0)
        if time.time() - last_time < SAME_TARGET_INTERVAL:
            wait_time = SAME_TARGET_INTERVAL - (time.time() - last_time)
            logger.warning(f"发送过于频繁，需等待 {wait_time:.0f} 秒")
            return {"success": False, "message": f"发送过于频繁，需等待 {wait_time:.0f} 秒"}
        
        # 随机延迟
        delay = random.uniform(MESSAGE_INTERVAL_MIN, MESSAGE_INTERVAL_MAX)
        time.sleep(delay)
        
        # 发送消息
        result = self.send_message(wechat_account_id, to_wxid, content, msg_type)
        
        # 更新计数
        self._daily_send_count[daily_key] = self._daily_send_count.get(daily_key, 0) + 1
        self._last_send_time[target_key] = time.time()
        
        return result

                        "seq": self.ws_seq

                logger.error(f"WebSocket发送失败: {e}")
                return {"success": False, "message": str(e)}

## 发送消息（待实现）

发送消息需要 WebSocket 连接，流程如下：

```
1. 获取 WebSocket Token
   POST https://s2.siyuguanli.com:9991/token
   参数: grant_type=password, username=xxx, password=xxx
   Headers: client=kefu-client, verifysessionid=xxx, verifycode=xxx
   ↓ 返回 access_token

2. 连接 WebSocket
   wss://s2.siyuguanli.com:9993
   ↓ 发送 CmdSignIn 命令

3. 发送消息
   命令: CmdSendMessage
   参数:
   - wechatAccountId: 发信微信号ID（数据库ID，如 85676）
   - wechatFriendId: 好友ID（数据库ID，如 21323731）
   - wechatChatroomId: 群ID（发群消息时使用）
   - msgType: 消息类型（1=文本, 3=图片, 43=视频, 47=表情, 49=小程序）
   - content: 消息内容
```

**当前问题**：WebSocket Token 获取需要验证码，验证码已过期。
