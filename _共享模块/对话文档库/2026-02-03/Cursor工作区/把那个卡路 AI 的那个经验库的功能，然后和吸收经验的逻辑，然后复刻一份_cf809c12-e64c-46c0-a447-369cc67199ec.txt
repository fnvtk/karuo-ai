## 工作原理

```
每次对话 → 有价值内容 → 存入经验库（待沉淀→已整理）
    │
    └→ 执行时检查：已整理 > 10？

         ├─ 否 → 继续
         └─ 是 → 分配 Skill 吸收 → 吸收完成 → 删除原文件
```

## 自动捕捉规则

### 什么内容会被识别为"有价值"？

| 类型 | 特征 | 示例 |
|:---|:---|:---|
| **操作步骤** | 包含具体的操作流程 | "先执行A，再执行B，最后C" |
| **解决方案** | 解决了某个具体问题 | "这个bug是因为X，解决方法是Y" |
| **配置方法** | 涉及环境/工具配置 | "MongoDB连接需要设置这些参数" |
| **最佳实践** | 总结出的经验规律 | "做私域项目，先算账再动手" |
| **常用命令** | 重复使用的命令组合 | "部署时总是执行这三条命令" |

### 不捕捉的内容

- ❌ 一次性的简单问答
- ❌ 闲聊内容
- ❌ 敏感信息（密码、密钥）
- ❌ 临时性的数据

经验库检查 - 执行时自动检查，已整理 > 10 触发 Skill 吸收（自动迭代经验）

from pathlib import Path

# 存客宝AI 根目录

CUNKEBAO_ROOT = SCRIPT_DIR.parent.parent

def count_organized_files() -> tuple[int, list[tuple[str, Path]]]:
    """统计已整理目录下所有 .md 文件（扁平结构，无子目录）"""
    files = []
    if not ORGANIZED_DIR.exists():
        return 0, []
    for f in ORGANIZED_DIR.glob("*.md"):
        if f.name.lower() != "readme.md":
            # 尝试从文件内容提取分类（首300字符内）
            cat = "未分类"
            try:
                text = f.read_text(encoding="utf-8")[:300]
                if "**分类**：" in text or "**分类**:" in text:
                    m = re.search(r"\*\*分类\*\*[：:]\s*(\S+)", text)
                    if m:
                        cat = m.group(1).strip()
            except Exception:
                pass
            files.append((cat, f))
    return len(files), files


def main_check() -> int:
    """执行检查，若 > 10 输出待吸收清单"""
    count, files = count_organized_files()
    print(f"📊 经验库已整理文件数：{count}")

    if count <= THRESHOLD:
        print(f"✅ 未超过阈值（{THRESHOLD}），无需触发吸收")

    print("-" * 50)

    for cat, fps in sorted(by_cat.items()):
        skill = CATEGORY_TO_SKILL.get(cat, "02_项目组/文件整理、03_开发组/开发模板")
        print(f"\n【{cat}】→ 分配 {skill}")
        for fp in fps:
            try:
                rel = fp.relative_to(ORGANIZED_DIR)
            except ValueError:
                rel = fp.name
            print(f"  - {rel}")
    print("-" * 50)
    print("\n执行步骤：")
    print("1. 分配相应 Skill 读取上述文件")
    print("2. 将经验吸收到 SKILL.md 或 references/")
    print("3. 吸收完成后运行：python3 experience_check.py --cleanup <文件路径...>")
    print("   或手动删除 _经验库/已整理/ 下已吸收的文件")
    return 1  # 返回 1 表示触发了吸收流程


def main_cleanup(paths: list[str]) -> int:
    """吸收完成后删除指定文件"""
    deleted = 0
    for p in paths:
        fp = Path(p)
        if not fp.is_absolute():

        if fp.exists():
            try:

            deleted += 1
        else:
            print(f"  ⚠️ 跳过（不存在）: {p}")
    return 0 if deleted else 1


if __name__ == "__main__":
    if len(sys.argv) > 1 and sys.argv[1] == "--cleanup":

        if not paths:
            print("用法: python3 experience_check.py --cleanup <文件路径...>")
            sys.exit(1)
        sys.exit(main_cleanup(paths))
    sys.exit(main_check())
