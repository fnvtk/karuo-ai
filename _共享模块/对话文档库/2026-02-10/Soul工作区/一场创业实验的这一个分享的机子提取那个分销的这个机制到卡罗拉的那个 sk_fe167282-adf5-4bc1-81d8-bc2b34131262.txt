### 绑定机制流程

```
用户点击分享链接（含推荐码ref参数）
    ↓
记录访问日志（referral_visits）
    ↓
检查用户是否已有有效绑定
    ├─ 无绑定 → 创建新绑定（30天有效期）
    ├─ 同一推荐人 → 续期30天
    └─ 不同推荐人 →
        ├─ 已过期 → 允许抢夺，创建新绑定
        └─ 未过期 → 拒绝，保持原关系
```

### 佣金计算流程

```
用户付款成功（支付回调触发）
    ↓
查找用户的有效绑定关系（status=active & 未过期）
    ├─ 无绑定 → 跳过分佣
    └─ 有绑定 →
        佣金 = 订单金额 × 90%
        ↓
        更新推广者 pending_earnings
        更新绑定 status → converted
        记录 commission_amount / order_id
        ↓
        WebSocket 实时通知推广者
```

## 常见问题

1. **为什么只做一级分销？** 多级分销有法律风险（涉嫌传销），一级分销合规且追踪简单。
2. **30天锁定期的意义？** 给推广者足够的转化窗口，同时避免永久绑定导致资源浪费。
3. **佣金90%是否太高？** 初期推广阶段用高佣金吸引推广者，后期可在system_config中调低。
4. **绑定被抢夺怎么办？** 只有过期后才能被抢夺，未过期的绑定受保护。
5. **新项目怎么快速接入？** 复制 `lib/modules/distribution/` 整个目录 + API路由 + 数据库建表，改配置即可。
6. **提现到哪里？** 支持微信零钱（企业付款V3 API）和支付宝（单笔转账），具体对接见小程序支付Skill。

## 2. referral_bindings 表（绑定关系）

```sql
CREATE TABLE referral_bindings (
  id VARCHAR(50) PRIMARY KEY COMMENT '绑定ID',
  referrer_id VARCHAR(50) NOT NULL COMMENT '推广者ID',
  referee_id VARCHAR(50) NOT NULL COMMENT '被推广者ID',
  referral_code VARCHAR(20) NOT NULL COMMENT '推荐码',
  status ENUM('active', 'converted', 'expired') DEFAULT 'active' COMMENT '状态',
  binding_date TIMESTAMP DEFAULT CURRENT_TIMESTAMP COMMENT '绑定时间',
  expiry_date TIMESTAMP NOT NULL COMMENT '过期时间（绑定时间+30天）',
  conversion_date TIMESTAMP NULL COMMENT '转化时间（付款时间）',
  commission_amount DECIMAL(10,2) DEFAULT 0 COMMENT '佣金金额',
  order_id VARCHAR(50) NULL COMMENT '关联订单ID',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
  
  UNIQUE KEY unique_referrer_referee (referrer_id, referee_id),
  INDEX idx_referrer_id (referrer_id),
  INDEX idx_referee_id (referee_id),
  INDEX idx_status (status),
  INDEX idx_expiry_date (expiry_date)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COMMENT='分销绑定关系表';
```

### 字段说明

| 字段 | 类型 | 说明 |
|:---|:---|:---|
| referrer_id | VARCHAR(50) | 推广者（分享链接的人） |
| referee_id | VARCHAR(50) | 被推广者（点击链接的人） |
| status | ENUM | active=绑定中, converted=已转化(付款), expired=已过期 |
| expiry_date | TIMESTAMP | 绑定时间 + 30天，过期后可被其他推广者抢夺 |
| commission_amount | DECIMAL | 转化后记录的佣金金额 |
| order_id | VARCHAR(50) | 转化时关联的订单ID |

### 状态流转

```
创建绑定 → active
    ├─ 用户付款 → converted（终态，不再变化）
    └─ 超过30天 → expired（可被重新绑定）
```

```typescript
async function processReferralCommission(userId: string, orderId: string, amount: number) {
  // 1. 查找有效绑定关系
  const binding = await db.query(`
    SELECT * FROM referral_bindings 
    WHERE referee_id = ? AND status = 'active' AND expiry_date > NOW()
    LIMIT 1
  `, [userId]);
  
  if (!binding) return; // 无绑定，跳过分佣
  
  // 2. 获取分佣比例配置
  const config = await getDistributionConfig();
  const commissionRate = config.distributorShare / 100; // 0.9
  const commission = amount * commissionRate;
  
  // 3. 更新推广者收益
  await db.query(`
    UPDATE users SET pending_earnings = pending_earnings + ? WHERE id = ?
  `, [commission, binding.referrer_id]);
  
  // 4. 更新绑定记录
  await db.query(`
    UPDATE referral_bindings SET 
      status = 'converted', 
      commission_amount = ?,
      conversion_date = NOW(),
      order_id = ?
    WHERE id = ?
  `, [commission, orderId, binding.id]);
  
  // 5. 实时通知推广者
  sendWebSocketNotification(binding.referrer_id, {
    type: 'commission',
    amount: commission,
    orderId
  });
}
```

## 签名算法（关键）

```typescript
// MD5签名生成（微信支付V2）
function generateSign(params: Record<string, string>, mchKey: string): string {
  // 1. 按字母排序参数
  const sortedKeys = Object.keys(params).sort();
  
  // 2. 拼接参数（过滤空值和sign字段）
  const signString = sortedKeys
    .filter(key => params[key] && key !== 'sign')
    .map(key => `${key}=${params[key]}`)
    .join('&');
  
  // 3. 拼接商户密钥
  const signWithKey = `${signString}&key=${mchKey}`;
  
  // 4. MD5加密并转大写
  return crypto.createHash('md5')
    .update(signWithKey, 'utf8')
    .digest('hex')
    .toUpperCase();
}
```

## 常见问题

1. **签名校验失败？** 检查：参数排序是否正确、mchKey是否正确、编码是否UTF-8、是否过滤了空值和sign字段。
2. **回调收不到？** 检查：notify_url是否HTTPS、服务器是否可外网访问、防火墙是否放行、是否返回了SUCCESS给微信。
3. **小程序调不起支付？** 检查：openId是否正确、小程序是否关联了商户号、trade_type是否为JSAPI。
4. **支付宝和微信如何共存？** 使用工厂模式（factory.ts），通过gateway参数选择支付网关，统一接口。
5. **如何接入新项目？** 复制 `lib/payment/` 目录 + API路由 + 建orders表 + 配.env，改回调URL即可。
6. **提现到零钱怎么做？** 使用 `lib/wechat-transfer.ts`（V3 API），需要商户证书（apiclient_key.pem），最低1元。

```typescript
    notify_url: `${process.env.NEXT_PUBLIC_BASE_URL}/api/miniprogram/pay/notify`,
    trade_type: 'JSAPI',
    openid: params.openId
  };
  unifiedOrderParams.sign = generateSign(unifiedOrderParams, process.env.WECHAT_MCH_KEY);
  
  // 4. 发送XML请求
  const xmlBody = dictToXml(unifiedOrderParams);
  const response = await fetch('https://api.mch.weixin.qq.com/pay/unifiedorder', {
    method: 'POST',
    headers: { 'Content-Type': 'application/xml' },
    body: xmlBody
  });
  const result = xmlToDict(await response.text());
  
  // 5. 生成前端支付参数
  const payParams = {
    timeStamp: Math.floor(Date.now() / 1000).toString(),
    nonceStr: generateNonceStr(),
    package: `prepay_id=${result.prepay_id}`,
    signType: 'MD5'
  };
  payParams.paySign = generateSign({
    appId: process.env.WECHAT_APPID,
    ...payParams
  }, process.env.WECHAT_MCH_KEY);
  
  return { orderSn, prepayId: result.prepay_id, payParams };
}
```

```typescript
async function handlePayNotify(xmlData: string) {
  // 1. 解析XML
  const data = xmlToDict(xmlData);
  
  // 2. 验证签名
  const receivedSign = data.sign;
  delete data.sign;
  const calculatedSign = generateSign(data, process.env.WECHAT_MCH_KEY);
  if (receivedSign !== calculatedSign) {
    return '<xml><return_code>FAIL</return_code><return_msg>签名验证失败</return_msg></xml>';
  }
  
  // 3. 检查支付结果
  if (data.result_code !== 'SUCCESS') {
    return '<xml><return_code>FAIL</return_code></xml>';
  }
  
  // 4. 查找订单
  const order = await db.query('SELECT * FROM orders WHERE order_sn = ?', [data.out_trade_no]);
  if (!order || order.status === 'paid') {
    // 已处理或不存在，返回成功避免重复通知
    return '<xml><return_code>SUCCESS</return_code></xml>';
  }
  
  // 5. 更新订单状态
  await db.query(`
    UPDATE orders SET status = 'paid', transaction_id = ?, pay_time = NOW() WHERE order_sn = ?
  `, [data.transaction_id, data.out_trade_no]);
  
  // 6. 更新用户购买记录
  if (order.product_type === 'fullbook') {
    await db.query('UPDATE users SET has_full_book = TRUE WHERE id = ?', [order.user_id]);
  } else if (order.product_type === 'section') {
    await db.query(`
      UPDATE users SET purchased_sections = JSON_ARRAY_APPEND(
        COALESCE(purchased_sections, '[]'), '$', ?
      ) WHERE id = ?
    `, [order.product_id, order.user_id]);
  }
  
  // 7. 处理分销佣金（如果有分销系统）
  await processReferralCommission(order.user_id, order.id, order.amount);
  
  // 8. 返回成功（必须返回，否则微信会重复通知）
  return '<xml><return_code><![CDATA[SUCCESS]]></return_code><return_msg><![CDATA[OK]]></return_msg></xml>';
}
```

```typescript
async function pollPaymentStatus(tradeSn: string, gateway: string) {

  for (let i = 0; i < maxAttempts; i++) {
```

## 7. XML工具函数

```typescript
// JS对象转XML
function dictToXml(params: Record<string, string>): string {
  const xmlParts = Object.entries(params)
    .map(([key, value]) => `<${key}><![CDATA[${value}]]></${key}>`);
  return `<xml>${xmlParts.join('')}</xml>`;
}

// XML转JS对象
function xmlToDict(xml: string): Record<string, string> {
  const result: Record<string, string> = {};
  const regex = /<(\w+)><!\[CDATA\[(.*?)\]\]><\/\1>|<(\w+)>(.*?)<\/\3>/g;
  let match;
  while ((match = regex.exec(xml)) !== null) {
    const key = match[1] || match[3];
    const value = match[2] || match[4];
    result[key] = value;
  }
  return result;
}

// 生成随机字符串
function generateNonceStr(length = 32): string {
  const chars = 'ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789';
  return Array.from({ length }, () => chars[Math.floor(Math.random() * chars.length)]).join('');
}
```

## 8. 注意事项

1. **金额单位**：微信支付使用「分」，前端传「元」，后端转换 `Math.round(amount * 100)`
2. **幂等处理**：回调可能重复通知，需检查订单是否已处理（`status === 'paid'` 直接返回SUCCESS）
3. **签名编码**：必须使用 UTF-8 编码
4. **HTTPS要求**：回调URL必须是HTTPS
5. **超时处理**：统一下单返回的prepay_id有效期2小时，前端需提示超时重新发起
6. **退款**：退款需要商户证书（apiclient_cert.pem），调用 `/pay/refund` 接口

```javascript
/**
 * 小程序支付工具类
 * 封装支付流程，供各页面调用
 */
const app = getApp();

const PaymentUtil = {
  /**
   * 发起支付
   * @param {Object} options 支付参数
   * @param {string} options.productType - 产品类型：section/fullbook/match
   * @param {string} options.productId - 产品ID
   * @param {number} options.amount - 金额（元）
   * @param {string} options.description - 订单描述
   * @returns {Promise<Object>} 支付结果
   */
  async pay(options) {
    const { productType, productId, amount, description } = options;
    
    // 1. 获取openId
    let openId = app.globalData.openId;
    if (!openId) {
      await app.login();
      openId = app.globalData.openId;
    }
    if (!openId) {
      throw new Error('获取openId失败，请重新登录');
    }
    
    // 2. 调用后端创建订单
    wx.showLoading({ title: '正在创建订单...' });
    const orderRes = await app.request('/api/miniprogram/pay', {
      method: 'POST',
      data: {
        openId,
        productType,
        productId,
        amount,
        description: description || '商品购买'
      }
    });
    wx.hideLoading();
    
    if (!orderRes.success) {
      throw new Error(orderRes.message || '创建订单失败');
    }
    
    // 3. 调起微信支付
    const payParams = orderRes.data.payParams;
    const payResult = await this.callWechatPay(payParams);
    
    return {
      success: true,
      orderSn: orderRes.data.orderSn,
      ...payResult
    };
  },
  
  /**
   * 调用微信支付
   * @param {Object} payParams - 支付参数
   * @returns {Promise}
   */
  callWechatPay(payParams) {
    return new Promise((resolve, reject) => {
      wx.requestPayment({
        timeStamp: payParams.timeStamp,
        nonceStr: payParams.nonceStr,
        package: payParams.package,
        signType: payParams.signType || 'MD5',
        paySign: payParams.paySign,
        success(res) {
          resolve({ paySuccess: true, ...res });
        },
        fail(err) {
          if (err.errMsg.includes('cancel')) {
            reject(new Error('用户取消支付'));
          } else {
            reject(new Error(`支付失败: ${err.errMsg}`));
          }
        }
      });
    });
  },
  
  /**
   * 查询订单状态
   * @param {string} orderSn - 订单号
   * @returns {Promise<Object>}
   */
  async queryOrder(orderSn) {
    return await app.request(`/api/payment/status/${orderSn}`, {
      method: 'GET'
    });
  }
};

module.exports = PaymentUtil;
```

```javascript
Page({
```

```javascript
// app.js
App({
  globalData: {
    openId: null,
    baseUrl: 'https://soul.quwanzhi.com' // 替换为你的域名
  },
  
  /**
   * 统一请求封装
   */
  request(url, options = {}) {
    return new Promise((resolve, reject) => {
      wx.request({
        url: `${this.globalData.baseUrl}${url}`,
        method: options.method || 'GET',
        data: options.data,
        header: {
          'Content-Type': 'application/json',
          ...options.header
        },
        success(res) {
          if (res.statusCode === 200) {
            resolve(res.data);
          } else {
            reject(new Error(`请求失败: ${res.statusCode}`));
          }
        },
        fail(err) {
          reject(new Error(`网络错误: ${err.errMsg}`));
        }
      });
    });
  },
  
  /**
   * 微信登录获取openId
   */
  login() {
    return new Promise((resolve, reject) => {
      wx.login({
        success: async (res) => {
          if (res.code) {
            try {
              const result = await this.request('/api/miniprogram/login', {
                method: 'POST',
                data: { code: res.code }
              });
              this.globalData.openId = result.openId;
              resolve(result);
            } catch (err) {
              reject(err);
            }
          } else {
            reject(new Error('登录失败'));
          }
        },
        fail: reject
      });
    });
  },
  
  /**
   * 获取openId
   */
  async getOpenId() {
    if (this.globalData.openId) return this.globalData.openId;

  }
```
