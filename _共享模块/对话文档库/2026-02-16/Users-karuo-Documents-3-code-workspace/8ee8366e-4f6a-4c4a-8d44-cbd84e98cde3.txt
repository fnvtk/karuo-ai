## 视频切片 × 剪映：整体关系

- **不是**直接「调用剪映软件」做剪辑，而是：  
  切片流程（转录 → 高光/切片 → 成片）做完后，**把结果导入剪映**，在剪映里做二次编辑或发布。
- 和剪映的对接方式有三种，对应三个脚本，都是「生成/推送剪映草稿」：

| 方式 | 脚本 | 依赖 | 说明 |
|------|------|------|------|
| **1. 推荐** | `create_jianying_draft.py` | `pyJianYingDraft` | 用库生成草稿 JSON，直接写到剪映草稿目录 |
| **2. 手写草稿** | `generate_jianying_draft.py` | 无 | 手写剪映草稿 JSON 结构，生成 `draft_content.json` + `draft_meta_info.json` |
| **3. HTTP API** | `jianying_process.py` | CapCutAPI 服务 `localhost:9001` | 通过 HTTP 调用 VectCutAPI，远程创建草稿、加视频/文字/字幕 |

---

## 1. `create_jianying_draft.py`（推荐：pyJianYingDraft）

- **作用**：用 `pyJianYingDraft` 为**每个切片**生成一个剪映草稿，自动加 Hook（前 3 秒）、CTA（最后 5 秒）。
- **剪映草稿目录**：  
  `~/Movies/JianyingPro/User Data/Projects/com.lveditor.draft`
- **输入**：
  - 切片目录：`CLIPS_DIR`（默认 `/Users/karuo/Movies/玩值电竞_切片/clips`）
  - 高光 JSON：`HIGHLIGHTS_JSON`（含 `hook_3sec`、`cta_ending`）
- **流程**：  
  创建 1080×1920 竖屏草稿 → 加视频轨道 → 加 Hook 文字（黄色、偏上、可弹跳）→ 加 CTA 文字（白色、偏下、可淡入）→ `script.save()` 写到剪映草稿目录。
- **使用**：装好 `pyJianYingDraft` 后直接跑该脚本；打开剪映即可在草稿列表里看到新项目。

---

## 2. `generate_jianying_draft.py`（手写草稿 JSON）

- **作用**：不依赖剪映 API/库，**手写剪映草稿格式**，生成：
  - `draft_content.json`（画布、轨道、视频素材、字幕、Hook/CTA 文字等）
  - `draft_meta_info.json`
  - `draft_materials/` 目录
- **能力**：
  - 用 ffprobe 取视频信息，用 SRT 解析字幕
  - 支持多种字幕样式：default、impact、elegant、modern
  - 可选前 3 秒 Hook、结尾 CTA
- **输出**：生成 `dfd_<uuid>` 草稿文件夹；可选 `--to_jianying` 复制到剪映项目目录（同上 `JianyingPro/.../com.lveditor.draft`）。

---

## 3. `jianying_process.py`（HTTP 调用 CapCutAPI）

- **作用**：通过 **CapCutAPI**（VectCutAPI，默认 `http://localhost:9001`）远程「调用」剪映侧能力：创建草稿、加视频、加文字、加字幕、保存草稿。
- **流程**：
  1. `check_capcut_api()` 检查 9001 是否可用；
  2. 若可用：对每个切片 `create_draft` → `add_video_to_draft` → `add_text_to_draft`（Hook/CTA）→ `add_captions_to_draft`（可选）→ `save_draft`，可选 `--to_jianying` 把生成的草稿复制到剪映目录；
  3. 若不可用：生成《剪映导入说明.md》，指导手动把切片导入剪映并加 Hook/CTA。
- **剪映草稿目录（本脚本里）**：  
  `~/Library/Containers/com.lemon.lvpro/Data/Movies/JianyingPro/User Data/Projects`（与方案 1 的路径不同，可能是不同版本/安装方式）。

---

## 和主流程的衔接

- **统一入口** `video_clipper.py` 的 `vclip -i "视频" --clip` 做的是：转录 → 去语助词 → 高光/切片 → 成片（含字幕烧录等），**没有**在流水线里自动调上面三个脚本。
- 剪映相关是**后置步骤**：你可以在切片/成片生成后，**任选一种**方式把结果导入剪映：
  - 用 `create_jianying_draft.py` 批量生成带 Hook/CTA 的剪映草稿（推荐）；
  - 或用 `generate_jianying_draft.py` 单视频生成草稿；
  - 或起 CapCutAPI 后用 `jianying_process.py` 走 HTTP 调用。

---

## 小结

- **「调用剪映」**在视频切片里 = **生成/推送剪映草稿**，不是调剪映 GUI 或进程。
- **三种实现**：  
  - 库方案：`create_jianying_draft.py`（pyJianYingDraft，写草稿目录）；  
  - 手写 JSON：`generate_jianying_draft.py`；  
  - 远程控制：`jianying_process.py`（CapCutAPI 9001）。
