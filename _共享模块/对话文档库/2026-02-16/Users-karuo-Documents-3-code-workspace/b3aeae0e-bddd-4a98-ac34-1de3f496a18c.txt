  contents: # 建成第一个网站：从0到1的探索之路

2004年，我白天上课，晚上在厦门国贸大厦三楼的网吧里。

这是我舅投资的网吧，由我和我表弟运营。

那年圣诞节，我坐在网吧办公室的桌旁边，手指冻得有些僵硬。

网吧老板陈总说，有个业务可以赚钱想找我聊聊。

我看了看他的电脑，发现他的电脑屏幕上显示着一个简单的WAP页面，上面写着"欢迎访问我的彩玲网站"。

以及他给我看了后台的收益。

"如果我能做一个WAP网站，用户通过手机就能订阅短信服务和下载铃声..."我喃喃自语，思绪如春潮般汹涌。

2004年正是中国移动增值业务黄金时期，手机铃声和短信内容服务爆发，各种"笑话短信"、"情感短信"和"和弦铃声"十分流行，市场规模已达几百亿元。

为了了解整个行业的运作机制，我花了一周时间研究当时的SP/CP模式。

在这个模式下，产业链分为三层：

运营商（中国移动、中国联通）位于顶端控制着计费通道。

SP（Service Provider，服务提供商）位于中间拥有接入运营商网络的资质。

CP（Content Provider，内容提供商）位于底端负责内容制作和获取用户。

我决定从CP角色切入这个市场。

为了给网站一个好记的名字，我注册了51313.com这个域名。

在那个年代，数字域名非常受欢迎，尤其是谐音域名更是抢手货。

51313，读起来像"我要想你"，非常适合做铃声和短信内容业务。

经过两周的自学和无数次失败尝试，我终于在一个周日的凌晨两点，完成了第一个能够正常运行的WAP页面。

它简陋得可怜——只有几个黑白文字链接和一个简单的提交表单。

但当我用诺基亚手机成功访问并下载了一段铃声时，那种成就感无可比拟。

最初的想法很简单：用户通过WAP网站订阅内容服务或下载铃声，按照当时流行的"包月"或"按条"两种收费形式支付费用。

"包月"一般收费5-10元，用户可以一个月内无限接收某类短信。

"按条"则是每条信息收费0.5-2元不等。

但问题在于，作为一个没有SP资质的个人创业者，我无法直接接入运营商的计费系统。

后来我拖了认识了一位在当地SP公司工作的朋友黄总。

通过她，我了解到一个重要的分成模式：运营商与SP按15:85的比例分成，而SP再从自己所得的85%中给CP分成30-50%。

也就是说，如果一条短信收费1元，运营商拿走0.15元，SP拿走0.425-0.595元，CP能获得0.255-0.425元。

但要成为CP仍然需要与SP签署正式合作协议，这对刚起步的我来说门槛依然不低。

机会往往在绝境中出现，我在一次偶然的机会中走进了陈总经营的网吧。

"你知道吗？网吧每天有几百人次流量，如果能利用起来..."陈总一边调试着网吧的电脑，一边随口说道。

那一刻，灵感如闪电般击中了我。

"网吧用户基数大，而且都是年轻人，正是铃声和短信内容的目标用户群体，如果将WAP服务与网吧主页和桌面结合..."我开始在纸上勾勒想法。

陈总的网吧有30台电脑，这个规模在当地校园周边算中等。

经过一番详谈，我提出了合作方案：我开发一个定制版内部主页和桌面快捷方式，嵌入短信业务和铃声下载的推广页面，用户在上网过程中可以方便地订阅各类短信服务或下载铃声。

最关键的是，我们将这个系统与网吧的收银系统集成，用户可以直接用上网账户余额支付，无需使用手机话费，这大大降低了消费门槛和信任壁垒。

实际操作流程是这样的：用户在网吧浏览我们的页面，选择想要的短信服务或铃声。

通过网吧收银系统支付费用。

我们记录用户的手机号码和订购信息。

然后我通过与黄总所在SP公司的合作通道，将用户订单批量提交，由SP公司执行实际的短信发送或铃声下载。

这种模式中，网吧收取30%，我拿走40%，SP公司获得剩余的30%。

这个模式巧妙地解决了几个关键问题：

一是绕过了用户对移动增值业务的不信任（当时行业内乱收费现象较多）。

二是规避了我没有SP资质的障碍。

三是为网吧创造了额外收入来源，增强了合作粘性。

2004年底到2005年初，这个项目迅速扩展。

我开发了更丰富的内容，包括校园笑话、情感短信、流行歌曲铃声等，精准匹配网吧年轻用户群体的喜好。

第一个月，系统仅产生了300元的收入。

第三个月，收入突破了2000元。

半年后，当系统推广到附近3家网吧后，月收入稳定在5000元左右。

用户可以通过三种方式使用我们的服务：

一是在网吧直接用上网费支付下载铃声或订阅短信内容。

二是回到家后，通过手机WAP浏览器访问51313.com进行下载和订阅。

三是通过短信指令直接订阅服务。

"这个系统最大的价值在于什么？"某天，一位刚接触互联网创业的大学同学李冰问我。

"解决了信任问题，"我不假思索地回答，"在2004年，大多数人对通过手机支付还很陌生，担心被扣费。但通过网吧这个场景，用户可以用看得见的上网费余额进行消费，大大降低了心理门槛。"

这个项目持续了近一年，直到2005年底，铃声和短信内容业务逐渐饱和。

一方面是由于监管部门开始整顿SP乱收费问题，另一方面是随着竞争加剧，SP与CP的分成比例不断下降。

2005年7月，信息产业部发布了《关于同意电信业务经营许可证》，进一步规范了增值业务市场，对没有资质的CP来说，生存空间越来越小。

2005年初，我将51313.com域名转手卖给了一家做相关业务公司，这也为我的第一个创业项目画上了一个圆满的句号。

回首这段经历，我常常感慨：第一个项目就像人生的第一次恋爱，不一定完美，却永远难以忘怀。

它带着青涩与热情，带着无数的试错与调整，但正是这些，构成了宝贵的成长资产。

这次经历让我学到了几个重要的道理：

场景化消费模式的价值——将服务嵌入网吧这一用户自然聚集的场景中，能显著提高转化率和降低信任门槛。

技术与商业模式的有机结合——将WAP技术与当时流行的手机增值服务结合，找到了适合的商业模式。

分成模式中的位置选择——在运营商-SP-CP的产业链中，找到自己适合的切入点比盲目追求高比例更重要。

用户痛点与信任问题的解决——通过网吧收银系统解决了当时移动支付不发达带来的障碍。

把握时代机遇的敏锐性——在2004年手机铃声和短信内容服务爆发期，快速切入并从中获利。

尽管它当时没有联网，甚至在很长一段时间内都没有实现联网，但这并没有阻止它成为我童年中最珍贵的宝藏。

这台电脑开启了我对未知世界的探索之旅，成为了我的"魔法盒"，也为我未来构建数字资产、打造个人IP埋下了最初的种子。

  contents: # 网吧里的巨大诱惑

2000年秋天，我刚升入高一，那是一个互联网刚刚在中国兴起的年代。

学校对学生管理很严格，晚上10点熄灯后不允许外出。

但对于我这样对新鲜事物充满好奇的少年来说，学校围墙外那个神秘的"网吧"世界，有着巨大的吸引力。

那是一个月黑风高的夜晚，宿舍熄灯后，我和室友小张、小李开始密谋我们的"越狱计划"。

"听说学校下面新开了一家网吧，里面有最新的游戏。"小张压低声音说。

"《红色警戒》你们玩过吗？听说可以多人联网对战！"小李眼中闪着兴奋的光芒。

我们三个人蹑手蹑脚地来到宿舍楼后面的围墙边。

围墙大概有2米高，对于我们这些十六七岁的少年来说，并不算什么障碍。

"我先上去探路。"我自告奋勇，踩着小张的肩膀翻过了围墙。

那一刻，心跳如鼓，既紧张又兴奋。

翻过围墙的瞬间，仿佛整个世界都变得不同了，一种前所未有的自由感涌上心头。

我们沿着小路摸黑走了大概10分钟，终于看到了那家传说中的网吧——"飞宇网络"。

推开门的那一刻，我被眼前的景象震撼了：几十台电脑整齐排列，屏幕上闪烁着各种游戏画面，键盘敲击声此起彼伏，空气中弥漫着烟味、方便面的香气，还有一种说不出的电子设备的味道。

"三位同学，要上网吗？2元一小时。"网管是个二十多岁的年轻人，看起来很和善。

我们找了三台相邻的电脑坐下。

当我第一次点击桌面上的"红色警戒"图标时，心情无比激动。

游戏加载完毕，我选择了"多人游戏"模式。

很快，我就和小张、小李以及网吧里的其他玩家组成了一个8人对战房间。

"这局我选苏联，造天启坦克！"我在聊天框里打字。

"我选美国，爱国者导弹防空！"小张回复。

那种多人实时对战的刺激感，是我从未体验过的。

每一次建造建筑、训练士兵、发动攻击，都需要与其他7个真实玩家进行策略博弈。

我第一次理解了什么叫"网络效应"——参与的人越多，游戏就越有趣。

这一玩就是整整一夜。

当我们意识到时间时，天已经快亮了。

"糟了！马上要查房了！"小李惊呼。

我们匆忙结账，沿着来路狂奔回学校。

翻墙进入宿舍楼时，我的心跳得比刚才玩游戏时还要快。

第二天早上，果然被班主任王老师发现了。

"卡若，你昨晚是不是没睡好？"王老师看着我困倦的样子问道。

我站在教室门口，困得直点头，但心里却还在回味昨晚游戏的精彩瞬间。

从那以后，我开始思考如何"规避风险"——什么时候去网吧不容易被发现，如何安排时间既能玩游戏又不影响学习。

没想到，这种风险评估的思维，后来成了我商业决策中的重要能力。

几个月后，网吧里开始流行一种新的应用——语音聊天室。

那是一个周末的下午，我在网吧里发现了一个叫"碧海银沙聊天室"的软件。

出于好奇，我点击进入了一个名为"青春校园"的房间。

"大家好，有人在吗？"我试探性地在麦克风前说道。

"你好，你是哪个学校的？"一个清脆的女声传来。

"我是泉州一中的，你呢？"

"我是泉州三中的，我叫小雨。"

我们开始聊起各自的学校生活、兴趣爱好。

令我惊讶的是，我们有很多共同话题——都喜欢看《还珠格格》，都喜欢听周杰伦的歌，都对计算机很感兴趣。

聊了大概一个小时后，小雨突然问："你是不是叫卡若？"

"是啊，你怎么知道？"我很惊讶。

"我是林小雨啊！初中时我们是同班同学！"

那一刻，我简直不敢相信自己的耳朵。

林小雨是我初中时暗恋过的女生，没想到在这个虚拟的网络空间里重新相遇了。

这次偶遇让我深深震撼。

我意识到，互联网不仅仅是一个游戏平台，更是一个能够连接人与人的神奇世界。

它可以跨越时间和空间的限制，让失散的朋友重新相聚，让陌生的人成为朋友。

这不就是我后来一直在做的"私域连接"的雏形吗？

但是，沉迷网吧也带来了严重的后果。

期中考试成绩出来后，我的排名从班级前10名跌到了30名开外。

父亲看到成绩单后，脸色铁青。

"卡若，你最近是怎么了？成绩怎么下降这么多？"

母亲也很担心："是不是学习压力太大了？还是身体不舒服？"

我不敢告诉他们真相，只能含糊地说："可能是最近学习方法不对。"

班主任王老师也找我谈话："卡若，你是个聪明的孩子，但最近的状态让我很担心。你要知道，高中三年很关键，不能因为一时的放松而影响了前途。"

那一刻，我内心充满了愧疚和自责。

我意识到，诱惑就像沼泽，陷进去容易，爬出来难。

我开始反思自己的行为，并制定了一个"平衡计划"：

每周只去网吧一次，每次不超过3小时。

去网吧前必须完成当天的作业。

将在网吧学到的计算机知识应用到学习中。

这个计划的执行并不容易，但它教会了我一个重要的道理：自律不是完全拒绝诱惑，而是学会控制和平衡。

几个月后，我的成绩逐渐回升，同时我对计算机和互联网的理解也越来越深入。

现在回想起来，那段在网吧的日子，虽然看似是青春的叛逆，但实际上却是我人生中极其重要的启蒙阶段。

我学到了网络效应的力量——参与者越多，价值越大。

我学到了风险管控的重要性——任何行为都要评估风险和收益。

我学到了连接的价值——互联网最大的魅力在于连接人与人。

我学到了平衡与自律——成长就是在诱惑和责任之间找到平衡。

这些认知，为我后来的互联网创业之路奠定了重要的思维基础。

  contents: # 超级个体：1人公司

2023年，站在厦门环岛路的沙滩上，海风吹拂着我的脸庞，思绪却飞向了更远的未来。

过去几年，我见证了互联网从流量红海到私域深耕的转变，也亲身实践了如何构建数字资产，如何通过私董会实现价值共创。

然而，在这些探索的背后，我逐渐意识到一个更深层次的趋势正在悄然兴起——"超级个体"的时代。

这不再是传统意义上的"自由职业者"或"斜杠青年"，而是那些拥有独特认知、稀缺技能，并能够高效整合资源、创造巨大影响力的个体。

他们不再依附于大型组织，而是凭借个人IP和连接力，在数字世界中开辟自己的领地，像一个高效运转的"1人公司"。

正如我常说的，"个体崛起的时代，认知决定你的高度，链接决定你的广度。"

这不仅仅是一种趋势，更是一种必然，也是我在IP财富旅程中，对未来商业形态的深刻预判。

我的"云阿米巴模式"在过去几年中取得了显著成功，通过将大团队拆分为小单元，实现了高效运营和利润共享。

但在实践中，我发现了一些更有趣的现象：那些表现最突出的小团队负责人，他们不仅能完成既定任务，还能主动思考、积极创新，甚至开始构建自己的小影响力圈子。

这让我意识到，传统的管理模式虽然有效，但如何进一步激发个体的潜能，让他们从"螺丝钉"变成"引擎"，像独立的"1人公司"一样运作，才是未来的关键。

2023年中，我与一位内容创作领域的伙伴进行了一次深度交流。

他当时正在为一个内容创作团队的效率瓶颈而烦恼。

我向他分享了我的"写作即编程"理念，并提出了"赋能个体，构建IP"的策略。

我建议他，与其仅仅关注内容的量产，不如帮助团队中的优秀写手打造个人IP，让他们成为各自领域的KOL。

例如，一个擅长情感文案的写手，可以帮助他规划个人抖音账号，分享写作技巧，积累粉丝，从而实现从"写手"到"情感导师"的升级。

我们还探讨了如何利用AI工具，为这些个体提供内容创作、数据分析、用户运营等方面的支持，让他们能够更高效地进行创作和变现，从而独立运作如"1人公司"。

我与一家知名内容平台的管理者合作，也印证了"超级个体"的巨大潜力。

该平台本身就是一个强大的内容IP平台，其庞大的用户群体和知识体系为个体提供了肥沃的土壤。

我与该管理者探讨了如何将平台优质内容与我的私域流量运营经验相结合，帮助更多知识付费领域的讲师、咨询师打造个人品牌，使其能够像"1人公司"一样进行独立运营。

例如，一位在该平台有课程的老师，通过我的私域引流和社群运营方案，将其课程用户沉淀到自己的微信社群，再通过每周一次的线上直播答疑、每月一次的线下沙龙，将普通学员转化为高粘性粉丝，甚至成为其付费咨询的客户。

这种合作，让个体在平台的赋能下，实现了价值的倍增。

然而，"超级个体"的崛起并非没有挑战。

我曾观察到一些追求极致自由的个体，他们在获得巨大影响力的同时，也面临着自我管理、持续产出和风险抵抗的巨大压力。

他们享受着"真正的自由，是掌控自己的时间，创造无限的价值"的快感，但有时也会陷入缺乏外部支持、资源匮乏的困境。

这让我反思，如何才能让这些像"1人公司"一样运作的个体在自由生长的前提下，也能拥有持续发展的韧性。

在一次与一位行业观察者的对话中，她提到了一个案例：一位曾经风光无限的个人IP，因为内容更新频率降低、与粉丝互动减少，导致影响力迅速下滑。

这位观察者强调，"你的IP，就是你在这个世界的通行证，也是你最稳固的资产。"

但这资产需要持续的维护和投入。

这并非简单的并购或合作，而是通过搭建一个开放的平台，实现资源的裂变与认知的重构。

我与杨红的多次交流，让我对"平台赋能"有了更深刻的理解。

杨红在行业内拥有广泛的资源和深厚的人脉。

杨红则负责对接更多像这样的优质个体，扩大平台的合作广度。

王诚鹏在金融、不良资产处置领域拥有丰富的经验和资源。

与李冰（木子）和吉咪宇（小吉）的合作，不再是传统的雇佣关系，而是基于项目合作的"合伙人"模式。

最大的挑战在于"信任"的构建和"去中心化"管理。

传统的企业，通过层级和制度来约束员工，而生态型组织则更依赖于共同的愿景、清晰的规则和强大的信任。

我曾与陈鹭明（明哥）探讨过这个问题，他是一位在传统企业管理方面经验丰富的资深人士。

这样，每个人都能看到自己的付出如何转化为实际收益，从而增强对生态的信任和归属感。

明哥也提出了一些关于风险控制和法律合规的建议，这些都被我纳入了生态系统的规则设计中。

  contents: # 超级个体：1人公司

2024年7月，我在飞书上写下了这一年的核心目标：帮助1万个超级个体资产放大3倍。

这不是一个口号，而是我用11年私域经验验证出来的路径。

那时候我刚从杭州30天挑战回来，团队背叛的阴影还没完全消散，但我已经看清了一件事——传统的雇佣关系正在瓦解，未来属于超级个体。

什么是超级个体？

我给它下了一个定义：一个人，借助AI和系统工具，能做到过去10人团队的产出，实现年入百万的"1人公司"。

这个定义不是空想，而是我亲眼见证的真实案例。

2024年9月，我和一个叫龙楷的年轻人聊了一个下午。

他之前在黑谷做销售，后来跑到厦门想创业。

他跟我说："卡若，我想做一个能赚钱的项目，但我没资金，没团队，只有我一个人。"

我问他："你有什么？"

他说："我有执行力，我愿意学，我对商业有一定的认知。"

我说："够了。这就是超级个体的起点。"

我给他讲了我的碎片时间项目。

这个项目的核心逻辑很简单：让宝妈、残疾人、学生等群体，利用碎片时间做简单任务，比如挂机微信、发朋友圈、剪辑视频、点赞抖音。

每个任务设计成15分钟到半小时就能完成，做完直接结算。

2024年上半年，我们已经有5000个微信号在挂机运转。

龙楷听完眼睛亮了："这不就是众包模式吗？"

我说："不只是众包。众包是你给别人打工，超级个体是你自己当老板。"

"你来找宝妈挂机，每个月给她们1000-2000的补贴。"

"这些微信号用来做什么？用来承接我们的知识付费产品、游戏课程、私域咨询。"

"你从中抽30%作为运营费用，宝妈赚了，你也赚了，我也赚了。"

"这就是云阿米巴——分不属于对方的钱，按创造价值分钱。"

龙楷当场决定加入。

三个月后，他一个人管理着80个兼职宝妈，月流水稳定在8万以上，自己到手2万多。

他就是一个典型的超级个体。

但龙楷的成功不是偶然，背后有一套系统在支撑。

2024年9月的一次技术会议上，我跟团队定下了存客宝的核心方向：模块化赋能超级个体。

什么意思？

就是把我们积累的私域工具、AI功能、运营SOP，全部打包成可复用的"工具箱"。

比如AI改写朋友圈——我们的系统能自动采集行业顶级销售的朋友圈内容，用AI改写成你自己的风格，每天自动发5条。

过去需要一个专职文案，现在系统3秒搞定。

比如自动养号——每个微信号会随机分配AI人设，自动跟好友互动、点赞朋友圈、发起聊天。

过去需要50个人管理100个微信，现在5个人就能管理200个。

比如客资回收——用户通过付款码支付后，手机号自动进入私域池，系统自动打标签、分配跟进人员。

过去需要手工录入，现在全自动。

这些工具，就是超级个体的"外挂"。

有了这套系统，一个人真的可以做到过去10人团队的产出。

2024年9月，我见了一个叫流光的创业者。

他是做性格色彩培训的，之前一个人单打独斗，一年勉强赚个三四十万。

我跟他说："你现在最大的问题是什么？"

他说："流量。我讲课没问题，但找不到客户。"

我说："这就对了。超级个体的第一课，就是解决流量问题。"

我给他设计了一套方案：

第一步，用我们的矩阵系统，每个月拍10-20条短视频，通过30个账号同时分发。

每条视频曝光量3万到10万，按最低算，一个月也有近100万曝光。

第二步，视频挂链接，用户点击后直接加微信。

每100个曝光能加1个精准用户，一个月能加1000人。

第三步，这1000人进入私域后，通过AI自动养号、朋友圈运营、社群管理，逐步建立信任。

第四步，每个月做一场99元的沙龙，30人参加就是3000收入，现场转化3-5个高价咨询客户，每个客户2万。

流光算了一笔账：一场沙龙收入3000，加3个咨询客户6万，一个月做4场沙龙，收入就是25万以上。

他一个人，加上我们的系统，就能实现这个产出。

这就是超级个体的威力。

但我要说一个残酷的事实：不是所有人都能成为超级个体。

2024年下半年，我见了很多想创业的人。

有的人一听"一个人就能年入百万"，眼睛放光，问我要秘籍、要捷径。

我都拒绝了。

因为超级个体不是躺着赚钱，而是把10倍的效率用在正确的方向上。

它需要三个条件：

第一，你得有一技之长。

流光能做咨询是因为他性格色彩讲了十几年，龙楷能管理团队是因为他执行力超强。

没有专业能力，再好的工具也只是摆设。

第二，你得愿意持续输出。

超级个体的核心资产是个人IP，而IP需要持续的内容喂养。

我见过太多人一开始热情高涨，一个月后就断更了。

抖音的算法不会因为你累了就给你流量。

第三，你得接受延迟满足。

前三个月可能一分钱都赚不到，半年后才能稳定出单，一年后才能形成复利。

很多人熬不过第三个月。

所以我常说：超级个体不是让你轻松，而是让你高效。

轻松是偷懒者的幻想，高效是奋斗者的武器。

2024年年底，我做了一个年度总结。

这一年，我们孵化了超过200个超级个体。

有做游戏知识付费的，月入5万。

有做婚恋咨询的，一场沙龙收入2万。

有做私域代运营的，服务10家企业，年入100万。

有做碎片时间管理的，管理100个兼职，月流水20万。

他们都有一个共同点：一个人，借助系统，实现了过去团队的产出。

这让我更加坚信，超级个体不是未来，而是现在。

AI让答案变得廉价，好问题才能让你脱颖而出。

同样，AI让效率变得廉价，独特的价值才能让你脱颖而出。

未来的商业世界，不再是大公司吃小公司的游戏。

而是一群超级个体，各自精通一个领域，通过平台连接，形成一个去中心化的生态。

这就是我对"1人公司"的理解。

不是孤军奋战，而是站在巨人的肩膀上，用10倍效率创造10倍价值。

那么，当这些超级个体聚合在一起，又会形成怎样的组织形态？

这就是我下一章要讲的——超级企业。

但他遇到了一个问题：钱不够了。

"卡若，我想走资本路线，但投资人觉得我的模式没有壁垒。"陈垚说。

我问他："你觉得壁垒是什么？"

他想了想："可能是品牌？或者是门店数量？"

我摇了摇头："壁垒不是你有什么，而是别人做不到什么。"

"你有40家门店，别人砸钱也能开40家。"

"但如果你有43万游戏用户、100个签约主播、一套AI私域运营系统，别人砸钱也复制不了。"

"这才叫壁垒。"

陈垚愣住了："但我没有这些啊。"

我笑了："我有。所以我们可以合作。"

这就是超级企业的核心逻辑——把一群超级个体聚合起来，形成一个别人无法复制的生态。

我跟陈垚讲了我的商业模式。

我的公司叫玩值，做了11年私域运营，沉淀了三个核心资产：

第一，43万游戏用户，分布在各个项目里，都是精准的、带IP属性的粉丝。

第二，100多个签约主播，从全国冠军到垂直领域KOL，每个人都是流量入口。

第三，一套AI私域运营系统，能解决微信矩阵、自动养号、内容生产、数据分析等所有问题。

这三个资产，单独拿出来都值钱，但组合在一起才是壁垒。

我跟陈垚说："我们不是简单的合作，而是成立一家运营公司。"

"你出场地和运营团队，我出流量和系统。"

"酒店的会员数据进我的私域池，我帮你做二次变现——卖游戏课程、电商产品、周边商品。"

"利润三七分，你七我三。"

陈垚问："为什么你只拿三成？"

我说："因为我要的不是一家店的利润，而是整个生态的价值。"

"你的40家店，每家店有3000个会员，就是12万用户。"

"这12万用户进入我的私域系统，加上我原有的43万用户，就是55万。"

"用户规模越大，估值越高，未来的资本故事才讲得通。"

陈垚听懂了。

他说："所以你是在用资本的逻辑做生意？"

我说："对。这就是超级企业和传统企业的区别。"

"传统企业追求的是利润最大化，每一笔钱都要赚到手。"

"超级企业追求的是生态最大化，短期可以不赚钱，甚至贴钱，但要把用户和数据留在生态里。"

"等生态成熟了，随便切一个点就能变现。"

这个思路不是我拍脑袋想出来的，而是从地摊项目里验证出来的。

2024年，我在厦门做了一个地摊项目。

政府提供场地，我投资餐车，招商让摊贩入驻，收取租金和管理费。

这个项目很简单，10个月就能回本，现金流稳定。

但我没有止步于此。

我在每个地摊上都装了付款码，用户扫码支付后，手机号自动进入我的私域池。

厦门有2000多个摊点，每天几万人扫码支付。

一个月下来，私域池里就多了十几万用户。

这些用户，我可以卖给周边的美容店、餐厅、健身房。

SDM新开了一家美容店，我给他们导流，每个用户收15块。

一个月导1000个用户，就是1.5万。

这还只是导流。

我还搭建了一个类似美团的小程序，地摊上的商品可以在线下单。

每笔订单我抽20%。

地摊老板赚了钱，周边商家有了流量，用户享受了便利，我也有了收入。

四方共赢，这就是生态。

2024年9月，我总结了超级企业的三个核心原则：

第一，不做从0到1的项目，只做从1到100的放大。

从0到1太难了，失败率太高。

我只找那些已经跑通了商业模式、有一定用户基础的项目，然后用我的流量和系统帮他们放大10倍。

比如PK电竞酒店，陈垚已经开了40家店，证明模式可行。

我进去，帮他把40家变成400家，把12万用户变成120万用户。

第二，成立运营公司，而不是做顾问或投资人。

顾问只出主意不担责任，投资人只出钱不干活。

但成立运营公司就不一样了——我出系统、出方法、出资源，对方出团队、出场地、出客户。

双方都有投入，都有风险，也都有收益。

这样的合作才稳固，才能长期。

2024年，我用这个模式合作了十几个项目：

与小鹅通销冠团队成立运营公司，卖知识付费软件，他们50人管100个微信，我帮他们用5人管200个微信。

与残疾人机构合作，让残疾人做视频剪辑和上传，按客资数量结算，每人每月一两千。

与婚恋行业合作，帮他们打造婚恋师IP，培训收费2980，婚恋局收费99起步。

每个项目，我都是运营公司的股东，而不是顾问或投资人。

第三，用现金分润，而不是股权绑定。

这是我从团队背叛那次学到的教训。

2023年，我给核心团队分了股份，结果他们拿着股份走了，还去做了竞品。

从那以后，我就想通了一个道理：千万不要想着和别人分股份，股份永远没有现金性感。

现金是最直接、最透明的激励方式。

每个月能拿到多少钱，一目了然，不会有任何争议。

而且现金分润更加灵活，可以根据贡献大小随时调整，不像股份一旦分出去就很难收回。

所以我设计了"云阿米巴"模式：

第一条：分不属于对方的钱。

只分我们共同创造的增量价值，不分我原有的存量收入。

第二条：按创造价值分钱。

根据每个人的实际贡献来分配收益，多劳多得，能者多得。

第三条：用稳定流量+便捷私域体系绑定合作方。

通过提供持续的流量支持和完善的工具体系，让合作方离不开我们的平台。

这个模式的好处是：合作方能够立即看到收益，我也能够保持对核心资源的控制，双方都有持续合作的动力。

2024年10月，我把公司的发展方向定义为"超级企业留存及估值体系"。

什么意思？

就是把所有合作项目的用户数据，都沉淀到统一的私域系统里。

每个用户都有标签、有行为记录、有消费能力评估。

这些数据，就是公司的核心资产。

当数据积累到一定规模，就可以用资本的方式进行估值和变现。

比如，一个能产生10万用户的项目，如果每个用户能带来10元收益，整个项目就值100万。

我用这个逻辑去跟投资人讲故事，比单纯讲"我有多少收入"更有吸引力。

2024年12月，我定下了一个更大的目标：1万台手机KPI。

什么意思？

就是用1万台手机，每台手机运行一个微信号，每个微信号承载500个用户。

1万台手机，就是5000万私域用户。

5000万用户，哪怕每个用户只产生1块钱价值，就是5000万的估值。

这个目标听起来疯狂，但我已经迈出了第一步。

2024年，我们已经有5000个微信号在稳定运转。

2025年，我要把这个数字翻一倍。

然后一步一步，走向1万，走向10万，走向更大的生态。

回顾我的整个创业历程，从1998年第一次接触互联网，到2024年构建超级企业生态。

26年了。

我从一个沉迷网吧的少年，变成了一个管理数千万私域用户的企业主。

这一路，有成功，有失败，有背叛，有坚持。

但有一件事从未改变：对效率和杠杆的极致追求。

超级企业不是终点，而是新的起点。

未来的商业世界，不再是层级森严的金字塔。

而是一群超级个体，通过共同的平台和生态，实现价值的无限放大。

我的IP财富旅程，也将在这个生态中，找到最广阔的舞台，实现最深远的价值。

这不仅仅是我的故事，更是未来商业的故事。

  contents: # 我的童年

1984年5月26日，农历四月二十六，凌晨0点0分。

我出生在福建厦门一个普通的工人家庭。

父亲是个木匠，母亲在纺织厂上班。

那时候的厦门还不是现在的经济特区，只是一个靠海的小城市。

家里条件不算好，但也不算差。

至少，在我7岁之前，我以为所有的家庭都是这样的——父亲每天早出晚归，母亲忙完工厂还要忙家务，而我就在这种平凡中慢慢长大。

直到1991年，一切都变了。

那年我7岁，刚上小学一年级。

有一天放学回家，发现家里的气氛很奇怪。

父亲坐在客厅的沙发上抽烟，一根接一根，烟灰缸里堆满了烟头。

母亲在厨房里收拾东西，眼睛红红的。

我问："妈妈，你怎么了？"

母亲蹲下来，摸了摸我的头，说："智群，爸爸妈妈要分开住了。"

我当时不懂什么叫离婚，只知道从那天开始，我就跟着母亲生活了。

父亲搬出了家，偶尔会来看我，但每次都很匆忙，话也不多。

母亲一个人既要上班，又要照顾我，压力很大。

我能感觉到她的疲惫，但那时候的我，除了乖一点，什么也帮不上。

父母离异对我的影响，在很多年后才真正显现出来。

小学的时候，我还算听话，成绩也不错。

但到了初中，叛逆期来了，而且来得特别猛烈。

1997年，我13岁，初一。

那时候班里有个同学叫阿明，家里条件很好，每天都有零花钱买零食。

我很羡慕，但又不好意思跟母亲要钱。

有一天，我看到阿明把钱包放在桌子上就去上厕所了。

我鬼使神差地走过去，从他钱包里拿了5块钱。

5块钱，在1997年可以买很多东西了。

我用这5块钱买了一包方便面和一瓶汽水，那种满足感让我至今难忘。

但这件事很快就被发现了。

阿明回来发现钱少了，告诉了老师。

老师让全班同学都把书包翻出来检查，当然什么也没找到，因为钱已经被我花掉了。

但我心里很慌，脸也红了。

老师是个很有经验的人，一眼就看出了问题。

她把我叫到办公室，没有大声训斥，只是问："智群，是不是家里给的零花钱不够？"

我低着头不说话。

她又说："如果真的是你拿的，现在承认还来得及。老师不会告诉你妈妈，但你要保证以后不会再犯。"

我哭了，承认了。

老师没有惩罚我，只是让我给阿明道歉，然后把5块钱还给他。

但这件事在我心里留下了很深的印象。

不是因为羞耻，而是因为我第一次意识到：想要什么东西，不能靠偷，要靠自己赚。

这个想法在当时可能很幼稚，但它确实在我心里埋下了一颗种子。

初二的时候，我开始变得更加叛逆。

不是因为学习不好，而是因为我觉得学校教的那些东西对我来说没什么用。

我更喜欢观察身边的大人是怎么赚钱的。

我发现母亲每天上班很辛苦，但工资并不高。

而我们楼下开小卖部的王叔叔，每天就坐在店里，看起来很轻松，但生意却很好。

我经常去他的店里买东西，顺便观察他是怎么做生意的。

王叔叔人很好，有时候会跟我聊天。

有一次我问他："王叔叔，开店赚钱吗？"

他笑了笑说："赚钱是赚钱，但也不容易。你看我每天早上6点就要起来进货，晚上10点才能关门。"

我说："但是你不用像我妈妈那样去工厂上班啊。"

王叔叔想了想，说："这倒是。给别人打工和自己做生意，确实不一样。"

这句话我记了很多年。

初三的时候，我的叛逆达到了顶峰。

我开始逃课，和一些"不良少年"混在一起。

不是去打架斗殴，而是去网吧打游戏。

1998年，厦门的网吧刚刚兴起，一小时2块钱。

我没有钱，但我发现了一个"商机"。

网吧里有很多大学生和上班族，他们打游戏很厉害，但没时间练级。

而我虽然年纪小，但手速快，学东西也快。

我开始帮他们代练游戏角色，一个小时收1块钱。

这样我既能免费上网，还能赚点零花钱。

母亲发现我经常不回家吃饭，问我去哪里了。

我说在同学家写作业。

她信了，因为我的成绩虽然下降了，但还没有到不及格的程度。

但纸包不住火。

有一天母亲下班早，来学校接我，发现我根本不在学校。

她找了一圈，最后在网吧里找到了我。

那天晚上，母亲哭了。

她说："智群，妈妈一个人带你不容易，你不能这样让妈妈担心。"

我也哭了，不是因为害怕，而是因为心疼母亲。

我说："妈妈，我不是在玩，我是在赚钱。"

然后我把代练的事情告诉了她。

母亲愣住了，她没想到我这么小就开始想着赚钱。

她问："那你为什么不好好读书？读书以后也能赚钱啊。"

我说："读书要很多年才能赚钱，但是代练现在就能赚钱。"

母亲不知道该怎么回答我。

那天晚上，我们母子俩聊了很久。

母亲最终妥协了，她说："如果你真的想做这个，妈妈不反对。但是你不能因此耽误学习，至少要把初中读完。"

我答应了。

从那以后，我开始过一种双重生活。

白天在学校上课，晚上和周末去网吧代练。

我的"生意"越做越大，从一开始的一个小时1块钱，涨到了一个小时2块钱。

而且我开始招募其他同学帮我一起做，我负责接单和分配任务，他们负责具体操作。

这可能是我人生中第一次体验到"管理"的感觉。

1999年，我15岁，初三毕业。

中考成绩不算好，但也不算差，勉强能上一个普通高中。

母亲问我："你想继续读书，还是去学技术？"

我想了很久，说："妈妈，我想继续读书，但我也想继续做生意。"

母亲说："那你就去读书吧，生意的事情以后再说。"

但我心里已经有了一个更大的计划。

我要用高中三年的时间，把代练生意做得更大，然后用赚到的钱去买电脑，建立自己的工作室。

这个计划在当时看来很疯狂，但我相信自己能做到。

因为我已经发现了一个秘密：在这个世界上，机会总是留给那些敢于尝试的人。

而我，就是那个敢于尝试的人。

回头看我的童年，父母离异确实给我带来了很大的影响。

但这种影响不全是负面的。

它让我比同龄人更早地意识到：生活不会因为你年纪小就对你手下留情。

你要么被生活推着走，要么主动去改变生活。

我选择了后者。

这种选择，为我后来的创业之路奠定了最重要的基础——不怕困难，敢于行动。

当然，那时候的我还不知道，真正的挑战才刚刚开始。

高中三年，将是我人生中最关键的转折点。

那里有我的第一台电脑，我的第一个网站，还有我真正意义上的第一桶金。

def blocks_to_markdown(blocks):
    """将飞书文档块转换为Markdown"""
    markdown = []
    
    for block in blocks:
        block_type = block.get('block_type')
        block_id = block.get('block_id', '')
        
        # 标题
        if block_type == 3:  # heading1
            text = extract_text(block.get('heading1', {}).get('elements', []))
            markdown.append(f"# {text}\n")
        elif block_type == 4:  # heading2
            text = extract_text(block.get('heading2', {}).get('elements', []))
            markdown.append(f"## {text}\n")
        elif block_type == 5:  # heading3
            text = extract_text(block.get('heading3', {}).get('elements', []))
            markdown.append(f"### {text}\n")
        elif block_type == 6:  # heading4
            text = extract_text(block.get('heading4', {}).get('elements', []))
            markdown.append(f"#### {text}\n")
        
        # 段落
        elif block_type == 2:  # text
            text = extract_text(block.get('text', {}).get('elements', []))
            if text.strip():
                markdown.append(f"{text}\n")
        
        # 列表
        elif block_type == 7:  # bullet
            text = extract_text(block.get('bullet', {}).get('elements', []))
            markdown.append(f"- {text}\n")
        elif block_type == 8:  # ordered
            text = extract_text(block.get('ordered', {}).get('elements', []))
            markdown.append(f"1. {text}\n")
        
        # 引用
        elif block_type == 11:  # quote
            text = extract_text(block.get('quote', {}).get('elements', []))
            markdown.append(f"> {text}\n")
        
        # 代码块
        elif block_type == 12:  # code
            code = block.get('code', {})
            language = code.get('language', '')
            text = extract_text(code.get('elements', []))
            markdown.append(f"```{language}\n{text}\n```\n")
        
        # 表格
        elif block_type == 31:  # table
            table = block.get('table', {})
            rows = table.get('rows', [])
            if rows:
                # 表头
                header = rows[0]
                header_cells = [extract_text(cell.get('elements', [])) for cell in header.get('cells', [])]
                markdown.append("| " + " | ".join(header_cells) + " |\n")
                markdown.append("| " + " | ".join(["---"] * len(header_cells)) + " |\n")
                
                # 数据行
                for row in rows[1:]:
                    cells = [extract_text(cell.get('elements', [])) for cell in row.get('cells', [])]
                    markdown.append("| " + " | ".join(cells) + " |\n")
        
        # 任务列表
        elif block_type == 17:  # todo
            todo = block.get('todo', {})
            text = extract_text(todo.get('elements', []))
            checked = "x" if todo.get('style', {}).get('done') else " "
            markdown.append(f"- [{checked}] {text}\n")
        
        # 高亮框
        elif block_type == 19:  # callout
            callout = block.get('callout', {})
            text = extract_text(callout.get('elements', []))
            markdown.append(f"> 💡 {text}\n")
        
        # 分割线
        elif block_type == 27:  # divider
            markdown.append("---\n")
        
        # 递归处理子块
        children = block.get('children', [])
        if children:
            child_md = blocks_to_markdown(children)
            markdown.extend(child_md)
    
    return markdown

def extract_text(elements):
    """从元素中提取文本"""
    text_parts = []
    for elem in elements:
        if 'text_run' in elem:
            content = elem['text_run'].get('content', '')
            style = elem['text_run'].get('text_element_style', {})
            
            # 处理样式
            if style.get('bold'):
                content = f"**{content}**"
            if style.get('italic'):
                content = f"*{content}*"
            if style.get('strikethrough'):
                content = f"~~{content}~~"
            
            text_parts.append(content)
        elif 'mention' in elem:
            mention = elem['mention']
            if mention.get('type') == 1:  # user

def extract_text(elements):
    """提取文本"""
    text_parts = []
    for elem in elements:
        if 'text_run' in elem:
            content = elem['text_run'].get('content', '')
            style = elem['text_run'].get('text_element_style', {})

def blocks_to_markdown(blocks):
    """转换为Markdown"""
    markdown = []
    
    for block in blocks:
        block_type = block.get('block_type')
        
        if block_type == 3:  # heading1
            text = extract_text(block.get('heading1', {}).get('elements', []))
            markdown.append(f"# {text}\n\n")
        elif block_type == 4:  # heading2
            text = extract_text(block.get('heading2', {}).get('elements', []))
            markdown.append(f"## {text}\n\n")
        elif block_type == 5:  # heading3
            text = extract_text(block.get('heading3', {}).get('elements', []))
            markdown.append(f"### {text}\n\n")
        elif block_type == 6:  # heading4
            text = extract_text(block.get('heading4', {}).get('elements', []))
            markdown.append(f"#### {text}\n\n")
        elif block_type == 2:  # text
            text = extract_text(block.get('text', {}).get('elements', []))
            if text.strip():
                markdown.append(f"{text}\n\n")
        elif block_type == 7:  # bullet
            text = extract_text(block.get('bullet', {}).get('elements', []))
            markdown.append(f"- {text}\n")
        elif block_type == 8:  # ordered
            text = extract_text(block.get('ordered', {}).get('elements', []))
            markdown.append(f"1. {text}\n")
        elif block_type == 11:  # quote
            text = extract_text(block.get('quote', {}).get('elements', []))
            markdown.append(f"> {text}\n\n")
        elif block_type == 17:  # todo
            todo = block.get('todo', {})
            text = extract_text(todo.get('elements', []))
            checked = "x" if todo.get('style', {}).get('done') else " "
            markdown.append(f"- [{checked}] {text}\n")
        elif block_type == 19:  # callout
            callout = block.get('callout', {})
            text = extract_text(callout.get('elements', []))
            markdown.append(f"> 💡 {text}\n\n")
        elif block_type == 27:  # divider
            markdown.append("---\n\n")
        
        # 递归处理子块
        children = block.get('children', [])
        if children:
            child_md = blocks_to_markdown(children)
            markdown.extend(child_md)
    
    return markdown
