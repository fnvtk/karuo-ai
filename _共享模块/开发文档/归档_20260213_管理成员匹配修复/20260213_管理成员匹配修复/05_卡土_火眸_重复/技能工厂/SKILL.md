## Skill 模板

### 目录结构

```
skill-name/
├── SKILL.md           # 必须：触发规则+使用说明
├── scripts/           # 可选：可执行脚本
│   └── main.py
├── references/        # 可选：参考文档
│   └── guide.md
└── assets/            # 可选：模板资源
    └── template.md
```

## 示例：飞书日程转周报

### 用户输入

```
帮我创建一个飞书日程转周报的Skill
```

### 生成结果

```
飞书周报/
├── SKILL.md
├── scripts/
│   └── 生成周报.py
└── references/
    └── 周报模板.md
```

**SKILL.md**:
```markdown
---
name: 飞书周报
description: 飞书日程转周报。触发词：生成周报、飞书日程、本周工作、周报生成。自动从飞书日程提取本周工作，生成周报Markdown。
---

# 飞书周报

自动从飞书日程生成周报。

## 使用方法

1. 导出飞书日程（本周）
2. 运行生成脚本
3. 获得周报Markdown

## 周报格式

- 本周完成
- 下周计划
- 需要协助
```

## 创建指令格式

```
创建Skill: [名称]
功能: [做什么]
触发词: [关键词列表]
输入: [需要什么]
输出: [产出什么]
```

## Skill 质量标准

| 维度 | 标准 |
|------|------|
| 触发词 | 覆盖用户可能的表达（≥3个） |
| 说明 | 简洁明了，一看就懂 |
| 脚本 | 可直接执行，有错误处理 |
| 文档 | 有示例，有边界说明 |
