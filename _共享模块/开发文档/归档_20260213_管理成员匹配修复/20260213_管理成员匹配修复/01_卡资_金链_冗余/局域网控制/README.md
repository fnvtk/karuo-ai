### 1. 自动设备发现
自动扫描局域网（192.168.x.0/24）中的所有设备：
- 自动识别设备类型（手机/电视/电脑）
- 自动检测控制方式（ADB/VNC/SSH）
- 获取设备详细信息（品牌/型号/系统版本）

- Android 设备：自动 ADB 连接
- VNC 设备：打开系统 VNC 客户端
- 电脑设备：SSH/RDP 连接

- 基于 scrcpy 的高性能镜像
- 60fps 流畅显示
- 延迟 < 100ms
- 支持录屏和截图

### 4. 应用管理
- 安装/卸载应用
- 查看已安装应用
- 应用权限管理
- 应用启动/停止

### 5. 系统诊断

- 问题修复

## 🔧 常见问题

### Q1: 扫描不到设备？

**原因**：
- 设备和电脑不在同一网段
- 防火墙阻止了网络扫描
- 设备未开启 ADB 调试

**解决**：
```bash
# 1. 检查网络
ping 192.168.2.15

# 2. 检查 ADB 服务
adb kill-server
adb start-server

# 3. 手动连接
adb connect 192.168.2.15:5555
```

### Q2: ADB 连接失败？

**解决**：
```bash
# 尝试不同端口
adb connect 192.168.2.15:5555
adb connect 192.168.2.15:5037
adb connect 192.168.2.15:4444

# 检查设备端
对我说："检查 192.168.2.15 的 ADB 状态"
```

### Q3: 屏幕镜像卡顿？

**解决**：
```bash
# 降低分辨率
scrcpy -m 1024

# 降低比特率
scrcpy -b 2M

# 降低帧率
scrcpy --max-fps 30
```

### Q4: 应用安装失败？

**解决**：

