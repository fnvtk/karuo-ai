- **操作链路**：后台发指令 → 云端 API → WebSocket 下发 → 手机 Hook 执行 → 结果回传；实时同步：Hook 拦截新消息 → 上报云端 → 后台推送。  
- **技术**：手机端 Hook（LSPosed/Xposed 等）+ 常驻 Agent + WebSocket；云端 API Gateway + WebSocket Hub + 业务与数据库；管理端 007 Web + Swagger。
