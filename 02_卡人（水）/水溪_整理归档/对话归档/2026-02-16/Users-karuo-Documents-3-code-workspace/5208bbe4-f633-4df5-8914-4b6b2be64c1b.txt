## 一、前置条件

| 项目 | 要求 |
|------|------|
| nas-2 | 192.168.1.201，群晖 DS1825+ |
| 网络 | Mac 与 nas-2 同一局域网 |
| 群晖 DSM | ≥6.2（推荐 7.0+） |
| macOS | ≥10.14 |
| 存储空间 | ≥Mac 总数据量的 1.5 倍 |

## 二、群晖 nas-2 端配置（首次需检查）

### 2.1 创建/启用 Time Machine 共享

1. 登录 DSM：http://192.168.1.201:5000
2. **控制面板** → **共享文件夹** → 新建或编辑已有共享
3. 勾选 **「启用 Time Machine 支持」**
4. 建议取消「启用回收站」（节省空间）

### 2.2 启用 SMB 服务

1. **控制面板** → **文件服务** → **SMB**
2. 勾选「启用 SMB 服务」
3. 高级设置中启用 SMB 2.0 和 3.0
4. 勾选「Bonjour Time Machine 播送」（可选，便于发现）

### 2.3 用户权限

- 用户 `fnvtk` 对 Time Machine 共享文件夹需有**读写**权限
- 密码：`zhiqun1984`（小写）

## 五、常见问题

### Q1: 错误 45 - 不兼容的文件系统

**原因**：群晖共享文件夹未启用 Time Machine 支持  
**解决**：DSM → 共享文件夹 → 编辑 → 勾选「启用 Time Machine 支持」

### Q2: 连接超时

**原因**：网络不通或 SMB 未启用  
**解决**：`ping 192.168.1.201` 检查；DSM 确认 SMB 已启用

### Q3: 下次备份：连接电源后

**说明**：macOS 默认在连接电源时自动备份，可节能  
**手动备份**：菜单栏时间机器图标 → 立即备份

### Q4: 已有 DiskStation.local 的共享/Backup

**说明**：DiskStation.local 即 nas-2 的 mDNS 名称，同一台设备  
**操作**：若已配置，无需重复；若想切换到 nas-2 专用共享，可运行脚本指定共享名

def check_dns(host: str) -> bool:
    """检查 DNS 解析"""
    try:
        socket.gethostbyname(host)
        return True
    except socket.gaierror:
        return False


def check_https(host: str, timeout: float = 5.0) -> tuple[bool, str]:
    """检查 HTTPS 连通性"""
    url = f"https://{host}/"
    ctx = ssl.create_default_context()
    try:
        req = urllib.request.Request(url, headers={"User-Agent": "TraeConnectivityCheck/1.0"})
        urllib.request.urlopen(req, timeout=timeout, context=ctx)
        return True, "OK"
    except urllib.error.URLError as e:
        return False, str(e.reason) if hasattr(e, "reason") else str(e)
    except Exception as e:
        return False, str(e)

    print("Trae.ai 登录 - 网络连通性检测")
    print("=" * 50)
    print()

    all_ok = True
    for host, port, desc in DOMAINS:
        dns_ok = check_dns(host)
        if not dns_ok:
            print(f"❌ {desc} ({host})")
            print(f"   DNS 解析失败")
            all_ok = False
            continue

        https_ok, msg = check_https(host)
        if https_ok:
            print(f"✅ {desc} ({host}) - 可访问")
        else:
            print(f"❌ {desc} ({host})")
            print(f"   HTTPS 失败: {msg}")

    print()

if __name__ == "__main__":
