- 一级用 `#`，二级 `##`，三级 `###`。

### 4. 表格与代码块

- **表格**：用 `|:---|` 做对齐，多用于「能力↔助手」「类型↔说明」「账号↔值」等对照。
- **代码块**：放路径、命令、配置示例、流程图（ASCII/简单框图），规则里也有 Python 示例。

### 5. 安全与敏感信息

- 账号/API 类文档注明「勿提交 Git、勿上传网盘、仅本机」；文件名带「索引」「明文」等提示。
