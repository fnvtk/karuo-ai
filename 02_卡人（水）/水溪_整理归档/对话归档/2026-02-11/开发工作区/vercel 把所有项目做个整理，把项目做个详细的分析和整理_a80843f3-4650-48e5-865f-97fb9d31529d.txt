## 二、如何获取 Vercel 线上完整列表

在终端任选一种方式：

**方式 A：CLI 登录后列出**

```bash
npx vercel login    # 按提示完成登录
npx vercel project ls
```

**方式 B：使用 Token（适合 CI/脚本）**

1. 打开 [Vercel Dashboard → Settings → Tokens](https://vercel.com/account/tokens) 创建 Token。
2. 执行：
```bash
VERCEL_TOKEN=你的Token npx vercel project ls
```

**方式 C：用 API 拉取（可写进脚本）**

```bash
# 需先设置 VERCEL_TOKEN
curl -H "Authorization: Bearer $VERCEL_TOKEN" "https://api.vercel.com/v9/projects"
```

获取到线上列表后，可把项目名、域名、生产/预览 URL、最近部署时间等补充到本文「四、Vercel 线上项目清单」中。
