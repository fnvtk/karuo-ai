- **TikHub**：https://tikhub.io ，需注册获取 API Key；扣子插件里配置后即可在工作流中「输入抖音链接 → 输出文案等信息」。

## 三、豆包接口（视频内容 → 文字）

- **场景**：已有**视频文件**或**视频 URL**，需要「视频内语音转文字」或「画面内容理解」。  
- **豆包/火山引擎**：  
  - 语音识别：端到端语音大模型 API（如豆包语音）可做**语音转文字**。  
  - 多模态：豆包多模态能力可理解画面+文字，输出描述或摘要。  
- **与抖音解析的衔接**：  
  1. 先用 **Evil0ctal API** 或 **douyin_parse** 从分享链接拿到**无水印视频 URL** 或下载到本地。  
  2. 再调豆包「语音识别」或「多模态」接口，得到视频内文字/解说文案。  

这样实现「丢一个抖音链接 → 最终得到视频里的文字内容」。

TIMEOUT = 15


def normalize_url(raw: str) -> str:
    """从分享口令或文本中提取纯净的抖音链接。"""
    raw = raw.strip()
    # 短链
    m = re.search(r"(https?://v\.douyin\.com/[A-Za-z0-9_-]+/?)\s*", raw)
    if m:
        return m.group(1).rstrip("/") if not m.group(1).endswith("/") else m.group(1).rstrip("/")
    # 长链
    m = re.search(r"https?://(www\.)?douyin\.com/(video|discover)[^\s]*", raw)
    if m:
        return m.group(0).split()[0] if " " in m.group(0) else m.group(0)
    if raw.startswith("http"):
        return raw.split()[0]
    return raw


def fetch_video_data(url: str, api_base: str = DEFAULT_API_BASE) -> dict:
    """请求 hybrid video_data 接口，返回 JSON。"""
    api_url = f"{api_base.rstrip('/')}/api/hybrid/video_data"
    params = {"url": url, "minimal": "false"}
    r = requests.get(api_url, params=params, timeout=TIMEOUT)
    r.raise_for_status()
    return r.json()


def extract_text_from_response(data: dict) -> dict:
    """从 API 返回中提取文案等文字信息。"""
    out = {
        "title": "",
        "desc": "",
        "author": "",
        "author_nick": "",
        "stats": {},
        "cover_url": "",
        "video_url": "",
    }
    # 结构依 Evil0ctal 返回格式（可能在不同版本有差异）
    if isinstance(data, dict):
        # 常见字段
        aweme = data.get("aweme_detail") or data.get("item") or data
        if isinstance(aweme, dict):

            out["title"] = aweme.get("title") or out["desc"][:50] or ""
            author = aweme.get("author") or {}
            if isinstance(author, dict):
                out["author"] = author.get("unique_id") or author.get("uid") or ""
                out["author_nick"] = author.get("nickname") or ""
            out["stats"] = {

    return out

- **地址**：`https://cn.apihz.cn/api/fun/douyin.php`
- **方式**：GET/POST
- **参数**：`id=88888888&key=88888888&url=视频链接`（公共参数，无需注册）
- **说明**：免费用户为共享节点，成功率偏低，可多次重试；注册后使用自己的 id/key 可提高成功率。
- **文档**：https://www.apihz.cn/api/fundouyin.html

### 2. aag.moe（完全免 Key）

- **地址**：`https://api.aag.moe/api/dywsyjx`
- **方式**：GET
- **参数**：`url=视频链接`
- **返回**：`code`、`message`、`title`（文案）、`url`（无水印直链）
- **文档**：https://api.aa1.cn/doc/dyjx.html（聚合站说明，实际接口为 aag.moe）

### 3. AbeimAPI（免 Key，稳定性视网络而定）

- **地址**：`http://res.abeim.cn/api-dy_video_jx`
- **方式**：GET/POST
- **参数**：`url=链接`
- **返回**：作者、标题、点赞/评论、无水印链接、封面、音乐等
- **文档**：https://res.abeim.cn/api/dy_video_jx/doc.html

### 4. 92k.fun 文案提取（需固定 key，免费）

- **地址**：`http://api6.92k.fun/v1/api/transcribe/dzp_wa`
- **方式**：POST，JSON body `{"key":"sk-123456","link":"抖音链接"}`
- **说明**：公共 key 免费，服务可能不稳定（返回 500 时重试或换源）。
