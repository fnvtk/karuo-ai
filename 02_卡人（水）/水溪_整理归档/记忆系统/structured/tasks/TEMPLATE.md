# 任务结果卡模板

> 每个子任务/并行线程完成后，按此格式追加到 `structured/tasks/YYYY-MM-DD.md`。  
> 仅记录**结果**，不记录过程细节（过程留在对话归档供审计）。

---

## [任务名称]

| 字段 | 内容 |
|:---|:---|
| **goal** | 任务目标（一句话） |
| **boundary** | 边界与归属域（五行/成员/技能） |
| **acceptance** | 验收标准 |
| **key_actions** | 关键动作（≤5 条） |
| **artifact_paths** | 产出文件路径 |
| **status** | done / partial / failed |
| **next_step** | 后续（若有） |
