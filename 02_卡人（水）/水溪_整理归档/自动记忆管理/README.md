## 🎯 功能概述

自动记忆管理系统实现了卡若AI的**长期记忆能力**：

1. ✅ **自动读取**：每次对话开始时自动读取记忆文档
2. ✅ **智能分类**：自动识别内容类型（项目/技术/商业/学习/偏好/人脉/原则）
3. ✅ **自动存储**：对话结束时自动保存优质内容
4. ✅ **去重机制**：避免重复存储相同内容

## 🚀 快速使用

### 方式1：自动调用（推荐）

每次对话结束时，卡若AI会自动：
1. 识别优质内容
2. 智能分类
3. 保存到对应位置

**无需手动操作！**

### 方式2：手动调用

```python
from _共享模块.memory import MemoryManager

mm = MemoryManager()

# 保存内容（自动分类）
mm.save_content("完成了本地模型集成")

# 保存内容（指定分类）
mm.save_content("完成了本地模型集成", category="项目")

# 读取所有记忆
memories = mm.read_all_memories()
```

### 方式3：命令行调用

```bash
# 保存记忆
python scripts/memory_manager.py save "完成了本地模型集成" "项目"

# 读取记忆
python scripts/memory_manager.py read

# 分类内容
python scripts/memory_manager.py classify "完成了本地模型集成"
```

## 🔧 技术实现

### 核心文件

- `scripts/memory_manager.py` - 记忆管理核心逻辑
- `SKILL.md` - 技能文档
- `_共享模块/memory/__init__.py` - 共享模块入口

### 主要功能

1. **read_all_memories()** - 读取所有记忆文档
2. **classify_content()** - 智能分类内容
3. **save_content()** - 保存内容（自动选择位置）
4. **check_duplicate()** - 检查重复（最近7天）

---

## ⚠️ 注意事项

1. **去重机制**：相同内容7天内不重复存储
2. **精简原则**：每条记忆控制在1-2行
3. **时间戳**：自动添加时间戳 `[HH:MM]`
4. **隐私保护**：敏感信息（密码、密钥）不存储
