## 提取方法

### 手动提取（推荐首次）

1. 在 Cursor 工作台中打开对应 Agent
2. 浏览对话记录，识别关键产出（脚本、配置、分析结果、经验教训）
3. 按以下规则分类存放：
   - **脚本产出** -> 对应 Skill 的 `scripts/` 目录
   - **配置/文档** -> 对应 Skill 的 `references/` 目录
   - **经验教训** -> `_经验库/待沉淀/`
   - **商业分析** -> `_经验库/待沉淀/` 或 土簿/商业工具集/references/

### 自动提取（后续日常化）

```bash
# Agent 对话记录位置
ls ~/.cursor/projects/*/agent-transcripts/*.txt

# 提取脚本（见 Phase 4 记忆系统）
python _共享模块/memory/collect_daily.py
```
