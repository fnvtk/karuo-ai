## 🧠 需求思考模板

### 1. 需求本质（5 Whys）

**Why 1**: 用户为什么需要这个？
- 

**Why 2**: 为什么会出现这个需求？
- 

**Why 3**: 不解决会有什么后果？
- 

**Why 4**: 为什么现在需要解决？
- 

**Why 5**: 本质需求是什么？
- 

**结论**：本质需求是 [一句话总结]

---

### 2. CRITIC审视

| 维度 | 问题 | 思考结果 |
|:---|:---|:---|
| **C澄清** | 这个需求具体指什么？范围在哪？ | |
| **R假设** | 有哪些隐藏假设？这些假设成立吗？ | |
| **I证据** | 有数据/案例支持吗？来源可靠吗？ | |
| **T视角** | 用户/开发者/运营会怎么看？ | |
| **I后果** | 做/不做的短期和长期影响？ | |
| **C原点** | 这真的是最重要的问题吗？ | |
