## 二、电子书（书）在小程序里的用法

## 四、规则（重点）

### 4.1 链接与绑定规则

| 规则 | 说明 |
|------|------|
| **链接带 ID** | 分发者发出去的链接带自己的 ID；用户通过该链接进小程序，即与分发者绑定。 |
| **一级关系** | **仅一级**：只看「谁发的链接把用户带进来」。用户后续每次通过同一链接进，都算该分发者的。 |
| **绑定时长** | 通过你链接进来的人，**一个月内**在小程序的消费都与你有关。 |
| **持续有效** | **每天都有发内容**的人，绑定持续有效；「每天有发一单」的，**一直都有**。 |

### 4.2 留存与「被抢走」规则

- **长期不发**：若你长期不发内容，别人发了，新用户就可能通过别人的链接进来 → 相当于**客户被别的人「抢走」**。
- **发得多 → 绑定多 → 收益多**：坚持每天发、每天分享的人，绑定用户越来越多，**收益会越来越高**。

### 4.3 收益与分成规则

| 规则 | 说明 |
|------|------|
| **90% 给分发** | **约 90% 收益给实际做分发的人**，平台留少部分。 |
| **谁发得多谁拿得多** | 发得多、绑定多、付费多 → 收益多；做积累。 |
| **客资与收益** | 引导用户留客资、私聊、留联系方式，**有付费、有留资才有结算**；分发者能看到自己带来的**付款人数、绑定用户**等。 |

### 4.4 可见数据（分发者端）

- 通过**我的链接**进小程序的人数。
- 我绑定的**用户/客户**名单。
- 我带来的**付款人数**（及相应收益）。

### 4.5 执行与沟通原则（会上强调）

- 规则以**当前确定版本**为准；**除非有大改动，不必反复讲**，**把规则讲清楚、固定下来**即可。
- 核心就一句话：**每天能发、每天坚持发内容的人，收益会越来越高**。
- 培训、对外介绍时：**要按规则讲清楚**，避免「大概、也许、可能」；南风明确说必须自己先搞清细节再对外讲。

## 五、与「书」直接相关的规则小结

1. **发书链接**：不用先收费，直接发带 ID 的小程序链接（到群、瞬间等）。
2. **谁点谁绑定**：通过你链接进来的人，**一个月内**在小程序里为书等内容付费，都算你的。
3. **一级、一月**：只有一级；绑定以**月**为单位。
4. **坚持发**：每天发内容、发链接，绑定才持续，否则可能被他人「抢走」。
5. **收益**：谁发得多谁收益多；约 90% 给分发者，具体以小程序后台数据为准。
