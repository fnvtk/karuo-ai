## 📝 创建应用步骤

### 1. 登录飞书开放平台

访问 https://open.feishu.cn/，使用飞书账号登录。

### 2. 创建应用

1. 点击「创建应用」
2. 选择「企业自建应用」
3. 填写应用名称：如「我的云盘工具」
4. 填写应用描述
5. 点击「创建」

### 3. 获取凭证

在应用详情页的「凭证与基础信息」中获取：
- App ID
- App Secret

### 4. 配置权限

在「权限管理」中添加以下权限：

| 权限标识 | 权限名称 | 说明 |
|:---|:---|:---|
| `drive:drive` | 云盘 | 读写云盘文件 |
| `wiki:wiki` | 知识库 | 读写知识库 |
| `minutes:minutes` | 妙记 | 读取妙记内容 |
| `search:docs:readonly` | 文档搜索 | 搜索文档 |

### 5. 配置网页登录

在「安全设置」>「重定向URL」中添加：
```
http://localhost:5050/api/auth/callback
```

### 6. 发布应用

1. 进入「版本管理与发布」
2. 创建新版本
3. 提交审核（企业内部应用通常自动通过）
4. 发布上线

## ⚙️ 更新应用配置

如果需要使用自己的飞书应用，修改 `feishu_api.py` 中的配置：

```python
CONFIG = {
    'APP_ID': '你的App ID',
    'APP_SECRET': '你的App Secret'
```

## 🔐 OAuth授权流程

```
用户 → 点击授权登录 → 跳转飞书授权页 → 用户确认 → 
回调获取code → 用code换取token → 保存token → 完成
```

### Token说明

| Token类型 | 有效期 | 用途 |
|:---|:---|:---|
| `app_access_token` | 2小时 | 应用级别访问 |
| `user_access_token` | 2小时 | 用户级别访问（需授权） |
| `refresh_token` | 30天 | 刷新user_access_token |

本工具会自动：
- 保存Token到 `.feishu_tokens.json`
- 在Token过期前自动刷新
- 下次启动自动加载已保存的Token

---

## 🔒 安全建议

1. **不要泄露App Secret** - 这是敏感信息
2. **Token文件加入.gitignore** - 避免提交到版本库
3. **定期检查权限** - 只申请必要的权限
4. **使用HTTPS** - 生产环境务必使用HTTPS

## ❓ 常见问题

### Q: 授权后一直提示"未登录"?

检查以下几点：
1. 重定向URL是否正确配置
2. 应用是否已发布
3. 后端服务是否正常运行

### Q: 提示"无权限访问"?

1. 检查应用是否申请了对应权限
2. 权限是否已审核通过
3. 用户是否有对应数据的访问权限

### Q: Token无法刷新?

refresh_token有效期30天，超过后需要重新授权。
