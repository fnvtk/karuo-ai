# 对话归档

将 Cursor 对话按日期归档保存。

## 归档路径

`/Users/karuo/Documents/个人/3、工作台/AI的每日对话/`

## 文件命名

`YYYY-MM-DD｜AI的每日对话.md`

## 触发条件

**仅当用户说"归档今日对话"时执行**

## 文件格式

```markdown
YYYY-MM-DD｜一句话标题（20-40字概括）

说明：Cursor 每日对话原文归档（正文不总结、不改写）

---

USER: ...

ASSISTANT: ...

---

USER: ...

ASSISTANT: ...
```

## 规则

### 同日文件处理

1. 同一天只允许一个文件
2. 发现多个同日文件时自动合并
3. 合并后删除重复文件

### 追加规则

1. 已有文件时，新对话追加到末尾
2. 追加前插入分隔线 `---`
3. 追加完成后更新第一行标题

### 敏感信息处理

检测到密码/密钥/token时：
- 自动替换为 `[REDACTED]`
- 或部分打码（保留首尾）
- 打码前提示用户

## 正文原则

- **原样保存**：不做任何总结、提炼、重写
- **逐条记录**：按对话顺序交替记录
- **禁止总结**：正文绝对不能总结
