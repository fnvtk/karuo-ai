## 一、先发布插件（必做）

### 1.1 打开插件配置

1. 访问 https://project.feishu.cn/openapp/home
2. 点击「我的应用」中的 **Plugin ID MII_698...** 卡片
3. 左侧点 **基本信息**，填写以下必填项：

| 字段 | 填写内容 |
|:---|:---|
| 图标 | 可上传 `references/玩值电竞插件图标.svg`（或任意 .svg/.png ≤2MB） |
| 插件名称 | `玩值电竞任务同步` |
| 插件短描述 | `将甘特图任务同步到飞书项目工作项` |
| 插件分类 | 任选 1 个（如「API 授权」或「任务管理」） |
| 插件详情 | `玩值电竞落地任务从本地 Markdown 甘特图同步到飞书项目需求管理` |

### 1.2 创建并发布版本

1. 左侧点 **插件发布**
2. 点击蓝色 **创建版本** 按钮
3. 勾选 **Open API** 相关权限（创建工作项、读取项目等）
4. 提交并 **发布**

### 1.3 在空间中安装（必做，否则 10211）

发布后必须安装到空间，API 才能生效：

1. 进入 **玩值电竞-账号金融** 空间
2. **空间配置** → **插件管理** → **添加插件**
3. 切到 **企业插件** 筛选，搜索「**玩值电竞任务同步**」（Plugin ID: MII_698EA68807C08CB2）
4. 注意：需安装「玩值电竞任务同步」，不是「飞书项目」MCP Server
5. 点击 **添加**，确认权限后完成安装
6. 安装完成后重新执行同步脚本

## 三、安装步骤（发布后执行）

### Step 1：进入玩值电竞空间

1. 打开 https://project.feishu.cn
2. 登录飞书账号（存客宝公司）
3. 在左侧空间列表中找到并进入 **玩值电竞** 空间
4. 若空间名称为「账号金融」等，请确认属于玩值电竞项目

### Step 2：打开空间配置

1. 在空间内，点击左上角 **空间名称** 或右上角 **设置图标**
2. 进入 **空间配置** / **空间设置**
3. 左侧菜单选择 **插件管理**

### Step 3：添加插件

1. 在 **插件管理** 页面，点击 **添加插件**
2. 切到 **企业插件** 筛选
3. 在插件列表中搜索「玩值电竞任务同步」或 Plugin ID `MII_698EA68807C08CB2`
4. 找到对应插件后，点击 **添加** / **安装**
5. 在弹层中确认 **权限清单**（创建工作项、读取项目等）
6. 点击 **确认** 完成添加

### Step 4：确认空间权限

1. 在 **空间配置** 中检查 **成员与权限**
2. 确保调用 API 的用户（USER_KEY: 756877947514450739）具备：
   - 空间成员身份
   - 创建工作项权限
   - 若插件需「管理员」身份：将该用户设为空间管理员
3. 在 **插件管理** 中确认插件状态为 **已启用**

### Step 5：验证安装

1. 进入空间内的 **项目** 或 **需求** 列表
2. 确认可以正常创建/编辑工作项
3. 运行同步脚本验证：
   ```bash
   python3 "/Users/karuo/Documents/个人/卡若AI/02_卡人（水）/_团队成员/水桥/飞书管理/scripts/wanzhi_feishu_project_sync.py"
   ```

4. 若仍报 10211，检查下方「排查清单」

## 四、排查清单（10211 报错时）

| 检查项 | 操作 |
|:---|:---|
| 插件是否已安装到玩值电竞空间 | 空间配置 → 插件管理，确认该插件在列表中且已启用 |
| Plugin ID / Secret 是否正确 | 对照 `00_账号与API索引.md` 中的飞书项目凭证 |
| 用户是否在空间内 | 空间配置 → 成员与权限，确认 USER_KEY 对应用户为成员 |
| 插件版本是否已发布 | 开发者后台 → 插件版本管理，确认有已发布版本 |
| 租户/空间是否匹配 | 插件需在「玩值电竞」对应租户下安装，不能只在其他空间 |

## 五、开发者后台（可选）

若需重新获取 Plugin ID / Secret 或发布新版本：

1. 访问 https://project.feishu.cn/openapp/home
2. 左下角 **开发者后台** → 选择对应插件
3. **凭证**：查看 Plugin ID、Plugin Secret
4. **版本管理**：创建版本 → 勾选 Open API 权限 → 发布
5. 发布新版本后，需在 **插件管理** 中 **更新** 到最新版本
