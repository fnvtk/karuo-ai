## 六、API 标准格式（通用）

SMS-Activation.net、Tiger-SMS、GrizzlySMS 等多家平台共用同一套 API 格式（兼容 sms-activate.org 标准）：

```
GET {base_url}?api_key={KEY}&action={ACTION}&service={SERVICE}&country={COUNTRY}
```

| Action | 说明 | 返回示例 |
|--------|------|----------|
| getBalance | 查余额 | `ACCESS_BALANCE:100.0` |
| getPrices | 查价格/库存 | `{"86":{"wc":{"cost":9642,"count":124349}}}` |
| getNumberV2 | 购买号码 | `ACCESS_NUMBER:订单ID:手机号` |
| getActiveActivations | 查激活/短信 | JSON（含 smsCode） |
| setStatus (status=6) | 确认完成 | `ACCESS_ACTIVATION` |
| setStatus (status=8) | 取消退费 | `ACCESS_CANCEL` |
