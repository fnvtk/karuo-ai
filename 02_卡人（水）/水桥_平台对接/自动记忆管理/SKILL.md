## ⚠️ 注意事项

1. **去重**：相同内容不重复存储
2. **精简**：每条记忆控制在1-2行
3. **时间戳**：自动添加时间戳 `[HH:MM]`
4. **分类准确**：使用本地模型智能分类
5. **隐私保护**：敏感信息（密码、密钥）不存储

### 3. 分类机制

#### 分类方法：关键词匹配

系统使用关键词匹配来智能分类内容：

```python
CATEGORY_KEYWORDS = {
    "项目": ["项目", "完成", "上线", "部署", "功能", "模块", "版本"],
    "技术": ["技术", "代码", "架构", "API", "SDK", "集成", "优化"],
    "商业": ["商业", "ROI", "成本", "收入", "分润", "合作", "招商"],
    "学习": ["学习", "读书", "课程", "笔记", "总结", "经验"],
    "偏好": ["偏好", "习惯", "喜欢", "不喜欢", "风格", "方式"],
    "人脉": ["人脉", "认识", "合作", "关系", "朋友", "合伙人"],
    "原则": ["原则", "规则", "规范", "决策", "价值观"]
}
```

#### 分类逻辑

1. 提取内容中的所有关键词
2. 统计每个分类的关键词匹配数
3. 返回匹配数最多的分类
4. 如果没有匹配，返回"其他"

#### 示例

```python
content = "完成了本地模型SDK统一封装，创建local_llm_sdk.py"

# 关键词匹配：
# - "完成" → 项目 (+1)
# - "SDK" → 技术 (+1)
# - "封装" → 技术 (+1)

# 结果：技术（匹配数最多）
```
