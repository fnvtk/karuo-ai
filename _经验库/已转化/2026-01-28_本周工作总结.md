## 🧠 记忆机制说明

### 1. 自动读取机制

**触发时机**：每次对话开始时

**执行流程**：
```python
# 自动执行（隐藏，不显示给用户）
from _共享模块.memory import MemoryManager
mm = MemoryManager()
memories = mm.read_all_memories()

# 读取内容：
# - 主记忆文档：/Users/karuo/Documents/个人/记忆.md
# - 分类记忆：/Users/karuo/Documents/个人/记忆/（人脉.md、原则.md等）
```

**作用**：
- 将记忆内容作为上下文
- 遵循卡若的长期偏好和规则
- 参考历史项目和技术决策
- 了解人脉关系和合作历史

### 2. 自动存储机制

**触发时机**：对话结束时，检测到优质内容

**识别标准**：
- ✅ 项目进展：完成、上线、功能更新
- ✅ 技术突破：新技术、问题解决、架构优化
- ✅ 商业洞察：ROI分析、商业模式、市场判断
- ✅ 学习笔记：读书、课程、经验总结
- ✅ 偏好变化：新习惯、工具偏好、沟通方式
- ✅ 人脉信息：新关系、合作进展
- ✅ 原则更新：决策原则、行为规范

**存储流程**：
```python
# 1. 智能分类
category = mm.classify_content("完成了本地模型集成")
# 输出：项目

# 2. 检查重复（7天内）
if not mm.check_duplicate(content):
    # 3. 格式化并保存
    mm.save_content(content, category="项目")
```

**存储格式**：
```markdown
### 2026-01-28
- [23:30] [项目] 完成了本地模型SDK统一封装
- [23:32] [技术] 实现了Cursor IDE本地模型集成
```

### 3. 分类机制

**分类方法**：关键词匹配

| 分类 | 关键词 | 存储位置 |
|:---|:---|:---|
| 项目 | 项目、完成、上线、部署、功能 | 记忆.md |
| 技术 | 技术、代码、架构、API、SDK | 记忆.md |
| 商业 | 商业、ROI、成本、收入、分润 | 记忆.md |
| 学习 | 学习、读书、课程、笔记、总结 | 记忆.md |
| 偏好 | 偏好、习惯、喜欢、风格 | 记忆.md |
| 人脉 | 人脉、认识、合作、关系 | 人脉.md |
| 原则 | 原则、规则、规范、决策 | 原则.md |

### 4. 去重机制

**检查范围**：最近7天内的记忆

**匹配逻辑**：
- 提取内容关键词
- 检查关键词匹配度（≥50%视为重复）
- 重复内容不保存
